# 供应链战略

供应链战略决定了供应链体系的设计方向和投入重点。不同行业、不同发展阶段的企业需要选择不同的供应链战略取向：精益、敏捷或韧性。

---

<!-- chunk: 供应链战略.一-三种供应链战略取向 -->
## 一、三种供应链战略取向

### 1.1 精益供应链（Lean）

| 维度 | 内容 |
|---|---|
| 核心目标 | 消除浪费，最低成本运营 |
| 适用场景 | 需求稳定、产品标准化、竞争以成本为主 |
| 关键做法 | JIT准时制、VSM价值流图、看板拉动、零库存 |
| 典型行业 | 汽车制造、家电、快消品 |
| 风险 | 柔性不足，突发事件影响大 |

### 1.2 敏捷供应链（Agile）

| 维度 | 内容 |
|---|---|
| 核心目标 | 快速响应需求变化，高柔性运营 |
| 适用场景 | 需求波动大、产品定制化、竞争以速度为主 |
| 关键做法 | 延迟分化、模块化设计、安全库存、多源供应 |
| 典型行业 | 时尚服装、消费电子、定制化制造 |
| 风险 | 成本偏高，库存可能过剩 |

### 1.3 韧性供应链（Resilient）

| 维度 | 内容 |
|---|---|
| 核心目标 | 抵抗冲击和快速恢复，供应链安全 |
| 适用场景 | 地缘政治风险高、供应中断概率大、关键物资 |
| 关键做法 | 多源供应、安全库存、备选供应商、产能冗余、可替代物料 |
| 典型行业 | 半导体、通信设备、军工、医疗 |
| 风险 | 成本最高，效率牺牲 |

### 1.4 三种取向对比

| 维度 | 精益 | 敏捷 | 韧性 |
|---|---|---|---|
| 成本 | 最低 | 中等 | 最高 |
| 速度 | 中等 | 最快 | 中等 |
| 柔性 | 最低 | 最高 | 高 |
| 抗风险 | 最低 | 中等 | 最高 |
| 库存策略 | 零库存/低库存 | 安全库存 | 安全库存+冗余 |
| 供应商策略 | 最少供应商 | 适度多源 | 多源+备选 |

<!-- chunk: 供应链战略.二-华为供应链战略选择 -->
## 二、华为供应链战略选择

华为面对复杂的全球环境，选择**"精益+敏捷+韧性"的复合战略**：

| 战略维度 | 华为做法 |
|---|---|
| 精益 | 持续降本、库存周转优化、流程效率提升 |
| 敏捷 | CTO按订单配置、快速交付承诺、[[ISC]]+数字化 |
| 韧性 | 多源供应、备选物料、国产替代、海外多供应中心 |

## 三、供应链战略制定方法

### 3.1 基于产品类型的战略匹配（Marshall Fisher模型）

| 产品类型 | 特征 | 推荐供应链战略 |
|---|---|---|
| 功能型产品 | 需求稳定可预测、生命周期长、利润率低 | 精益供应链 |
| 创新型产品 | 需求不确定、生命周期短、利润率高 | 敏捷供应链 |

### 3.2 基于不确定性矩阵的战略选择

```
供应不确定性 ↑
高           │ 敏捷+韧性        │ 韧性
             │ (高需求不确定     │ (低需求不确定
             │  高供应不确定)    │  高供应不确定)
低           │ 精益+敏捷        │ 精益
             │ (高需求不确定     │ (低需求不确定
             │  低供应不确定)    │  低供应不确定)
             └──────────────────┴──────────→
              低                    高
                需求不确定性
```

### 3.3 战略制定四步法

1. **理解客户需求**：需求稳定性、定制化程度、交付期望
2. **理解供应链能力**：当前成本、柔性、响应速度
3. **识别差距**：客户需求与供应链能力的差距
4. **制定战略**：选择精益/敏捷/韧性的组合和投入重点

<!-- chunk: 供应链战略.四-供应链战略与业务战略的对齐 -->
## 四、供应链战略与业务战略的对齐

| 业务战略 | 供应链战略 | 重点投入 |
|---|---|---|
| 成本领先 | 精益优先 | 降本、流程效率、库存优化 |
| 差异化/快速响应 | 敏捷优先 | 柔性、速度、定制化能力 |
| 风险规避/安全优先 | 韧性优先 | 多源、安全库存、备选方案 |
| 综合竞争 | 精益+敏捷+韧性 | 分品类/分场景差异化策略 |

<!-- chunk: 供应链战略.五-供应链战略执行框架 -->
## 五、供应链战略执行框架

```
供应链战略
    │
    ├── 供应网络设计（供应商布局、产能布局、物流网络）
    ├── 流程能力建设（S&OP、计划、采购、制造、交付）
    ├── 组织能力建设（供应链组织、人才、考核）
    ├── 数字化能力（数据底座、算法模型、智能运营）
    └── 风险管理（供应风险、合规风险、安全风险）
```
