# 供应商管理诊断工具

<!-- chunk: 供应商管理诊断.成熟度评估 -->
## 成熟度评估

| 级别 | 特征 |
|---|---|
| L1 | 供应商随意选，靠价格，无评估，质量问题频发 |
| L2 | 有供应商名单，但评估不系统，依赖个人关系 |
| L3 | 供应商分级，年度评估，质量KPI有追踪 |
| L4 | 战略供应商深度合作，联合研发，供应链可视 |
| L5 | 供应链协同平台，实时数据共享，风险自动预警 |

<!-- chunk: 供应商管理诊断.快速诊断问题 -->
## 快速诊断问题

1. **「关键物料有几家供应商？单一供应商依赖风险有多大？」**
   - 单一供应商依赖 → 严重风险
   - 双供应商 → 华为标准最低要求
   - 多供应商竞争 → L3+

2. **「供应商质量问题，平均多久发现？处理周期多长？」**
   - 到了客户端才发现 → L1-L2
   - 在入库检验发现 → L2-L3
   - 在供应商端就控制 → L3+（SQE机制）

3. **「有没有供应商年度评级机制？差的供应商会被淘汰吗？」**
   - 没有 → L1-L2
   - 有评级但没有淘汰 → L2-L3
   - 有评级且有淘汰 → L3+

## 服务包

**供应商管理体系建立（2-3个月，15-25人天）**：
- 供应商分级与评估标准
- 关键物料双供应商策略
- 供应商质量管理（SQE）体系
- 年度供应商大会机制

**报价**：25-45万
