# 华为供应链管理

## 核心概念与定义

华为供应链管理的核心理念是[[ISC]]（Integrated Supply Chain，集成供应链），通过流程打通从客户需求到产品交付的全链条，实现"客户需求驱动的端到端供应链"。华为供应链从"被动响应采购"演进为"战略采购+集成供应链+全球化交付"体系，采购不是简单的杀价和谈判，而是具有战略纵深的供应链管理。

## 方法论框架

### 一、传统采购四大误区（华为已克服）

| 误区 | 正确认知 |
|------|---------|
| 采购=杀价，重谈判技巧 | 采购要有战略纵深，关注TCO（总拥有成本）而非单价 |
| 采购=收礼应酬 | 采购要抛弃私心杂念，聚焦技能提升 |
| 采购管理=频繁换人防腐败 | 建立制度防腐，而非靠换人 |
| 采购控制=急催交货慢付款 | 诚信合作，与供应商共赢 |

### 二、华为采购管理两层职责

**基础职责**：保证供应（5R——合适的质量/数量/价格/时间/地点）

**高级职责**：
1. 采购要有战略纵深，不只盯着供应商谈判
2. 科学性采购，提高计划性
3. 持续加强风险管理，保障供应安全
4. 与世界最优秀供应商建立战略合作伙伴关系
5. 工程采购人员必须具有工程项目经验
6. 采购人员深入现场了解业务
7. 聚焦技能提升，抛弃私心杂念
8. 战略预备队成立软性采购分队

### 三、华为供应链组织架构

| 层级 | 组织 | 职责 |
|------|------|------|
| 总部 | 供应链管理部 | 供应链战略、流程、政策制定 |
| 产品线 | 供应链团队 | 产品维度供应链管理 |
| 区域 | 供应链代表 | 本地化交付和采购 |

### 四、ISC集成供应链核心流程

客户需求→需求预测→生产计划→采购执行→制造→物流交付→客户签收

**关键集成点**：
- 市场计划（订货预测/要货预测）与供应链计划集成
- 采购与研发集成（器件选型前置）
- 交付与[[LTC]]销售流程集成
- 财务与供应链集成（四算：概算/预算/核算/决算）

### 五、供应商管理

| 类型 | 管理方式 | 占比 |
|------|---------|------|
| 战略供应商 | 长期合作、联合创新、深度绑定 | 少数核心 |
| 优选供应商 | 稳定合作、优先采购 | 主力 |
| 合格供应商 | 正常交易 | 补充 |
| 限制供应商 | 逐步退出 | 淘汰 |

### 六、华为供应链演进历程

| 阶段 | 时间 | 特点 |
|------|------|------|
| 初级采购 | 1990年代 | 分散采购，缺乏体系 |
| 集中采购 | 2000年代初 | 统一采购平台，降低成本 |
| [[ISC]]变革 | 2000年代中 | 引入IBM ISC流程，端到端集成 |
| 全球化供应链 | 2010年代 | 全球采购+本地交付+风险管控 |

## 关键流程/步骤

1. **需求到供应流程**：市场计划（滚动18个月预测）→ S&OP产销协同 → 生产计划 → 采购计划 → 执行
2. **采购执行流程**：需求确认 → 供应商选择/招标 → 采购订单 → 到货检验 → 入库 → 付款
3. **交付管理流程**：订单接收 → 产能确认 → 生产排程 → 制造 → 品检 → 发货 → 安装调测 → 客户验收
4. **供应商管理流程**：供应商认证 → 分级管理 → 绩效考核 → 战略协同/淘汰

## 核心原则

1. **战略采购而非交易采购**：采购要有战略纵深，关注TCO和供应链安全，而非单纯压价。

2. **计划是供应链的灵魂**：预测准确率每提高1%，库存/采购成本/交付率/资金效率各提升5%。前端市场计划的准确性直接决定后端供应链效率。

3. **端到端集成**：[[ISC]]的核心是集成——从前端市场计划到后端交付回款全链条打通，消除信息孤岛和部门墙。

4. **与最优秀供应商战略合作**：不追求最便宜而是最优性价比，与核心供应商深度绑定，共同创新。

## 典型案例与实践要点

**案例1：北电的教训**。北电在互联网泡沫期大量囤积40亿美元产品，IT泡沫破裂后崩盘。根因：投机心理导致需求预测严重偏差、计划风险控制不到位、销售计划与市场不同步。反面印证了供应链计划准确性和产销协同的重要性。

**案例2：华为的"软件包驱动倒推流程"**。华为[[ISC]]变革时，不是先设计流程再上线系统，而是先上线软件包（ERP），从软件包运行中倒推流程优化。这是一种务实的变革路径。

**实践要点**：
- 市场计划预测准确率：订货预测92%+，要货预测80%+
- 计划集成：行销市场计划、供应链生产计划、产品线EPU综合平衡三方联动
- 供应链全球化布局：采购中心、制造基地、物流枢纽全球分布
- 供应链安全：多源采购、战略备料、地缘政治风险预案

## 与华为其他体系的关联

- 供应链与[[LTC]]流程集成：销售订单触发供应链交付，合同条款影响交付成本
- 供应链与[[IPD]]集成：器件选型在产品开发阶段前置（采购早期参与研发）
- 供应链与财经集成：四算（概算/预算/核算/决算）贯穿供应链全流程
- 供应链与质量管理集成：来料检验/过程质量控制/交付质量闭环
