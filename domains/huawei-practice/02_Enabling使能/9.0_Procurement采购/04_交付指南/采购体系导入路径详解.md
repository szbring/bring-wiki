# 采购体系导入路径详解

采购体系导入需从事务级逐步提升到战略级，分阶段推进，确保业务连续性和变革可持续性。

---

<!-- chunk: 采购体系导入路径详解.一-导入总体路径 -->
## 一、导入总体路径

| 阶段 | 目标 | 周期 | 成熟度跃迁 |
|---|---|---|---|
| 阶段0 准备期 | 诊断+立项 | 2-4周 | - |
| 阶段1 规范期 | L2——流程规范 | 3-6月 | L1→L2 |
| 阶段2 战略期 | L3——品类战略驱动 | 6-9月 | L2→L3 |
| 阶段3 协同期 | L4——供应商深度协同 | 9-12月 | L3→L4 |
| 阶段4 价值期 | L5——采购驱动价值 | 12-18月 | L4→L5 |

---

<!-- chunk: 采购体系导入路径详解.二-各阶段关键动作 -->
## 二、各阶段关键动作

### 阶段1 规范期

| 动作 | 周期 |
|---|---|
| 采购流程标准化 | 2月 |
| 供应商评估体系建立 | 1月 |
| 合规控制点设置 | 1月 |
| SRM系统基础功能上线 | 3月 |
| 采购团队培训 | 1月 |

### 阶段2 战略期

| 动作 | 周期 |
|---|---|
| 支出分析和品类划分 | 2月 |
| Top5品类战略制定 | 3月 |
| 供应商分级管理实施 | 2月 |
| TCO分析工具导入 | 1月 |
| 采购早期介入研发试点 | 2月 |

### 阶段3 协同期

| 动作 | 周期 |
|---|---|
| VMI/JMI试点 | 3月 |
| 供应商协同平台深化 | 3月 |
| EPI采购早期介入全面推广 | 2月 |
| 战略供应商联合规划 | 2月 |
| P2P流程自动化 | 3月 |

### 阶段4 价值期

| 动作 | 周期 |
|---|---|
| 供应商联合创新 | 持续 |
| 全链成本优化 | 6月 |
| 采购智能化（AI辅助） | 6月 |
| 生态协同建设 | 持续 |

<!-- chunk: 采购体系导入路径详解.三-风险管理 -->
## 三、风险管理

| 风险 | 预防 | 应急 |
|---|---|---|
| 采购团队抵触 | 培训+激励+榜样 | 先易后难，用结果说话 |
| 供应商不配合 | 先推战略供应商 | 激励+替代 |
| 合规阻力 | 高层推动+制度保障 | 专项治理 |
| 系统落地难 | 选型慎重+分步实施 | 先Excel后系统 |
