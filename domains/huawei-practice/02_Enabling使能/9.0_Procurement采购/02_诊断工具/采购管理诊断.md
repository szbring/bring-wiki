# 采购管理诊断工具

<!-- chunk: 采购管理诊断.成熟度评估 -->
## 成熟度评估

| 级别 | 特征 |
|---|---|
| L1 | 采购靠个人关系，无流程，价格不透明，腐败风险高 |
| L2 | 有采购流程，但合规性弱，成本管控效果有限 |
| L3 | 分类管理，战略采购与日常采购分离，有招投标 |
| L4 | 总拥有成本（TCO）视角，数字化采购平台 |
| L5 | 战略采购深度参与产品设计，供应商联合创新 |

---

<!-- chunk: 采购管理诊断.快速诊断问题 -->
## 快速诊断问题

1. **「采购人员有没有参与产品研发早期的物料选型？」**
   - 没有，研发定了才来采购 → L1-L2，采购太晚介入
   - 有时参与 → L2-L3
   - 从设计阶段就参与（EI早期介入）→ L3+

2. **「大额采购有没有招投标流程？审批权限是否清晰？」**
   - 没有，领导拍板 → L1-L2
   - 有流程但执行不严 → L2
   - 流程严格，权限清晰 → L3+

3. **「采购成本节约的KPI是多少？去年节约了多少？」**
   - 没有节约KPI → L1-L2
   - 有KPI但难量化 → L2-L3
   - 有明确数据 → L3+

## 服务包

**[[华为的流程管理]]规范化（2-3个月，15-20人天）**：
- 采购分类管理（战略/杠杆/瓶颈/常规）
- 采购审批权限矩阵
- 招投标流程设计
- 采购绩效考核体系

**报价**：20-40万
