# 采购成熟度评估

采购成熟度评估帮助企业定位[[华为的流程管理]]水平，识别差距，制定提升路径。

---

<!-- chunk: 采购成熟度评估.一-五级成熟度模型 -->
## 一、五级成熟度模型

| 级别 | 名称 | 核心特征 | 典型表现 |
|---|---|---|---|
| L1 | 事务级 | 采购是被动执行的后勤职能 | 下单/催货/付款，无品类策略 |
| L2 | 规范级 | 有基本流程和合规控制 | 有供应商评估，但以价格为主 |
| L3 | 战略级 | 品类战略驱动采购 | 有品类策略，供应商分级管理 |
| L4 | 协同级 | 采购与供应商深度协同 | VMI/JMI，供应商早期介入 |
| L5 | 价值级 | 采购驱动价值创造 | 联合创新，生态协同，数字化智能 |

---

<!-- chunk: 采购成熟度评估.二-七维度评估 -->
## 二、七维度评估

| 维度 | L1(1分) | L2(2分) | L3(3分) | L4(4分) | L5(5分) |
|---|---|---|---|---|---|
| 采购战略 | 无 | 有降本目标 | 有品类战略 | 采购战略对齐业务 | 采购驱动业务 |
| 供应商管理 | 随机选择 | 有评估 | 分级分类 | 深度协同 | 生态共建 |
| 成本管理 | 只看单价 | 有比价 | TCO分析 | 价值工程 | 全链成本优化 |
| 流程合规 | 无标准 | 有基本流程 | 流程标准化 | 流程数字化 | 智能合规 |
| 需求管理 | 被动响应 | 有需求审核 | 需求预测 | EPI早期介入 | 联合需求规划 |
| 风险管理 | 不识别 | 有基本风控 | 风险登记册 | 风险量化评估 | 预测性风控 |
| 数字化 | 手工/Excel | 有部分系统 | SRM/e采购 | P2P自动化 | AI辅助决策 |

**总分：__/35分**

| 总分 | 成熟度级别 |
|---|---|
| 7-10分 | L1 事务级 |
| 11-17分 | L2 规范级 |
| 18-24分 | L3 战略级 |
| 25-31分 | L4 协同级 |
| 32-35分 | L5 价值级 |

<!-- chunk: 采购成熟度评估.三-快速诊断 -->
## 三、快速诊断

### 「采购主要看什么？」

| 回答 | 成熟度 |
|---|---|
| 谁便宜买谁的 | L1 |
| 比价选最便宜的 | L1-L2 |
| 综合评估价格+质量+交付 | L3 |
| TCO总成本最优 | L4 |
| 采购驱动业务创新 | L5 |

### 「有多少供应商？怎么管的？」

| 回答 | 成熟度 |
|---|---|
| 很多，没数过 | L1 |
| 有数，但没分类 | L2 |
| 分了战略/优选/合格 | L3 |
| 有协同机制，VMI | L4 |
| 生态共建，联合创新 | L5 |
