# 战略采购方法论

战略采购超越了传统的事务性采购，将采购提升到战略层面，通过品类战略、供应商战略和TCO分析，实现采购从"成本中心"到"价值创造"的转型。

---

<!-- chunk: 战略采购方法论.一-战略采购-vs-传统采购 -->
## 一、战略采购 vs 传统采购

| 维度 | 传统采购 | 战略采购 |
|---|---|---|
| 目标 | 最低单价 | TCO总拥有成本最优 |
| 供应商关系 | 短期交易 | 长期战略合作 |
| 范围 | 执行采购订单 | 品类规划→寻源→执行→评估 |
| 决策依据 | 报价比较 | 供应市场分析+品类战略 |
| 采购介入时机 | 需求确认后 | 研发阶段早期介入 |
| 价值定位 | 降本 | 降本+保供+创新+风控 |

## 二、品类战略

### 2.1 品类定义

品类是一组具有相似供应市场特征和业务需求的物料/服务集合。品类管理是战略采购的核心方法。

### 2.2 克拉利奇矩阵（Kraljic Matrix）

根据供应风险和利润影响两个维度，将采购品类分为四类：

```
利润影响(支出/价值) ↑
高           │ 战略型         │ 瓶颈型
             │ Strategic      │ Bottleneck
             │                │
             │ 合作/联盟       │ 保障供应安全
             │ 联合创新        │ 发展备选供应
             │                │
低           │ 杠杆型         │ 常规型
             │ Leverage       │ Non-critical
             │                │
             │ 竞争招标        │ 简化/自动化
             │ 最大化议价      │ 降低管理成本
             └────────────────┴──────────→
              低                    高
                供应风险/稀缺性
```

### 2.3 品类战略制定五步法

| 步骤 | 内容 | 工具/方法 |
|---|---|---|
| 1. 支出分析 | 了解钱花在哪里 | 支出分析、帕累托分析 |
| 2. 供应市场分析 | 了解市场格局和趋势 | 波特五力、PESTEL |
| 3. 需求分析 | 了解内部需求特征 | 需求预测、规格分析 |
| 4. 战略制定 | 确定采购策略 | 克拉利奇矩阵、SWOT |
| 5. 行动计划 | 制定具体执行方案 | 路线图、里程碑 |

## 三、供应商战略

### 3.1 供应商分级分类

| 级别 | 定义 | 份额 | 管理策略 |
|---|---|---|---|
| 战略供应商 | 供应关键物料，替代困难 | 60-70% | 战略合作，联合创新 |
| 优选供应商 | 供应重要物料，质量稳定 | 20-30% | 稳定份额，持续改进 |
| 合格供应商 | 供应一般物料，满足要求 | 10-20% | 限定份额，定期评估 |
| 备选供应商 | 潜在替代，应急备用 | 0-5% | 认证维护，应急启用 |

### 3.2 华为CEG物料专家团

华为采购的核心组织是CEG（Commodity Expert Group，物料专家团），负责品类策略和供应商选择：

| CEG职责 | 内容 |
|---|---|
| 品类战略 | 制定品类采购策略和供应商战略 |
| 供应商选择 | 组织供应商认证和选择 |
| 价格管理 | 价格谈判和成本分析 |
| 供应商绩效 | 定期评估供应商表现 |
| 技术融合 | 推动供应商早期参与研发 |

### 3.3 华为供应商管理三阶九步法

| 阶段 | 步骤 | 内容 |
|---|---|---|
| 第一阶：供应商选择与认证 | 1. 供应商资质认证和考察 | 现场考察、资质审核 |
| | 2. 供应商物料族认证与测试 | 样品测试、小批量验证 |
| | 3. 比价/谈判/合同签订 | 招标/谈判、合同签署 |
| 第二阶：采购执行 | 4. 具体物料询报价 | RFQ/RFP流程 |
| | 5. 采购订单执行和管理 | 订单下发、跟踪、催交 |
| | 6. 物料交付与验收付款 | 来料检验、入库、付款 |
| 第三阶：供应商绩效考核 | 7. 供应商绩效评价 | QCDST多维度评分 |
| | 8. 供应商反馈和沟通 | 绩效反馈、改进计划 |
| | 9. 供应商评估管理 | 升级/降级/淘汰决策 |

## 四、TCO总拥有成本

### 4.1 TCO定义

TCO（Total Cost of Ownership）是采购品从获取到处置全生命周期发生的所有成本之和，远超采购单价。

### 4.2 TCO成本构成

| 成本类别 | 具体项 | 占比参考 |
|---|---|---|
| 获取成本 | 采购单价+运输+关税+保险 | 60-80% |
| 使用成本 | 安装+培训+运营+耗材 | 10-20% |
| 维护成本 | 维修+备件+停机损失 | 5-15% |
| 处置成本 | 回收+环保+退役 | 1-5% |
| 风险成本 | 质量缺陷+延迟+单一来源溢价 | 变动 |

### 4.3 TCO分析应用

| 场景 | TCO分析价值 |
|---|---|
| 供应商选择 | 不看单价看总成本 |
| 国产替代 | 评估国产化的综合成本 |
| 自制vs外包 | 全成本比较决策 |
| 投资决策 | 设备采购的全生命周期成本 |

## 五、采购早期介入研发（EPI）

华为要求采购在研发阶段早期介入，确保设计阶段就考虑可采购性和成本：

| 研发阶段 | 采购介入内容 |
|---|---|
| 概念阶段 | 供应市场调研、物料选型建议 |
| 计划阶段 | BOM成本估算、供应商预选 |
| 开发阶段 | 物料认证、供应商技术对接 |
| 验证阶段 | 小批量验证、供应方案确认 |
| 发布阶段 | 量产供应准备、备选方案 |
