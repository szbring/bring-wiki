# 采购合规与风控

采购是企业腐败风险最高的领域之一。华为建立了严格的采购合规与风控体系，确保采购活动阳光透明、风险可控。

---

<!-- chunk: 采购合规与风控.一-采购合规体系 -->
## 一、采购合规体系

### 1.1 三道防线

| 防线 | 责任方 | 职责 |
|---|---|---|
| 第一道 | 采购业务部门 | 流程执行中的自我控制 |
| 第二道 | 合规/风控部门 | 政策制定、监督检查 |
| 第三道 | 内部审计 | 独立审计和评估 |

### 1.2 核心合规原则

| 原则 | 内容 | 具体要求 |
|---|---|---|
| 阳光采购 | 采购过程公开透明 | 招标信息公开、评标标准透明 |
| 公平竞争 | 给所有供应商平等机会 | 不设歧视性条件、不偏向特定供应商 |
| 利益回避 | 避免利益冲突 | 采购人员与供应商有利害关系须申报回避 |
| 采购集中 | 采购是唯一授权的对外采购组织 | 禁止绕过采购部的私下采购 |
| 诚信廉洁 | 禁止一切商业贿赂 | 零容忍腐败，违者严惩 |

<!-- chunk: 采购合规与风控.二-反腐败机制 -->
## 二、反腐败机制

### 2.1 华为供应商反馈办公室

华为设立供应商反馈办公室，处理供应商对华为员工不公平行为和不道德行为的投诉，帮助华为遵守诚信承诺。

### 2.2 反腐关键措施

| 措施 | 内容 |
|---|---|
| 利益冲突申报 | 采购人员须申报与供应商的关系 |
| 轮岗制度 | 采购关键岗位定期轮岗 |
| 供应商多元 | 避免单一来源，保持竞争 |
| 价格对标 | 定期对标市场价格，识别异常 |
| 投诉渠道 | 供应商反馈办公室+内部举报渠道 |
| 审计检查 | 定期审计采购项目 |

### 2.3 采购人员行为准则

| 禁止行为 | 说明 |
|---|---|
| 索取/收受回扣 | 任何形式的回扣和佣金 |
| 收受礼品/宴请 | 超出正常商务礼仪范围 |
| 泄露采购信息 | 向供应商透露竞争对手报价 |
| 指定供应商 | 无合理理由指定特定供应商 |
| 绕过流程 | 不经正式流程私下采购 |
| 虚构采购 | 虚假采购套取资金 |

<!-- chunk: 采购合规与风控.三-供应商合规 -->
## 三、供应商合规

### 3.1 供应商合规要求

| 合规领域 | 要求 |
|---|---|
| 商业道德 | 反贿赂、反腐败、公平竞争 |
| 劳工权益 | 禁止童工、保障工时和薪酬 |
| 环境保护 | 环保合规、有害物质管控 |
| 信息安全 | 商业秘密保护、数据安全 |
| 出口管制 | 遵守出口管制法规 |

### 3.2 供应商合规审查

| 审查类型 | 频次 | 内容 |
|---|---|---|
| 入场审查 | 新供应商认证 | 全面合规评估 |
| 年度审查 | 每年 | CSR审查+合规自评 |
| 专项审查 | 触发式 | 涉嫌违规时专项调查 |
| 现场审计 | 按风险等级 | 高风险供应商现场审计 |

<!-- chunk: 采购合规与风控.四-采购风险分类与应对 -->
## 四、采购风险分类与应对

| 风险类别 | 具体风险 | 应对措施 |
|---|---|---|
| 合规风险 | 腐败/串标/利益冲突 | 三道防线+审计+投诉机制 |
| 供应风险 | 单一来源/断供/质量 | 多源供应+安全库存+备选方案 |
| 价格风险 | 价格波动/汇率/通胀 | 长协+套期保值+价格调整条款 |
| 合同风险 | 条款不利/法律纠纷 | 法务审核+标准合同模板 |
| 质量风险 | 来料不良/批次问题 | 来料检验+供应商质量体系审核 |
| 信息风险 | 数据泄露/商业秘密 | 保密协议+权限管控 |

<!-- chunk: 采购合规与风控.五-审计机制 -->
## 五、审计机制

| 审计类型 | 频次 | 内容 |
|---|---|---|
| 采购流程审计 | 每年 | 端到端流程合规性 |
| 供应商选择审计 | 每季度 | 招标/谈判过程合规性 |
| 价格审计 | 每半年 | 价格合理性分析 |
| 合同审计 | 每年 | 合同执行和条款合规 |
| 专项审计 | 触发式 | 投诉/异常触发的专项调查 |
