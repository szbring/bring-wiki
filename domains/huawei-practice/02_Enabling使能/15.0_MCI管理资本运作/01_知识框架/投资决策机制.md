# 投资决策机制

投资决策机制是MCI流程的核心，确保投资决策科学、规范、风险可控。华为的投资决策遵循"业务发起→专业评估→集体决策"的模式。

---

<!-- chunk: 投资决策机制.一-投资决策流程 -->
## 一、投资决策流程

```
业务需求识别 → 项目立项 → 尽职调查 → 投资方案设计 → 投资委员会评审 → 董事会决策 → 投资执行
     │              │           │              │               │              │
   业务部门      投资团队    外部顾问      投资团队       专业委员会      决策层
```

---

<!-- chunk: 投资决策机制.二-投资委员会 -->
## 二、投资委员会

### 2.1 组织架构

| 层级 | 机构 | 职责 |
|---|---|---|
| 决策层 | 董事会/常务董事会 | 最终投资决策 |
| 评审层 | 投资评审委员会 | 专业评审和风险把关 |
| 执行层 | 企业发展部/哈勃投资 | 项目执行和管理 |

### 2.2 投资评审委员会

| 角色 | 来源 | 职责 |
|---|---|---|
| 主任 | 财经委员会 | 主持评审 |
| 业务代表 | 业务部门 | 评估业务协同价值 |
| 技术专家 | 技术部门 | 评估技术价值和可行性 |
| 财务专家 | 财经部门 | 评估财务回报和风险 |
| 法务专家 | 法务部门 | 评估法律合规风险 |
| 风控专家 | 风控部门 | 评估整体风险 |

<!-- chunk: 投资决策机制.三-尽职调查 -->
## 三、尽职调查

### 3.1 尽调维度

| 维度 | 内容 | 关注点 |
|---|---|---|
| 业务尽调 | 业务模式、市场地位、竞争力 | 市场规模、竞争格局、客户集中度 |
| 财务尽调 | 财务状况、盈利能力、现金流 | 收入真实性、关联交易、或有负债 |
| 技术尽调 | 技术能力、知识产权、研发实力 | 专利质量、技术壁垒、研发团队 |
| 法务尽调 | 法律合规、诉讼、合同 | 股权清晰、合规经营、重大合同 |
| 人力尽调 | 核心团队、组织、文化 | 关键人依赖、团队稳定性、薪酬体系 |

### 3.2 尽调流程

```
信息收集 → 现场调研 → 访谈验证 → 问题清单 → 风险评估 → 尽调报告
```

### 3.3 尽调关键原则

| 原则 | 说明 |
|---|---|
| 独立性 | 尽调团队独立于业务发起方 |
| 全面性 | 覆盖业务/财务/技术/法务/人力各维度 |
| 交叉验证 | 多渠道获取信息，交叉核实 |
| 问题导向 | 重点关注风险和问题，而非验证预期 |

<!-- chunk: 投资决策机制.四-投资决策标准 -->
## 四、投资决策标准

### 4.1 战略匹配度

| 维度 | 评分（1-5分） |
|---|---|
| 与主航道相关性 | 1=无关 → 5=核心 |
| 技术协同价值 | 1=无 → 5=强协同 |
| 供应链保障价值 | 1=无 → 5=关键保障 |
| 市场拓展价值 | 1=无 → 5=新市场入口 |

### 4.2 财务回报

| 指标 | 要求 |
|---|---|
| IRR（内部收益率） | ≥ 基准收益率 |
| NPV（净现值） | > 0 |
| 投资回收期 | ≤ 可接受年限 |
| TCO/ROI | 优于替代方案 |

### 4.3 风险评估

| 风险类别 | 评估内容 |
|---|---|
| 市场风险 | 市场规模是否达预期、竞争格局变化 |
| 技术风险 | 技术是否可行、是否被替代 |
| 整合风险 | 能否成功整合、文化是否兼容 |
| 退出风险 | 退出渠道是否通畅、退出价值是否可保 |
| 合规风险 | 是否存在法律合规问题 |

## 五、投资决策机制优化

| 优化方向 | 具体措施 |
|---|---|
| 专业化 | 建立专家库及专家评审机制 |
| 数字化 | 信息化固化风控合规要求，关键控制自动化 |
| 一体化 | 投前投后一体化，产投协同落实 |
| 标准化 | 行业尽调模板、风险评分卡、决策标准统一 |
