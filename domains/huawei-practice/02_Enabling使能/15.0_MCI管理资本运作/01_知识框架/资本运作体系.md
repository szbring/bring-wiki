# 资本运作体系

华为MCI（Manage Capital Investment，管理资本运作）流程覆盖投资、融资、并购、重组四种资本运作模式，是华为财经体系的重要组成部分。

---

<!-- chunk: 资本运作体系.一-四种资本运作模式 -->
## 一、四种资本运作模式

| 模式 | 定义 | 目的 | 华为案例 |
|---|---|---|---|
| 投资 | 以获取收益或战略协同为目的的资金投放 | 技术获取、供应链保障、战略布局 | 哈勃投资半导体产业链 |
| 融资 | 通过借贷或股权方式获取资金 | 业务扩张、补充流动性、优化资本结构 | 员工持股、债券发行 |
| 并购 | 通过收购股权或资产获得企业控制权 | 快速获取技术/市场/人才 | 收购Neul（物联网） |
| 重组 | 对资产、业务、股权结构进行调整 | 提升效率、聚焦主业、剥离非核心 | 荣耀品牌独立 |

## 二、华为资本运作两个阶段

### 2.1 第一阶段（2000-2016）：保守低调

| 特征 | 内容 |
|---|---|
| 主导部门 | 企业发展部（隶属财经委员会） |
| 投资发起 | 业务部门发现技术需求→提报企业发展部→评估→常务董事会决策 |
| 投资风格 | 产品投资为主，小金额并购获取关键技术 |
| 投资方向 | 通信相关技术，少量互联网财务投资 |
| 典型案例 | 收购Neul（物联网）、投资XMOS（芯片设计） |

### 5.2 第二阶段（2019至今）：战略激进

| 特征 | 内容 |
|---|---|
| 主导机构 | 哈勃投资（专门子公司） |
| 投资节奏 | 26个月布局37家公司，月均1-2个项目 |
| 投资风格 | 战略投资+VC投资并重 |
| 投资方向 | 半导体全产业链（材料/芯片/EDA/射频/AI） |
| 投资目的 | 供应链国产替代、扶持国内半导体企业 |

## 三、《华为基本法》投资原则

> "我们中短期的投资战略仍坚持产品投资为主，以期最大限度地集中资源，迅速增强公司的技术实力、市场地位和管理能力……我们不从事任何分散公司资源和高层管理精力的非相关多元化经营。"

### 3.1 三条投资原则

| 原则 | 内容 |
|---|---|
| 产品投资为主 | 围绕主航道，不搞非相关多元化 |
| 集中资源 | 不分散公司资源和高管精力 |
| 战略导向 | 投资服务战略，不是财务投机 |

<!-- chunk: 资本运作体系.四-资本运作治理架构 -->
## 四、资本运作治理架构

```
股东会
  │
  ├── 董事会（BOD）── 战略与经营决策
  │     │
  │     ├── 常务董事会 ── 重大投资决策
  │     │
  │     └── 财经委员会（FC）── 投资预算与管控
  │           │
  │           └── 企业发展部/哈勃投资 ── 投资执行
  │
  └── 监事会 ── 监督
```

| 治理层级 | 职责 |
|---|---|
| 股东会 | 重大资本运作审批 |
| 董事会/常务董事会 | 投资决策、预算审批 |
| 财经委员会 | 投资预算管理、风险管控 |
| 企业发展部/哈勃投资 | 项目发掘、尽职调查、投后管理 |

<!-- chunk: 资本运作体系.五-资本运作与全面预算的关系 -->
## 五、资本运作与全面预算的关系

华为全面预算包含投资/筹资预算，资本运作纳入全面预算管理：

| 预算类别 | 内容 |
|---|---|
| 经营预算 | 日常经营收支预算 |
| 战略专项 | 战略投资项目预算 |
| 投资/筹资预算 | 投资支出、融资计划、资本结构 |
| 集团财务预算 | 三大报表、税务预算等 |
