# 资本运作成熟度评估

[[华为的经营管理]]成熟度评估帮助企业定位投资管理水平，识别差距，制定提升路径。

---

<!-- chunk: 资本运作成熟度评估.一-五级成熟度模型 -->
## 一、五级成熟度模型

| 级别 | 名称 | 核心特征 | 典型表现 |
|---|---|---|---|
| L1 | 随机级 | 投资无体系，凭感觉决策 | 投资靠老板直觉，无流程无标准 |
| L2 | 规范级 | 有基本流程和审批 | 有投资审批流程，但尽调不充分 |
| L3 | 体系级 | 有完整投资管理体系 | 投委会+尽调+投后管理形成闭环 |
| L4 | 协同级 | 投前投后一体化，产投协同 | 投资与业务深度协同，赋能式投后 |
| L5 | 生态级 | 生态化投资，战略驱动 | 投资驱动产业生态，价值链整合 |

---

<!-- chunk: 资本运作成熟度评估.二-七维度评估 -->
## 二、七维度评估

| 维度 | L1(1分) | L2(2分) | L3(3分) | L4(4分) | L5(5分) |
|---|---|---|---|---|---|
| 投资战略 | 无 | 有方向无策略 | 有投资策略 | 投资战略对齐业务 | 投资驱动生态 |
| 决策机制 | 个人决策 | 有审批流程 | 投委会评审 | 专业化决策+数字化 | AI辅助决策 |
| 尽职调查 | 无或粗略 | 有基本尽调 | 五维度尽调 | 标准化尽调模板 | 智能尽调工具 |
| 投后管理 | 无 | 偶尔关注 | 定期报告+监控 | 赋能式投后 | 生态化投后 |
| 退出管理 | 不考虑 | 事后考虑 | 有退出预案 | 主动管理退出 | 退出策略前置 |
| 风险管理 | 不识别 | 事后应对 | 风险登记册 | 风险量化评估 | 预测性风控 |
| 数字化 | 无 | Excel | 部分系统 | 投资管理平台 | 智能投资系统 |

**总分：__/35分**

| 总分 | 成熟度级别 |
|---|---|
| 7-10分 | L1 随机级 |
| 11-17分 | L2 规范级 |
| 18-24分 | L3 体系级 |
| 25-31分 | L4 协同级 |
| 32-35分 | L5 生态级 |

<!-- chunk: 资本运作成熟度评估.三-快速诊断 -->
## 三、快速诊断

### 「公司怎么做投资决策？」

| 回答 | 成熟度 |
|---|---|
| 老板拍板 | L1 |
| 开会讨论一下 | L2 |
| 有投委会评审流程 | L3 |
| 专业化评审+数字化支撑 | L4-L5 |

### 「投后有人管吗？」

| 回答 | 成熟度 |
|---|---|
| 没有/不太关注 | L1-L2 |
| 定期看报告 | L3 |
| 有专人负责赋能 | L4 |
| 生态化运营 | L5 |
