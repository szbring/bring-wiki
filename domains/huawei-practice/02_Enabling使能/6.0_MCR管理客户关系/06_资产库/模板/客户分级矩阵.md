# 客户分级矩阵

客户分级矩阵是MCR流程的基础工具，用于对企业所有客户进行系统化分级，确保资源投入与客户价值匹配。

---

<!-- chunk: 客户分级矩阵.一-使用说明 -->
## 一、使用说明

1. 根据企业实际情况调整各项阈值
2. 先按B-RFM评分，再叠加战略价值和成长价值
3. 综合评分后确定SABC级别
4. 每季度评审更新

## 三、B-RFM评分

| 客户 | 最近合作时间 | B-R评分 | 年项目数 | B-F评分 | 年营收贡献 | B-M评分 | B-RFM总分 |

|  |  | /5 |  | /5 |  | /5 | /15 |

### B-Recency评分标准
- 5分：3个月内 → 4分：3-6月 → 3分：6-12月 → 2分：1-2年 → 1分：>2年

### B-Frequency评分标准
- 5分：≥5个项目/年 → 4分：3-4个 → 3分：2个 → 2分：1个 → 1分：0个

### B-Monetary评分标准
- 5分：>500万 → 4分：200-500万 → 3分：100-200万 → 2分：50-100万 → 1分：<50万

<!-- chunk: 客户分级矩阵.六-综合评级 -->
## 六、综合评级

| 客户 | 当前价值(40%) | 战略价值(30%) | 成长价值(30%) | 综合得分 | 客户级别 |
|---|---|---|---|---|---|
|  |  |  |  | /100 | S/A/B/C |

### 综合评级标准
- S类：80-100分 — 战略客户，最高优先
- A类：60-79分 — 伙伴客户，重点投入
- B类：40-59分 — 普通客户，标准服务
- C类：<40分 — 长尾客户，低成本运营
