# 客户关系诊断工具

<!-- chunk: 客户关系诊断工具.成熟度评估 -->
## 成熟度评估

| 级别 | 特征 |
|---|---|
| L1 | 客户关系完全依赖个人销售，公司没有机构化关系 |
| L2 | 有CRM录入，但客户分级不清，资源投入无差异化 |
| L3 | 有客户分级，大客户有专人负责，定期回访机制 |
| L4 | QBR机制运转，客户健康度可量化，机构化关系多层覆盖 |
| L5 | 战略合作伙伴关系，共同参与客户战略规划 |

---

<!-- chunk: 客户关系诊断工具.快速诊断问题 -->
## 快速诊断问题

1. **「你们Top10客户，如果主要负责销售离职，关系会断吗？」**
   - 肯定会断 → L1，严重依赖个人
   - 可能会有影响 → L2
   - 基本不受影响 → L3+，机构化关系

2. **「大客户和普通客户，你们服务投入有差异化吗？差异体现在哪里？」**
   - 基本无差异 → L1-L2，缺客户分级
   - 有差异但不系统 → L2-L3
   - 有明确分级和差异化服务标准 → L3+

3. **「有没有定期和大客户高层做QBR（季度业务回顾）？」**
   - 没有 → L2以下
   - 偶尔有但不定期 → L2-L3
   - 每季度固定进行 → L3+

4. **「你们最近流失的大客户，流失的主要原因是什么？」**
   - 关系疏远/竞品挖走 → [[以客户为中心]]维护不足
   - 产品/服务问题 → 转[[ITR]]
   - 价格 → 需要评估价值传递

<!-- chunk: 客户关系诊断工具.访谈问题库 -->
## 访谈问题库

### 销售/客户成功负责人
- 客户如何分级？分级标准是什么？
- Top客户有没有专属的客户经理或团队？
- 多久与大客户高层见一次面？有没有固定机制？
- 客户健康度如何评估？有没有预警机制？

### CEO/高管
- 你们认为最重要的Top5客户是哪些？为什么？
- 这些客户如果流失，对公司影响有多大？
- 和这些客户的关系，有没有超出业务层面的战略合作？
