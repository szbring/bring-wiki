# 企业价值创造——立体分层的客户关系管理

在华为征战市场的35年间，无论是国内还是海外，客户关系管理都为企业的发展提供了强大的支撑，也是企业发展的核心命脉。

第二阶段“狼狈组织”・团队类型客户关系・与客户多个部门对接团队关系・部分关键客户关系做地级市时，此时，交换机产品开始出现同质化竞争，市场供给已大于需求。

第三阶段“销服融合”・团队类型客户关系・与客户业务部门全面对接・客户关系建设存在多接口，不统一从点状客户关系转变到做线和面的客户关系，需要更高的理论指导。

华为的解决方案就是“同质化产品+差异化小创新”，小创新是比如工程创新、软件服务能力等。

解决方案差异化的前提是理解客户的需求和痛点，让客户真正感觉到你在为他做贴身服务。

第四阶段“铁三角”・组织型客户关系・与客户业务部门全面对接・客户关系建设统一接口・存在不同层面客户关系建设面对市场变化，华为通过构建集团和省分公司的组织客户关系，在一些创新领域做长期嵌入式服务，与客户形成“你中有我，我中有你”的关系。

2008年，华为启动了LTC变革，对MCR流程、客户关系管理的系统化进一步增强。

从华为的客户关系发展历程可以看出，华为从构建个人客户关系逐渐转变为构建分层分级的客户关系，打造了完整立体的客户关系管理体系。

这三个层面客户关系的建立，是为公司的市场目标、销售目标和竞争目标服务的。

普遍客户关系：客户相关业务部门的客户关系，满足均衡、稳定的客户关系目标。
