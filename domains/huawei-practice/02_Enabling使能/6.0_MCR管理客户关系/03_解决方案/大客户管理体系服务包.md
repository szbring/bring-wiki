# 大客户管理体系服务包

大客户管理体系服务包是薄云咨询基于华为MCR方法论，面向B2B企业提供的体系化解决方案，帮助企业从"个人英雄式销售"转型为"组织化[[以客户为中心]]"。

---

<!-- chunk: 大客户管理体系服务包.一-服务包总览 -->
## 一、服务包总览

| 模块 | 内容 | 周期 | 交付物 |
|---|---|---|---|
| 模块一 | 客户关系诊断与规划 | 4周 | 诊断报告+提升规划 |
| 模块二 | 客户分级与差异化策略 | 4周 | 客户分级标准+服务策略 |
| 模块三 | KAM体系设计 | 6周 | 组织+流程+工具全套方案 |
| 模块四 | CRM策略与落地 | 8周 | CRM选型+实施+培训 |
| 模块五 | 持续陪跑 | 12周 | 季度回顾+优化迭代 |

---

<!-- chunk: 大客户管理体系服务包.二-模块一-客户关系诊断与规划 -->
## 二、模块一：客户关系诊断与规划

### 2.1 诊断内容

| 诊断项 | 方法 | 输出 |
|---|---|---|
| 成熟度评估 | 八维度评估模型 | 成熟度等级+差距分析 |
| 客户价值评估 | B-RFM+战略+成长三维模型 | 客户分级建议 |
| 关系现状盘点 | 客户关系地图+决策链分析 | 关系薄弱点清单 |
| 痛点与需求 | 高管访谈+销售调研 | 痛点清单+需求优先级 |

### 2.2 诊断交付物

1. 客户关系成熟度评估报告
2. 客户价值评估与分级建议
3. 关系现状盘点报告
4. 提升路径规划（12个月）

---

<!-- chunk: 大客户管理体系服务包.三-模块二-客户分级与差异化策略 -->
## 三、模块二：客户分级与差异化策略

### 3.1 核心内容

| 内容 | 说明 |
|---|---|
| 分级标准制定 | 结合行业特点，制定SABC分级标准 |
| 客户盘点分级 | 对现有客户进行分级 |
| 差异化服务策略 | 各级别客户的服务投入标准 |
| 接触频率规范 | 各级别客户的拜访/沟通频次 |
| 资源分配规则 | 人力资源、预算分配规则 |

### 3.2 差异化服务矩阵

| 服务项 | S类 | A类 | B类 | C类 |
|---|---|---|---|---|
| 客户经理 | 专职 | 专职 | 兼职(2-3家) | 区域覆盖 |
| 高层拜访 | 月度 | 季度 | 半年 | 无 |
| QBR | 每季 | 每季 | 每半年 | 无 |
| 技术交流 | 月度 | 季度 | 按需 | 按需 |
| 方案定制 | 深度定制 | 适度定制 | 标准方案 | 标准方案 |
| 响应时效 | 2小时 | 4小时 | 24小时 | 48小时 |

## 四、模块三：KAM体系设计

### 4.1 组织设计

| 设计内容 | 交付物 |
|---|---|
| KAM组织架构 | 组织架构图+岗位说明书 |
| 铁三角角色定义 | AR/SR/FR职责+考核方案 |
| 高层Sponsor机制 | Sponsor名单+职责+考核 |
| 客户团队配置标准 | 各级别客户团队配置方案 |

### 4.2 流程设计

| 流程 | 核心环节 | 交付物 |
|---|---|---|
| 客户关系规划 | 洞察→评估→目标→策略→执行 | ABP模板+规划流程 |
| 客户接触管理 | 计划→执行→记录→分析 | 拜访流程+记录模板 |
| QBR运营 | 准备→回顾→改进 | QBR流程+议程模板 |
| 客户健康度监测 | 评分→预警→行动 | 评分模型+预警规则 |

### 4.3 工具设计

| 工具 | 用途 | 交付物 |
|---|---|---|
| 客户关系地图 | 可视化决策链和关系状态 | 模板+填写指南 |
| 九宫格评估 | 关键客户关系评估 | 模板+评估标准 |
| ABP模板 | 战略客户经营计划 | 模板+填写指南 |
| 客户健康度仪表盘 | 客户关系实时监测 | 评分卡+可视化方案 |

## 五、模块四：CRM策略与落地

### 5.1 服务内容

| 阶段 | 内容 | 周期 |
|---|---|---|
| CRM策略 | 需求分析、选型建议、实施规划 | 2周 |
| 系统实施 | 配置、数据迁移、集成、测试 | 4周 |
| 培训推广 | 分角色培训、试点运行、全面推广 | 2周 |

### 5.2 CRM功能规划

| 功能模块 | 优先级 | 实施阶段 |

| 拜访管理+合同管理 | P1 | 第一期 |
| 客户关系地图+健康度 | P1 | 第二期 |
| Pipeline分析+预警 | P2 | 第二期 |
| AI辅助洞察+推荐 | P3 | 第三期 |

<!-- chunk: 大客户管理体系服务包.六-模块五-持续陪跑 -->
## 六、模块五：持续陪跑

### 6.1 陪跑内容

| 阶段 | 内容 | 频次 |
|---|---|---|
| 第1-4周 | 跟踪落地情况，解答问题 | 每周1次 |
| 第5-8周 | 优化流程，调整方案 | 每两周1次 |
| 第9-12周 | 评估效果，固化成果 | 每月1次 |

### 6.2 评估指标

| 指标 | 基线 | 目标 |
|---|---|---|
| 成熟度等级 | 诊断时等级 | 提升1-2级 |
| S/A类客户关系覆盖率 | 基线值 | 提升30% |
| 大客户续约率 | 基线值 | 提升15% |
| CRM使用率 | 0% | >80% |
| QBR执行率 | 0% | >90% |
