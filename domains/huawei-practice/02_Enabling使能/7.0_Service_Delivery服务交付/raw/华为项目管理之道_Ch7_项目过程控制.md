# 项目控制：结果重要过程也重要

## 7.1 过程控制的理念

> 结果重要，过程也重要。好的过程不一定保证好的结果，但坏的过程一定导致坏的结果。

## 7.2 过程控制方法

### 挣值管理（EVM）

| 指标 | 公式 | 含义 |
|------|------|------|
| PV | 计划价值 | 计划完成的工作价值 |
| EV | 挣值 | 实际完成的工作价值 |
| AC | 实际成本 | 完成工作的实际花费 |
| SV | EV-PV | 进度偏差 |
| CV | EV-AC | 成本偏差 |
| SPI | EV/PV | 进度绩效指数 |
| CPI | EV/AC | 成本绩效指数 |

### 偏差分析

- 正偏差：提前或节约——分析是否可持续
- 负偏差：落后或超支——分析根因制定纠偏

## 7.3 纠偏措施

- 赶工：增加资源缩短工期
- 快速跟进：并行执行缩短工期
- 范围调整：与客户协商缩减范围

## 7.4 过程质量保证

- 评审和检查点设置
- 过程文档化
- 质量度量指标跟踪

## 核心要点

1. **过程+结果**：两者不可偏废
2. **EVM量化控制**：用数据说话，不凭感觉
3. **及时纠偏**：偏差越小越容易纠正
4. **过程质量保障结果**：过程不出问题结果才可靠
