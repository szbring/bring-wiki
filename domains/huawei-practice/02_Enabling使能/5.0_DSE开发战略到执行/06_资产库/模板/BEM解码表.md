# BEM 解码表模板

<!-- chunk: bem解码表.使用说明 -->
## 使用说明

BEM（Business Execution Model）解码表是战略解码的核心工具，将SP中的战略意图逐步分解为可执行的KPI和PBC。本模板在BEM工作坊中使用，由高管团队共同完成。

---

<!-- chunk: bem解码表.第一层-战略意图 -->
## 第一层：战略意图

| 项目 | 内容 |
|------|------|
| 战略意图描述 | （3-5年要达到什么目标状态，需量化） |
| 对应SP章节 | SP第X章第X节 |
| 核心差距 | 当前状态 vs 目标状态的核心差距 |
| 优先方向 | 1-3年内最重要的战略方向 |

<!-- chunk: bem解码表.第四层-年度优先行动 -->
## 第四层：年度优先行动

从战略举措中筛选出"今年必须做"的3-5个优先行动：

| # | 优先行动 | 来源举措 | 目标成果 | 负责人 | 关键里程碑 | 所需资源 |
|---|---------|---------|---------|--------|-----------|---------|
| PA1 | | CSF#-# | | | Q1: / Q2: / Q3: / Q4: | |
| PA2 | | CSF#-# | | | Q1: / Q2: / Q3: / Q4: | |
| PA3 | | CSF#-# | | | Q1: / Q2: / Q3: / Q4: | |
| PA4 | | CSF#-# | | | Q1: / Q2: / Q3: / Q4: | |
| PA5 | | CSF#-# | | | Q1: / Q2: / Q3: / Q4: | |

### 优先行动筛选标准

- [ ] 是否直接支撑CSF？（如果没有这个行动，CSF能否实现？）
- [ ] 是否在年度内可启动并取得阶段性成果？
- [ ] 是否有明确的负责人？
- [ ] 资源是否已落实或可落实？
- [ ] 是否具有战略突破性（而非日常工作包装）？
