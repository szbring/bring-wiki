# 战略执行力评估

<!-- chunk: 战略执行力评估.一句话定义 -->
## 一句话定义

评估企业战略从制定到落地的全链条执行能力，识别[[DSTE]]断层的关键环节，为战略管理体系优化提供精准输入。

---

<!-- chunk: 战略执行力评估.一-评估框架 -->
## 一、评估框架

战略执行力评估覆盖"制定—解码—执行—监控"四个阶段，每个阶段设置关键评估维度：

```
战略制定力 ──→ 战略解码力 ──→ 战略执行力 ──→ 战略监控力
   (25%)         (30%)          (25%)          (20%)
```

---

<!-- chunk: 战略执行力评估.二-评估维度与评分标准 -->
## 二、评估维度与评分标准

### 2.1 战略制定力（25分）

| # | 评估项 | 1分 | 3分 | 5分 | 得分 |
|---|--------|-----|-----|-----|------|
| 1 | 战略清晰度 | 无书面战略 | 有战略但表述模糊 | 战略量化清晰，全员可述 | |
| 2 | 洞察深度 | 靠老板直觉 | 有调研但不系统 | 系统性五看分析 | |
| 3 | 高管对齐度 | 各自理解不同 | 大方向一致细节不同 | 高度一致可互相复述 | |
| 4 | SP质量 | 无SP | 有SP但走形式 | SP可指导BP制定 | |
| 5 | 战略更新机制 | 几年不变 | 年度回顾但不改 | 滚动更新有章法 | |

### 2.2 战略解码力（30分）

| # | 评估项 | 1分 | 3分 | 5分 | 得分 |
|---|--------|-----|-----|-----|------|
| 6 | CSF识别 | 无关键成功要素 | 有识别但未验证 | CSF经过讨论验证且聚焦 | |
| 7 | KPI承接 | KPI与战略无关 | 部分KPI承接战略 | KPI完整承接战略举措 | |
| 8 | PBC覆盖率 | 无PBC | 管理层有PBC | 全员PBC与战略关联 | |
| 9 | 优先行动聚焦 | 年度工作罗列 | 有重点但太多 | 3-5个真正优先行动 | |
| 10 | 预算对齐 | 预算与战略脱节 | 部分对齐 | 预算严格匹配战略优先级 | |
| 11 | 横向协同 | 部门各自为政 | 有协调但靠个人 | 有制度化的跨部门协同机制 | |

### 2.3 战略执行力（25分）

| # | 评估项 | 1分 | 3分 | 5分 | 得分 |
|---|--------|-----|-----|-----|------|
| 12 | 目标达成率 | <60% | 60-80% | >80% | |
| 13 | 战略举措完成率 | 大部分未完成 | 完成一半以上 | 核心举措全部推进 | |
| 14 | 资源到位率 | 战略项目缺人缺钱 | 基本到位 | 资源优先保障战略 | |
| 15 | 组织支撑度 | 组织结构阻碍执行 | 部分支撑 | 组织匹配战略需求 | |
| 16 | 人才就绪度 | 关键岗位无人 | 部分到位 | 关键人才充足 | |

### 2.4 战略监控力（20分）

| # | 评估项 | 1分 | 3分 | 5分 | 得分 |
|---|--------|-----|-----|-----|------|
| 17 | QBR机制 | 无季度回顾 | 有回顾但流于形式 | QBR有决策有纠偏 | |
| 18 | 执行追踪可视化 | 无追踪 | 有报告但滞后 | 实时看板红黄绿灯 | |
| 19 | 偏差响应速度 | 年底才发现偏差 | 季度发现季度处理 | 月度发现即时纠偏 | |
| 20 | 战略复盘质量 | 无复盘 | 有复盘但走过场 | 系统复盘驱动改进 | |

<!-- chunk: 战略执行力评估.三-评估结果解读 -->
## 三、评估结果解读

### 3.1 总分解读

| 总分 | 成熟度等级 | 典型特征 | 建议切入点 |
|------|-----------|---------|-----------|
| 80-100 | L4-L5 卓越 | 战略闭环运转，持续优化 | 保持领先，分享标杆 |
| 60-79 | L3 规范 | 有体系但执行有缺口 | 重点补短板 |
| 40-59 | L2 初步 | 有意识但无体系 | 从SP/BP流程建起 |
| <40 | L1 初始 | 战略管理缺失 | 先建立基本战略共识 |

### 3.2 四维度雷达图分析

绘制"制定—解码—执行—监控"四个维度的雷达图：

- **均衡型**：四维度得分接近 → 全面推进
- **制定强执行弱型**：制定分高，执行分低 → 重点补解码和追踪
- **监控短板型**：监控分明显低于其他 → 建立QBR和看板
- **解码断层型**：解码分低导致执行和监控都低 → 优先做BEM解码

<!-- chunk: 战略执行力评估.四-快速诊断访谈问题 -->
## 四、快速诊断访谈问题

### 对 CEO

1. 您能说出公司今年最重要的3件事吗？（考察战略清晰度和聚焦度）
2. 各部门负责人对这3件事的理解和您一致吗？（考察解码力）
3. 上季度战略执行出了什么偏差？怎么处理的？（考察监控力）

### 对部门负责人

4. 您的年度KPI是怎么来的？（考察解码力——是自己想的还是战略推导的）
5. 您的团队知道公司今年的战略重点吗？（考察战略传达力）
6. 季度回顾会讨论什么内容？（考察监控质量）

### 对普通员工

7. 您知道自己今年的工作重点是什么吗？（考察PBC落地）
8. 这些重点和公司战略有什么关系？（考察战略传导的深度）
