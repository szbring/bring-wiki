# 战略管理诊断工具

<!-- chunk: 战略管理诊断工具.成熟度评估-五级 -->
## 成熟度评估（五级）

| 级别 | 特征 |
|---|---|
| L1 | 无书面战略，老板一拍脑袋，方向随时变 |
| L2 | 有年度目标，但高管之间理解不一致，缺乏解码 |
| L3 | 有BLM/战略规划，战略解码到部门，有年度KPI |
| L4 | 战略动态调整，有市场洞察驱动，执行有追踪闭环 |
| L5 | 战略成为组织DNA，上下对齐，快速迭代应对变化 |

<!-- chunk: 战略管理诊断工具.快速诊断问题 -->
## 快速诊断问题

### 对 CEO

1. **「公司未来三年的战略是什么？能用三句话说清楚吗？」**
   - 说不清楚/每次说法不一样 → L1-L2，战略不清晰
   - 能说清楚但下面人不知道 → L2-L3，战略解码缺失
   - 上下对齐，全员知晓 → L3+

2. **「公司今年最重要的三件事是什么？是怎么确定的？」**
   - 拍脑袋决定 → L1-L2
   - 有讨论但无系统性方法 → L2
   - 基于市场分析+[[BLM]]推导 → L3+

3. **「去年制定的战略目标，完成情况怎样？主要差距在哪里？」**
   - 没有复盘机制 → L1-L2
   - 有复盘但没有改进行动 → L2-L3
   - 有复盘、有根因分析、有改进计划 → L3+

4. **「您对市场变化的判断，主要来自哪里？」**
   - 个人经验和直觉 → L1-L2
   - 有市场调研但不系统 → L2
   - 有系统性市场洞察机制 → L3+

### 对高管团队

5. **「你们高管对公司战略的理解是一致的吗？能描述出来吗？」**
   - 各说各的 → 战略对齐严重缺失
   - 大体一致但细节不同 → L2-L3
   - 高度一致 → L3+

6. **「你们部门的年度目标，和公司战略是如何关联的？」**
   - 说不清楚 → 战略解码缺失
   - 有关联但自己想的 → L2
   - 通过战略解码正式推导出来 → L3+

## BLM诊断维度评分表

| 维度 | 问题 | 1-5分 |
|---|---|---|
| 市场洞察 | 对行业趋势的理解是否深入和及时？ | |
| 战略意图 | 公司愿景/使命/战略目标是否清晰？ | |
| 创新焦点 | 资源配置是否聚焦在最重要的事情？ | |
| 业务设计 | 商业模式是否清晰且有竞争力？ | |
| 关键任务 | 战略落地的关键任务是否明确？ | |
| 正式组织 | 组织结构和流程是否支持战略？ | |
| 人才 | 关键岗位是否有合适的人？ | |
| 企业文化 | 文化是否与战略方向一致？ | |
| **合计** | | **/40** |

**评分解读**：
- 32-40分：L3-L4，有较好[[DSTE]]基础
- 20-31分：L2-L3，战略管理有待系统化
- 20分以下：L1-L2，需要从基础重建
