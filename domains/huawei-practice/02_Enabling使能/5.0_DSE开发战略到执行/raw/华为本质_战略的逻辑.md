# 来自《华为本质》- 战略的逻辑

## 核心观点

华为战略逻辑的核心是"管道战略"——只做管道不做水。华为将自身定位为输送信息资源的"管道"构建者，坚决不碰管道里的"水"（内容与应用），通过聚焦主航道、扩宽管道、动态调整，确立了在全球通信行业的领导地位。

## 一、管道战略

### 管道是华为的航道

华为将自身产品和解决方案定位于向自来水输水管那样的"管道"，致力于为客户构建安全强大的管网。华为集中资源构建三大管道业务——面向消费者、企业、运营商三类客户，分别成立各自的事业群。这些管道宏观上各自独立，微观上彼此连通、相互促进。

任正非说："公司要像长江水一样聚焦在主航道，发出巨大的电来。无论产品大小都要与主航道有关，新生幼苗也要聚焦在主航道上。"

### 扩宽管道但不碰里面的"水"

2015年冬季达沃斯论坛，任正非被问及设备安全问题时回答："华为的通信网络就犹如自来水管道，华为所做的是管道的铁皮，管道里面的水是运营商和互联网在走。华为搜它的信息干什么？"

轮值CEO徐直军进一步阐述："面对即将到来的大数据流量时代，华为选择定位管道，将管道作为核心战略。数据流量是水，华为做的是运水的管道。管道是华为的核心战略，是华为的航道，我们所有的业务都是沿着管道战略展开的，力出一孔。"

华为管道战略本质上是一种**集中性战略**和**双赢战略**：
- 面向个体消费者：开发产生流量和消费流量的网络终端，而非消费电子产品
- 面向企业客户：开发ICT基础设施，只做基础设备生产商，不碰细分领域的应用软件
- 面向运营商：聚焦端到端大管道架构，不提供任何内容

## 二、管道的发展规划

### 聚焦运营商网络业务

运营商业务占华为收入70%，华为在运营商领域已构筑领导地位。2009-2014年运营商业务年复合增长率达10.9%，是业界平均增速的两倍。华为在全球建立了300多个数据中心，在170多个行业标准组织担任185个核心职位。

华为运营商业务确定了"成为客户最佳合作伙伴，管道产业的领导者"的战略目标。

### 做精企业业务

2011年运营商市场增长放缓后，任正非将管道战略延伸到企业级市场。华为企业BG中国区过去5年年均复合增长率达36%。

企业业务的关键转折是"被集成"战略：华为不与合作伙伴和代理商争利，围绕合作伙伴的发展、培育、支持、激励来开展业务。中国区企业业务80%以上销售收入来自渠道合作伙伴。"华为能力有限"成为高管层反复强调的信条，消除伙伴顾虑。

华为还推出"一国一策"战略，在不同国家采取不同渠道策略。

## 三、管道的调整

华为管道调整坚持四个原则：
1. **战略聚焦**——力避过度宽泛计划，防止资源虚耗
2. **坚持革新**——出现问题第一时间做出反应
3. **量体裁衣**——根据不同发展阶段设计管道战略（当下、3-5年、10年）
4. **谋定后动**——建立在实践基础上的调整

### 抢占大数据制高点

任正非认为大数据是管道的基础，华为必须抢占大数据制高点。在大数据流量上要敢于抢占制高点，创造适应客户需求的高点产品。华为在数据业务上逐年加大投入，在关键地区集中资源发起"攻击"。

## 四、管道的探索

### 构建全连接世界

华为管道战略的终极目标是以客户为中心，构建全连接世界。具体体现在三个层面：
1. 让人和人更加紧密地连接，打破知识传播限制
2. 提供更高宽带连接，缩短时空距离
3. 构建物与物连接管道，迎接万物互联时代

### 打造"云-管-端"一体化能力

华为的"云"是云计算（SingleCLOUD平台），"管"为网络通道，"端"为终端设备。核心理念是"统一"——实现软硬件资源统一调度，使管道资源最大化利用。

近年来华为将"云-管-端"一体化的重点放在终端上，推出了自主品牌手机、移动宽带产品及融合类产品。凭借一体化战略，华为能为客户构建完整的管网系统，获得综合性竞争优势。

## 要点总结

| 战略维度 | 核心内容 |
|---------|---------|
| 管道定位 | 只做管道不做水，聚焦信息输送基础设施建设 |
| 三大业务 | 运营商（70%）、企业（被集成）、消费者（终端） |
| 战略聚焦 | 能力有限，力出一孔，拒绝多元化 |
| 动态调整 | 战略聚焦+坚持革新+量体裁衣+谋定后动 |
| 终极目标 | 构建全连接世界，云-管-端一体化 |
