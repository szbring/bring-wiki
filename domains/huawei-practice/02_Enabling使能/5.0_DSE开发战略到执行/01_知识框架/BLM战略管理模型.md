# BLM — 业务领先模型（Business Leadership Model）

<!-- chunk: blm战略管理模型.一句话定义 -->
## 一句话定义
华为战略规划与执行的核心方法论，将市场洞察、战略意图、创新焦点和执行能力整合为一个完整的战略管理闭环。

<!-- chunk: blm战略管理模型.blm的八大要素 -->
## BLM的八大要素

```
外部环境（差距）
      ↓
┌─────────────────────────────┐
│  战略制定侧          执行侧   │
│  ┌─────────┐   ┌─────────┐  │
│  │市场洞察 │   │业务设计 │  │
│  ├─────────┤   ├─────────┤  │
│  │战略意图 │   │关键任务 │  │
│  ├─────────┤   ├─────────┤  │
│  │创新焦点 │   │正式组织 │  │
│  └─────────┘   │人才     │  │
│                │企业文化 │  │
│                └─────────┘  │
└─────────────────────────────┘
            ↓
        业务结果
```

<!-- chunk: blm战略管理模型.八大要素详解 -->
## 八大要素详解

### 战略制定侧（左脑）

**1. 市场洞察**
- 行业趋势：未来3-5年行业会发生什么？
- 客户洞察：客户最核心的需求是什么？
- 竞争分析：竞争对手在做什么？
- 自身分析：我们的优势和劣势是什么？

**2. 战略意图**
- 使命：我们存在的意义是什么？
- 愿景：3-5年后我们要成为什么？
- 战略目标：具体要达成什么？（可量化）

**3. 创新焦点**
- 业务创新：商业模式、产品、服务的创新
- 管理创新：流程、组织、文化的变革
- 资源配置：把有限资源投入到最重要的地方

### 执行侧（右脑）

**4. 业务设计**
- 目标客户选择：服务谁？
- 价值主张：给客户什么独特价值？
- 盈利模式：如何获得收益？
- 价值链定位：在产业链中扮演什么角色？

**5. 关键任务**
- 实现战略必须完成的TOP5-8件事
- 每个任务有明确的责任人和时间节点

**6. 正式组织**
- 组织结构是否支持战略执行？
- 流程是否高效？
- 激励机制是否对齐战略目标？

**7. 人才**
- 关键岗位是否有合适的人？
- 人才能力是否支撑战略？
- 人才发展计划是否就绪？

**8. 企业文化**
- 核心价值观是否与战略对齐？
- 文化是否支持创新和变革？
- 领导者行为是否体现文化价值？

<!-- chunk: blm战略管理模型.blm在咨询项目中的应用 -->
## BLM在咨询项目中的应用

**战略规划项目**：带领客户高管团队完成BLM工作坊
- 通常1.5-2天
- 输出：战略地图 + 年度优先任务

**战略解码**：将BLM的战略意图分解为部门目标
- 用OKR或BSC（平衡计分卡）承接
- 确保战略从高层到基层的传导

<!-- chunk: blm战略管理模型.诊断问题 -->
## 诊断问题

- 公司的战略是否文字化、可理解？
- 所有高管是否对战略方向高度一致？
- 战略到执行的转化是否清晰？
- 资源分配是否与战略优先级一致？
