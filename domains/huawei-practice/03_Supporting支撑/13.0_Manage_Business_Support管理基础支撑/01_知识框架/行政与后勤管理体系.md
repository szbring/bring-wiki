# 行政与后勤管理体系

<!-- chunk: 行政与后勤管理体系.一句话定义 -->

## 二、办公管理

### 2.1 办公场地管理

| 管理项 | 核心内容 | 管理标准 |
|---|---|---|
| 选址与租赁 | 办公需求评估、选址标准、租赁谈判 | 人均面积标准、租金预算 |
| 空间规划 | 功能分区、工位安排、会议室配置 | 开放与封闭空间比例 |
| 装修与维护 | 装修标准、日常维护、定期翻新 | 装修预算标准、维护SOP |
| 搬迁管理 | 搬迁规划、过渡方案、员工沟通 | 搬迁检查清单 |

### 2.2 办公采购

| 采购类别 | 采购方式 | 管控要点 |
|---|---|---|
| 办公用品 | 集中采购+电商直采 | 预算控制、库存管理 |
| 办公设备 | 框架协议+定期采购 | 生命周期管理 |
| 服务采购（保洁/绿化等） | 年度招标 | SLA管理、质量考核 |

### 2.3 文秘与档案管理

- 公文流转标准化（起草→审核→签发→归档）
- 档案分类体系（按业务/年度/密级分类）
- 电子档案管理系统
- 保密文件特殊管控

## 三、资产管理

### 3.1 固定资产管理

```
采购 → 登记入库 → 领用 → 日常管理 → 盘点 → 报废/调拨
```

| 管理环节 | 关键活动 | 管控要点 |
|---|---|---|
| 采购审批 | 需求评估、预算审批 | 审批分级、供应商管理 |
| 登记入库 | 资产编号、信息录入 | 一物一码、台账登记 |
| 领用管理 | 领用登记、责任到人 | 领用审批、责任人确认 |
| 日常管理 | 维修保养、异动管理 | 定期巡检、变更审批 |
| 盘点管理 | 定期盘点、差异处理 | 年度全覆盖、差异分析 |
| 报废/调拨 | 报废鉴定、调拨审批 | 残值处理、跨部门调拨流程 |

### 3.2 车辆管理

| 管理项 | 内容 | 标准 |
|---|---|---|
| 车辆调度 | 用车申请、审批、调度 | 分级审批、优先级规则 |
| 费用管理 | 油费、维修、保险 | 预算控制、单车核算 |
| 安全管理 | 驾驶员管理、安全培训 | 驾驶员资质、安全记录 |
| GPS监控 | 行程记录、轨迹回放 | 规范用车、异常预警 |

## 四、后勤保障

### 4.1 餐饮服务

| 模式 | 特点 | 适用场景 |
|---|---|---|
| 自建食堂 | 质量可控，成本较高 | 大型园区/工厂 |
| 外包团餐 | 专业服务，成本可控 | 中型企业 |
| 餐补制度 | 灵活便捷，员工自主 | 小型/分散办公 |

### 4.2 差旅管理

| 管理项 | 内容 | 管控要点 |
|---|---|---|
| 差旅标准 | 住宿/交通/餐饮标准 | 按级别/城市分级 |
| 差旅审批 | 出差申请、预算审批 | 事前审批、超标特批 |
| 差旅预订 | 统一平台预订 | 集中采购、协议价格 |
| 差旅报销 | 票据审核、费用报销 | 发票核验、标准校验 |

### 4.3 健康与关怀

- 年度体检制度（按年龄/级别差异化方案）
- 办公环境健康（空气质量、照明、工效学）
- 心理健康支持（EAP员工帮助计划）
- 紧急医疗响应机制

## 五、安全管理

| 安全类型 | 管理内容 | 关键措施 |
|---|---|---|---|
| 消防安全 | 消防设施、演练、检查 | 年度消防演练、月度巡检 |
| 安保管理 | 门禁管理、访客管理、巡逻 | 人脸识别门禁、访客预约系统 |
| 信息安全 | 文件安全、设备安全 | 保密制度、设备加密、远程擦除 |
| 应急管理 | 突发事件响应、BCP | 应急预案、定期演练 |

## 六、行政数字化转型

| 传统模式 | 数字化模式 | 收益 |
|---|---|---|
| 纸质审批 | 移动OA/飞书审批 | 审批时效提升80% |
| 手工台账 | 资产管理系统 | 盘点效率提升、差异减少 |
| 电话订餐 | 餐饮小程序 | 员工满意度提升 |
| 人工巡检 | IoT+智能监控 | 实时预警、自动派单 |
| 纸质差旅报销 | 差旅平台+智能报销 | 报销周期缩短50% |
