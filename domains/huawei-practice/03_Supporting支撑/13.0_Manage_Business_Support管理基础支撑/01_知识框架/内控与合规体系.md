# 内控与合规体系

<!-- chunk: 内控与合规体系.一句话定义 -->
## 一句话定义
基于COSO框架建立内部控制五要素体系，通过三道防线实现内控有效、合规运营，是企业上市和持续经营的基础保障。

---

<!-- chunk: 内控与合规体系.一-coso内部控制五要素 -->
## 一、COSO内部控制五要素

华为内控体系基于COSO模型设计，包括五大部分：

| 要素 | 核心内容 | 华为实践 |
|---|---|---|
| 控制环境 | 诚信文化、治理架构、权责分配、胜任力 | 员工商业行为准则(BCG)全员签署 |
| 风险评估 | 目标设定、风险识别、风险分析、舞弊风险评估 | 全球业务流程定期风险评估 |
| 控制活动 | 授权审批、职责分离、核对校验、实物控制 | 全球流程控制手册、职责分离矩阵 |
| 信息与沟通 | 内部报告、外部报告、沟通渠道 | 多维度信息与沟通渠道 |
| 内部监督 | 持续监控、独立评估、缺陷报告 | 内部审计独立评估、定期控制评估报告 |

---

<!-- chunk: 内控与合规体系.二-内控三道防线 -->
## 二、内控三道防线

```
第一道防线：业务部门（自我管控）
    · 执行内控制度
    · 识别和管理本部门风险
    · 合规操作，留存记录
        ↓
第二道防线：风控/合规部门（专业监督）
    · 制定风控政策和标准
    · 监控合规执行情况
    · 识别系统性风险
        ↓
第三道防线：内部审计（独立评估）
    · 独立评估内控有效性
    · 专项审计（财务/业务/IT）
    · 向审计委员会直接汇报
        ↓
审计委员会（董事会层面监督）
```

## 三、合规管理体系

### 3.1 合规管理框架

| 要素 | 内容 |
|---|---|
| 合规管理环境 | 领导重视、合规文化、资源保障 |
| 合规职责 | 董事会、管理层、合规部门、业务部门各司其职 |
| 合规风险评估 | 识别合规义务、评估合规风险、制定应对措施 |
| 合规管理制度 | 合规管理办法、行为准则、操作指引 |
| 合规管控流程 | 合规审查、合规检查、举报与调查 |
| 合规评价 | 合规有效性评估、合规绩效考核 |
| 合规文化 | 培训宣传、价值观引导 |

### 3.2 关键合规领域

| 合规领域 | 主要法规/标准 | 关键管控点 |
|---|---|---|
| 贸易合规 | 出口管制、经济制裁 | 交易筛查、受限方清单、许可申请 |
| 反商业贿赂 | FCPA、UKBA、反不正当竞争法 | 礼品招待审批、第三方尽职调查 |
| 数据隐私 | GDPR、个保法、数据安全法 | 数据分类分级、隐私影响评估、跨境合规 |
| 竞争合规 | 反垄断法 | 定价策略审查、竞争对手接触规范 |
| 劳动合规 | 劳动法、社保法规 | 劳动合同、工时管理、社保缴纳 |

## 四、上市前内控建设要点

证监会/交易所重点关注：

| 关注领域 | 核心要求 | 建设要点 |
|---|---|---|
| 财务报告内控 | 防止财务造假 | 会计政策规范、收入确认、关联交易 |
| 关联交易管控 | 防止利益输送 | 关联方识别、交易审批、公允定价 |
| 重大决策流程 | 防止内部人控制 | 三重一大、授权体系、决策记录 |
| IT系统安全 | 防止数据造假 | 访问控制、审计轨迹、变更管理 |
| 信息披露 | 真实完整及时 | 披露制度、内幕信息管控 |

### 上市合规时间线

| 时间节点 | 工作内容 |
|---|---|
| IPO申报前24个月 | 内控体系建设启动 |
| IPO申报前18个月 | 三道防线建立并运行 |
| IPO申报前12个月 | 内控自我评价、缺陷整改 |
| IPO申报前6个月 | 内控审计准备、信息披露制度 |
| 申报时 | 内控经得起尽职调查 |

## 五、法务、合规、内控、风险协同

### 四者的价值主张

| 职能 | 核心思维 | 侧重点 |
|---|---|---|
| 法务管理 | 商业思维 | 维护企业合法权益、创造法律价值 |
| 合规管理 | 底线思维 | 确保法律法规和监管要求的遵循 |
| 内部控制 | 制衡思维 | 规范运作、职责分离、关键控制点 |
| 风险管理 | 权衡思维 | 平衡风险与收益、风险可承受范围内 |

### 协同机制

1. **统一目标** — 将四者目标统一描述，各自目标作为子目标
2. **协同组织** — 董事会下设合规与风控委员会统筹
3. **统一风险评估** — 一次评估所有类型风险，形成整合控制矩阵
4. **流程协同设计** — 合并同类项，突出特殊项
5. **制度合并汇编** — 统一的风险管控与合规管理制度手册
6. **统一考核** — 协同职责纳入绩效考核
