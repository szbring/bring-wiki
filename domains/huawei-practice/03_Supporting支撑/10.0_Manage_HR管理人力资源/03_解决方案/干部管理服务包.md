# 干部管理服务包

<!-- chunk: 干部管理服务包.产品定位 -->
## 产品定位

帮助客户建立从选拔、评价、培养到淘汰的[[华为干部管理]]闭环体系，打造「能上能下、之字形成长、从实战中来」的干部队伍，让干部成为组织战斗力的核心引擎。

---

<!-- chunk: 干部管理服务包.服务包概览 -->
## 服务包概览

| 服务包 | 周期 | 人天 | 适用客户 |
|---|---|---|---|
| 干部管理诊断 | 2周 | 4-6天 | 了解干部队伍现状 |
| 干部管理体系搭建 | 3-4个月 | 25-35天 | 无体系或体系混乱 |
| 干部梯队建设 | 4-6个月 | 20-30天 | 核心岗位缺接班人 |
| 干部培养项目 | 3-6个月 | 15-25天 | 干部能力不足 |

<!-- chunk: 干部管理服务包.sp1-干部管理诊断-4-6人天 -->
## SP1 — 干部管理诊断（4-6人天）

### 诊断内容

| 诊断维度 | 核心问题 | 数据来源 |
|---|---|---|
| 干部标准 | 有没有客观的干部选拔标准？ | 制度文件+访谈 |
| 干部结构 | 年龄/级别/司龄分布是否合理？ | HR数据 |
| 干部能力 | 管理能力是否匹配岗位要求？ | 360度评估 |
| 干部活力 | 能上能下机制是否运行？ | 晋降数据 |
| 继任储备 | 关键岗位有没有后备人选？ | 继任者计划 |

### 快速判断问题

1. **「核心岗位如果明天负责人离职，有几个人可以接？」**
   - 没有 → 继任者计划缺失
   - 有1个但不确定 → 梯队薄弱
   - 有2-3个合格人选 → 相对健康

2. **「过去3年，有几位干部被降级或免职？原因是什么？」**
   - 没有 → 干部能上能下机制缺失
   - 1-2位且是因为犯错 → 只有底线管理
   - 有绩效不达标被调整的 → 机制在运行

3. **「干部选拔的主要依据是什么？」**
   - 老板感觉/资历 → 缺乏标准
   - 部分量化指标 → 有基础但不完善
   - 三看原则（贡献+能力+价值观） → 体系成熟

### 交付物
- 干部队伍现状报告
- 关键风险清单
- 改进路径建议

### 报价参考
- 12-20万元

## SP2 — 干部管理体系搭建（3-4个月）

### Phase 1：干部标准设计（4-6周）

**核心交付**：
- 干部定义与分类（高/中/基层）
- 干部选拔「三看」标准细化
  - 贡献标准：PBC等级要求、关键经历要求
  - 能力标准：各级干部管理能力模型
  - 价值观标准：行为红线和底线
- 干部选拔流程设计
  - 推荐方式（自荐/组织推荐/民主推荐）
  - 评审委员会组成和运作
  - 任前公示和试用期

**干部能力模型示例**：

| 干部层级 | 核心能力 | 关键行为 |
|---|---|---|
| 基层干部 | 执行力+带人 | 能把目标拆解到个人，能辅导下属成长 |
| 中层干部 | 协调力+决策力 | 能推动跨部门协作，能做资源取舍决策 |
| 高层干部 | 战略力+文化力 | 能定义方向，能塑造文化，能带班子 |

### Phase 2：评价与淘汰机制设计（3-4周）

**核心交付**：
- 干部三维评价体系（贡献+能力+价值观）
- 干部年度述职制度
- 干部末位淘汰规则
  - 排名后10%处理方式（降级/转岗/免职）
  - 下来的干部再上去的机制
- 民主评议制度

### Phase 3：培养与梯队设计（4-6周）

**核心交付**：
- 之字形成长路径设计
- 轮岗规则（哪些岗位必须轮、轮多久）
- 继任者计划（每个关键岗位2名后备）
- 干部培养项目设计（青训班/高研班方案框架）

### 全套交付物
- 干部管理制度
- 干部选拔标准与流程
- 干部评价工具（述职模板+360题库）
- 末位淘汰操作指引
- 继任者计划模板
- 轮岗管理制度
- 干部能力模型手册

### 报价参考
- 40-65万元

<!-- chunk: 干部管理服务包.sp3-干部梯队建设-4-6个月 -->
## SP3 — 干部梯队建设（4-6个月）

### 服务范围

| 模块 | 内容 | 产出 |
|---|---|---|
| 人才盘点 | 对TOP30关键岗位现任者评估 | 人才九宫格 |
| 继任者识别 | 每个关键岗位确定2名后备 | 继任者名单 |
| 差距分析 | 后备人才vs岗位要求差距 | 个人发展计划 |
| 培养实施 | 轮岗+行动学习+导师制 | 能力提升 |
| 跟踪评估 | 半年复盘，动态调整 | 梯队状态报告 |

### 报价参考
- 30-50万元

## SP4 — 干部培养项目（3-6个月）

### 青训班方案（基层干部）

| 模块 | 时长 | 方式 |
|---|---|---|
| 管理基础 | 2天 | 集中授课+案例研讨 |
| 带队伍 | 2天 | 情景模拟+角色扮演 |
| 行动学习 | 2个月 | 带真实业务问题学习 |
| 结业答辩 | 1天 | 评审委员会评审 |

### 高研班方案（中层干部）

| 模块 | 时长 | 方式 |
|---|---|---|
| 战略思维 | 2天 | 沙盘推演+案例研讨 |
| 跨部门协作 | 1天 | 工作坊 |
| 经营决策 | 2天 | 商业模拟 |
| 行动学习 | 3个月 | 跨部门课题研究 |
| 结业答辩 | 1天 | 高层评审 |

### 报价参考
- 青训班：15-25万/期（20-30人）
- 高研班：20-35万/期（15-20人）

## 组合推荐

| 客户类型 | 推荐组合 | 预算 |
|---|---|---|
| 科技企业（干部标准缺失） | SP1 → SP2 | 50-80万 |
| 成长期企业（梯队断层） | SP2 + SP3 | 60-100万 |
| 大型企业（干部能力不足） | SP3 + SP4 | 45-75万 |
| 上市前企业 | SP1 + SP2（聚焦高管治理） | 40-60万 |
