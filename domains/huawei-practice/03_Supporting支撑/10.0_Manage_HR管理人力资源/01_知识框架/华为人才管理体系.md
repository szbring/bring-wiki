# 华为人才管理体系

> 任职资格、绩效管理、干部管理：华为人才管理的三大支柱

---

<!-- chunk: 华为人才管理体系.一-概述-华为为什么能留住和激发人才 -->
## 一、概述：华为为什么能留住和激发人才

华为的人才管理体系被业界广泛研究，其核心不是单一的"高薪"或"文化"，而是一套**系统性的牵引、激励和约束机制**，使每个员工清楚地知道：

1. **我应该具备什么能力，成长路径是什么**（任职资格体系）
2. **我今年的目标是什么，做到了能得到什么**（绩效管理体系）
3. **我要成为干部，需要满足什么标准**（干部管理体系）

这三大体系相互支撑，形成"识别人才→培养人才→使用人才→保留人才"的完整闭环。

### 1.1 三大体系的关系

```
                  ┌─────────────────┐
                  │   干部管理体系   │
                  │（战略人才池管理）│
                  └────────┬────────┘
                           │评价与选拔
              ┌────────────┴────────────┐
              ↓                         ↓
    ┌──────────────────┐   ┌────────────────────┐
    │   任职资格体系    │   │    绩效管理体系     │
    │（能力标准与发展） │   │（目标、评价与激励） │
    └──────────────────┘   └────────────────────┘
              ↑                         ↑
              └────────────┬────────────┘
                           │共同构成
                    人才选拔与晋升依据
```

## 二、任职资格体系：能力的"说明书"

### 2.1 任职资格体系的核心作用

**任职资格体系**回答了一个关键问题："什么样的人适合担任什么样的岗位/职级？"

在华为，任职资格体系完成以下几个关键功能：
- **能力标准化**：明确每个岗位族在不同级别的能力要求，消除"靠关系晋升"的空间
- **职业通道规划**：为员工提供清晰的成长路径图，知道要晋升需要发展哪些能力
- **人才评估基准**：为绩效评估和干部选拔提供客观的能力评估标准
- **培训精准化**：基于能力差距设计培训内容，避免"撒网式"培训

### 2.2 华为任职资格的分级体系

华为将员工职业发展分为两条通道，并建立严格的分级标准：

#### 管理通道
```
EMT成员（公司最高决策层）
    ↑
高级副总裁/副总裁（SVP/VP）
    ↑
总裁/常务副总裁
    ↑
高级经理/部门总监（Band 15-18）
    ↑
经理（Band 13-14）
    ↑
主管（Band 11-12）
```

#### 专业技术通道（以研发为例）
```
Fellow（院士级，极少数）
    ↑
专家/首席专家
    ↑
高级工程师/资深工程师
    ↑
工程师
    ↑
助理工程师
```

### 2.3 任职资格认证的核心要素

每个职级的任职资格认证从以下维度综合评估：

| 评估维度 | 权重 | 说明 |
|---------|------|------|
| **知识与技能** | 30% | 岗位所需的专业知识和技术技能 |
| **过往经历与成果** | 40% | 实际项目经历，特别是承担的挑战性任务和取得的成果 |
| **素质与潜力** | 20% | 价值观匹配度、学习能力、领导力潜力 |
| **同事与上级评价** | 10% | 360度反馈中的综合评价 |

### 2.4 任职资格与薪酬的联动

华为的薪酬体系与任职资格紧密挂钩：
- 每个职级有对应的薪酬Band（工资带宽）
- 晋升职级是实现薪酬跃升的主要通道
- 同一职级内，薪酬根据绩效在Band内浮动
- TUP（时间单位计划）和股票期权与职级挂钩

这一设计的关键价值：**让员工明白，要提升薪酬，核心路径是提升能力、获得晋升，而非单纯靠资历**。

### 2.5 推行任职资格体系的关键挑战

| 挑战 | 典型表现 | 解决方向 |
|-----|---------|---------|
| 标准不落地 | 资格标准写得好，但实际评审"走过场" | 引入外部评委，增加客观性 |
| 专业通道晋升难 | 技术人才被逼转管理通道 | 完善专业通道薪酬天花板，确保技术通道有吸引力 |
| 评审周期太长 | 晋升等待期过长，影响员工积极性 | 缩短评审周期，增加常态化评审机会 |
| 能力与岗位脱节 | 晋升后的岗位没有对应挑战 | 晋升与岗位调整挂钩，确保"用武之地" |

## 三、绩效管理体系：目标与激励的引擎

### 3.1 华为绩效管理的核心理念

华为绩效管理的核心是**价值创造→价值评价→价值分配**的闭环逻辑：

1. **价值创造**：员工和团队通过完成挑战性目标创造业务价值
2. **价值评价**：对价值创造结果进行客观、公正的评价
3. **价值分配**：将绩效评价结果转化为差异化的激励（薪酬、晋升、发展机会）

### 3.2 OKR与KPI的结合运用

华为的目标管理实践将战略目标逐级分解到个人：

```
公司战略目标（年度）
    ↓ DSTE战略解码
部门级KPI（年度/季度）
    ↓ 目标分解
团队级OKR（季度）
    ↓ 个人承诺
个人KPI（季度）
```

**关键特征**：
- 目标设定强调"挑战性"——华为内部有"130%目标"的说法，即目标设定得让大多数人需要努力才能完成
- 目标要有牵引性——不能只设"产出指标"，也要有"过程指标"（能力建设、团队发展等）
- 目标要双向对齐——部门目标要支撑公司目标，个人目标要支撑团队目标

### 3.3 绩效评估机制

#### PBC（个人业务承诺）

PBC（Personal Business Commitment）是华为个人绩效管理的核心工具，包含三个部分：

| PBC部分 | 内容 | 权重 |
|--------|------|------|
| **业务目标** | 本期要实现的业务成果（KPI） | 60% |
| **个人发展** | 本期要提升的能力和完成的发展行动 | 20% |
| **价值观行为** | 华为核心价值观的行为表现（客户第一、艰苦奋斗等） | 20% |

#### 绩效评级

华为的绩效评级通常采用强制分布：

| 评级 | 比例 | 含义 |
|-----|------|------|
| A（杰出） | ~10% | 显著超越目标，是团队的突出贡献者 |
| B+（良好） | ~25% | 达成目标，并有部分超越 |
| B（达标） | ~40% | 基本达成目标 |
| C（待提升） | ~20% | 部分未达目标，需要改进 |
| D（不合格） | ~5% | 严重未达目标 |

强制分布的关键价值：**避免"皆大欢喜"的评估，确保评估结果能够真正区分绩效差异**。

### 3.4 绩效与激励的联动

华为的激励体系通过以下机制将绩效与回报挂钩：

**短期激励**
- 年终奖与绩效评级直接挂钩（A级员工年终奖可能是D级员工的3-5倍）
- 绩效奖金按季度/年度发放

**中长期激励**
- TUP（时间单位计划）：按绩效和级别授予时间单位，5年内分批兑现
- 虚拟限制性股票：对核心骨干授予，绑定长期利益
- 这两类工具的设计使优秀员工的长期收益远超当期薪资，形成强力留存效果

**发展激励**
- 高绩效员工获得优先培训资源（海外轮岗、高潜项目、重点任务）
- A/B+员工进入晋升快通道

### 3.5 绩效反馈文化

华为非常重视绩效反馈的质量，要求：
- **季度中期对话**：管理者与员工在季度中期进行一次正式的PBC进展讨论
- **年度绩效面谈**：年底进行正式的绩效评估面谈，明确说明评级理由
- **即时反馈**：鼓励管理者在日常工作中及时提供正向/改进反馈，不把问题积累到年底

## 四、干部管理体系：领导力的培育与选拔

### 4.1 华为干部管理的战略地位

任正非多次强调："干部是公司最重要的战略资产。"华为相信，领导力是公司竞争力的关键来源之一，培养和使用正确的干部，决定了华为能否持续高速发展。

### 4.2 干部选拔的核心原则

#### 原则1：在实战中识别干部

华为不相信纯粹靠测评和推荐识别干部，而是坚持**"在重要任务中看人"**：
- 关键项目的表现是最重要的干部评价依据
- 重视在困难环境下（海外艰苦地区、危机项目）的表现
- "能扛事"是干部的基本素质

#### 原则2：品德与能力并重，品德优先

华为干部选拔的16字方针：**"有德有才，破格重用；有德无才，培养使用；有才无德，限制使用；无德无才，坚决不用"**

品德标准具体体现为：对客户诚信、对公司忠诚、团结协作、不贪腐、敢于承担责任。

#### 原则3：干部能上能下

华为有严格的干部末位淘汰和降级机制：
- 连续两年C/D绩效的干部自动降级
- 在重要项目中出现重大失误的干部要承担责任
- "下去"不是终点，表现恢复后可以重新晋升

### 4.3 干部管理的核心工具：九宫格

华为使用**绩效×潜力九宫格**对干部进行分类管理：

```
           绩效评级
潜    高  │ 待发展 │ 高潜力 │ 明星 │
力        ├────────┼────────┤──── │
评    中  │  一般  │  骨干  │ 绩优 │
级        ├────────┼────────┤──── │
      低  │  淘汰  │  稳定  │ 专家 │
              低       中       高
```

| 象限 | 标签 | 管理策略 |
|-----|------|---------|
| 高绩效+高潜力 | **明星** | 快速晋升，重要任务历练 |
| 高绩效+低潜力 | **绩优** | 保留，在现有岗位充分激励 |
| 低绩效+高潜力 | **待发展** | 辅导+换岗，给机会证明自己 |
| 低绩效+低潜力 | **淘汰** | 绩效改进计划，最终淘汰 |

### 4.4 干部培训体系：华为大学

华为大学是华为干部培训的主要平台，核心培训项目包括：

| 项目 | 对象 | 目标 |
|-----|------|------|
| **高级管理层培训（EDP）** | SVP/VP级 | 战略思维、全球领导力 |
| **新干部培训** | 首次担任管理岗位 | 管理基础、华为文化 |
| **海外轮岗项目** | 高潜力中层干部 | 全球化视野、跨文化领导力 |
| **实战攻关项目** | 各级干部 | 在真实业务挑战中锻炼 |

华为培训的核心理念：**"在战争中学习战争"**——培训不是课堂教学，而是在真实业务场景中历练和反思。

### 4.5 干部问责与免责机制

华为对干部有严格的问责机制：

**问责触发情形**
- 重大项目失败且有明显决策失误
- 在管辖范围内发生重大合规问题
- 连续绩效不达标
- 品德问题（腐败、失信）

**免责保护机制**
华为同时设计了免责机制，鼓励干部大胆创新：
- 对于探索性创新项目的失败，不追究个人责任
- 对于"尽力而为仍未达成"的挑战性目标，区别对待
- 防止干部因为害怕担责而"不作为"

## 五、三大体系的整合运作

### 5.1 年度人才管理节奏

```
Q1（1-3月）
- 年度PBC设定（个人绩效承诺）
- 任职资格认证申请（有条件滚动）
- 干部九宫格盘点（部门级）

Q2（4-6月）
- 季度绩效对话（进展确认）
- 干部培养计划中期检查

Q3（7-9月）
- 绩效中期评估（可调整目标）
- 高潜力人才项目申请

Q4（10-12月）
- 年度绩效评估
- 任职资格认证评审
- 公司级干部盘点（九宫格）
- 激励方案确认（年终奖/股票）
```

### 5.2 人才管理的三个常见误区

**误区1：把人才管理等同于绩效考核**
真相：绩效管理只是人才管理的一个环节。缺乏能力体系（任职资格）和领导力培养（干部管理），绩效考核就变成了纯粹的"奖罚工具"，失去了"发展人才"的功能。

**误区2：强制分布导致内部竞争激烈**
真相：强制分布必须与团队协作文化并行。华为有明确的规则：损害团队利益的个人行为，在价值观评价维度会受到负面评价，从而抵消绩效的正向评价。

**误区3：干部管理只是HR的事**
真相：干部管理是一把手工程，直线经理对团队成员的成长负有主要责任。HR只是提供工具和流程，管理者才是真正的"人才培养主体"。

## 六、薄云咨询的应用建议

### 快速诊断切入点
- 客户是否有清晰的岗位职级体系？晋升标准是否透明？
- 绩效考核结果是否真正区分了优秀和一般？年终奖差距有多大？
- 公司是否有系统性的干部培养机制，还是靠"自然生长"？

### 典型改进项目
1. **任职资格体系建设**：适合HR体系薄弱的成长型企业，6-12个月周期
2. **绩效管理改善**：将"走过场"的KPI考核升级为真正有牵引性的PBC体系
3. **干部盘点与培养计划**：结合九宫格工具，帮助企业建立系统性的领导力发展机制
