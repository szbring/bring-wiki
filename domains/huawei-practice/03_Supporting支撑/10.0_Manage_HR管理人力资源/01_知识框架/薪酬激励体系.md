# 华为薪酬激励体系

<!-- chunk: 薪酬激励体系.核心价值 -->
## 核心价值

> **「[[以奋斗者为本_Ch1_全力创造价值|以奋斗者为本]]」不是口号，而是用薪酬机制把利益向奋斗者倾斜** — 拉开差距，让贡献大的人获得回报，让混日子的人没有舒适区。

---

<!-- chunk: 薪酬激励体系.薪酬总包结构 -->
## 薪酬总包结构

华为员工薪酬由四大模块构成：

| 模块 | 性质 | 占比（中高级员工） | 决定因素 |
|---|---|---|---|
| 固定薪酬 | 短期/保障 | 30-40% | 任职资格级别 |
| 绩效奖金 | 短期/激励 | 20-30% | PBC绩效等级 |
| 虚拟受限股（TUP） | 中长期/激励 | 20-30% | 绩效+任职级别 |
| 内部持股 | 长期/绑定 | 10-20% | 资历+绩效+贡献 |

**设计原则**：
- 固定薪酬保生存，绩效奖金拉差距，股权激励绑长期
- 级别越高，股权占比越大，与公司长期利益深度绑定

<!-- chunk: 薪酬激励体系.固定薪酬-带宽制 -->
## 固定薪酬：带宽制

### 基于任职资格级别的薪酬带宽

| 任职级别 | 薪酬带宽（相对值） | 带宽幅度 | 中位值定位 |
|---|---|---|---|
| T1/M1 | 1.0x | - | 市场P40-P50 |
| T2/M2 | 1.3-1.8x | 38% | 市场P50-P60 |
| T3/M3 | 1.8-2.8x | 56% | 市场P60-P75 |
| T4/M4 | 2.5-4.0x | 60% | 市场P75 |
| T5/M5 | 3.5-6.0x | 71% | 市场P75-P90 |
| T6/M6 | 5.0x+ | - | 市场P90+ |

**带宽设计要点**：
- 同级别内上下浮动30-40%，给绩效差异化留空间
- 相邻级别有重叠（约20-30%），避免晋升即大幅涨薪的刚性预期
- 员工薪酬在带宽内的位置由绩效决定：A级在上沿，B级在中位，C级在下沿

### 薪酬调整机制

| 触发条件 | 调整方式 | 幅度 |
|---|---|---|
| 任职级别晋升 | 跨带宽调整 | 20-40% |
| 年度绩效A级 | 带宽内上调 | 10-15% |
| 年度绩效B级 | 带宽内微调 | 3-5% |
| 年度绩效C级 | 不调整 | 0% |
| 市场薪酬变化 | 带宽整体调整 | 参考市场调研 |

<!-- chunk: 薪酬激励体系.绩效奖金-差异化激励 -->
## 绩效奖金：差异化激励

### 奖金计算逻辑

```
个人奖金 = 部门奖金包 × 个人分配系数
部门奖金包 = 公司奖金总包 × 部门绩效系数
个人分配系数 = 个人绩效等级对应系数 × 任职级别权重
```

### 绩效等级与奖金系数

| 绩效等级 | 奖金系数 | 说明 |
|---|---|---|
| A | 2.0-3.0x | 远超平均，重奖 |
| B+ | 1.3-1.5x | 超过平均 |
| B | 1.0x | 平均水平 |
| C | 0.3-0.5x | 低于平均，惩罚性 |
| D | 0 | 无奖金 |

**关键机制**：
- 部门奖金包与部门绩效挂钩，避免「旱涝保收」
- 个人系数与级别权重相乘，高级别高绩效者奖金远超平均值
- A级与C级奖金差距可达5-10倍

## 虚拟受限股（TUP）：中长期激励

### TUP机制详解

TUP（Time Unit Plan）是华为2013年推出的虚拟股权激励，解决老员工「食利阶层」问题：

| 年度 | TUP权益 | 说明 |
|---|---|---|
| 第1年 | 0% | 锁定期，无分红权 |
| 第2年 | 33% | 开始享受分红 |
| 第3年 | 66% | 分红权益递增 |
| 第4年 | 100% | 享受全额分红 |
| 第5年 | 100%+增值收益 | 到期结算，重新分配 |

**TUP vs 老虚拟股对比**：

| 维度 | 老虚拟股 | TUP |
|---|---|---|
| 持有期限 | 长期（在职期间） | 5年周期 |
| 增值收益 | 跟随公司增值 | 到期结算增值 |
| 食利风险 | 老员工坐吃分红 | 5年周期，不奋斗就不再分配 |
| 适用对象 | 老员工为主 | 全体奋斗者 |

## 内部持股计划

### ESOP（员工持股计划）

- 华为不是上市公司，通过内部持股让员工成为公司主人
- 持股资格：绩效连续B+以上、任职T3/M3以上、司龄3年以上
- 持股上限：基于级别和绩效设定个人持股上限
- 退出机制：离职时公司回购，按净资产估值

**持股的意义**：
- 把员工从「打工人」变成「合伙人」
- 公司高速增长期，股权收益远超工资和奖金
- 绑定核心人才，离职成本极高

## 薪酬激励的设计原则

### 六大原则

| 原则 | 内涵 | 反面教训 |
|---|---|---|
| [[以奋斗者为本_Ch1_全力创造价值|以奋斗者为本]] | 利益向贡献者倾斜 | 大锅饭，干多干少差不多 |
| 拉开差距 | A级与C级薪酬差3-5倍 | 最高最低差不到20% |
| 长短结合 | 固定+奖金+股权三位一体 | 只发工资无激励 |
| 向外看 | 薪酬对标市场P75 | 内部平衡论，不顾市场 |
| 动态调整 | 绩效决定涨薪/降薪 | 涨上去就降不下来 |
| 总包管控 | 部门薪酬包与绩效挂钩 | 零和博弈，不管业绩 |

## 薄云咨询应用场景

**场景1：薪酬体系搭建**
- 客户特征：薪酬无标准，谈薪靠谈判，内部不公平
- 方案：任职资格定级+薪酬带宽设计+绩效挂钩机制
- 周期：3-4个月

**场景2：激励机制优化**
- 客户特征：大锅饭，高绩效与低绩效薪酬差距小
- 方案：强制分布+差异化奖金+股权激励设计
- 周期：2-3个月

**场景3：股权激励设计**
- 客户特征：上市前需设计员工持股/期权方案
- 方案：TUP或期权方案设计+行权条件+退出机制
- 周期：2-3个月
