# 华为激励机制

## 核心概念与定义

华为激励机制的核心是"[[以奋斗者为本_Ch1_全力创造价值|以奋斗者为本]]"的价值分配体系。企业经营的本质是利益分配——分好了人就充满动力，分不好就熵增走向死亡。华为最高战略是让组织充满活力，通过"分利、分权、分名"让个人充满动力。华为的激励不是简单的提成制，而是"获取分享制"——团队从创造的价值中获取、在团队内部分享，导向冲锋和协作而非个人英雄主义。

## 方法论框架

### 一、企业经营概念方程式

企业持续成功 = 战略 × 组织能力

组织能力 = 员工思维（愿不愿意干）× 员工能力（能不能干）× 员工治理（容不容许干）

激励机制解决的是"愿不愿意干"的问题。

### 二、落后分钱方法（业绩提成制）的六大问题

| 问题 | 表现 |
|------|------|
| 老板不讲诚信 | 提成比例随时调，员工不信任 |
| 团队不稳定 | 个人收益与团队无关，士气低落 |
| 不愿开拓新市场 | 新市场难出业绩，提成少 |
| 客户资源私有化 | 客户成个人资产，公司被绑架 |
| 不愿培养新人 | 新人成长威胁老员工利益 |
| 客户结构恶化 | 追逐容易签单的低价值客户 |

### 三、华为激励体系框架

| 层次 | 类型 | 工具 | 导向 |
|------|------|------|------|
| 物质激励 | 短期 | 工资+奖金 | 当期贡献 |
| 物质激励 | 中期 | 绩效奖金+专项奖 | 年度绩效 |
| 物质激励 | 长期 | TUP/虚拟受限股/ESOP | 长期奋斗 |
| 权力激励 |  | 干部任命+项目授权 | 成就感+影响力 |
| 精神激励 |  | 荣誉+认可+使命认同 | 内在驱动 |

### 四、获取分享制

**核心逻辑**：团队创造的价值→按规则获取→在团队内部分享

- **获取**：奖金包=收入×比例+利润×比例+超额分享，与创造的价值挂钩
- **分享**：部门内按个人绩效系数分配，跨部门按贡献度分配
- **导向**：团队协作（不搞个人提成）+导向冲锋（增量分配）+不让奋斗者吃亏

### 五、薪酬包弹性管理

总薪酬包 = 工资包 + 奖金包 + 长期激励包

薪酬包与销售收入/利润挂钩（弹性控制）：
- 收入增长→薪酬包增长→员工收入增长
- 收入下降→薪酬包缩减→倒逼提效减人

三个人干五个人的活，拿四个人的工资——增量分配导向。

### 六、股权激励三阶段

| 阶段 | 时间 | 方式 | 特点 |
|------|------|------|------|
| 全员持股 | 1990-2001 | 每股1元，15%分红 | 内部融资+员工归属感 |
| 虚拟受限股 | 2001-2008 | 期权改革，4年兑现 | 应对IT泡沫，控制风险 |
| 饱和配股 | 2008至今 | 按职级上限配股+TUP | 避免食利阶层，增量激励 |

### 七、多元化激励"五多"

1. **多重**：工资+奖金+股票+福利+专项奖，短中长期结合
2. **多次**：月度/季度/年度/项目里程碑，及时激励
3. **多种**：物质+权力+荣誉+成长，不只有钱
4. **多人**：团队激励+个人激励，不只奖个人
5. **多场景**：日常/项目/战役/创新，场景化激励

### 八、企业之熵与活力引擎

**熵增**：组织天然趋向混乱、僵化、惰怠
**活力引擎**：通过激励+流动+淘汰持续做熵减

- 激励：分钱/分权/分名（开放做功）
- 流动：干部轮岗/人才流动（打破平衡）
- 淘汰：末位10%淘汰（挤压激活）

## 关键流程/步骤

1. **价值创造→评价→分配闭环**：为客户创造价值→责任结果评价→按贡献分配
2. **奖金包计算**：组织奖金包=基础包×绩效系数+增量分享→部门二次分配→个人按PBC系数分配
3. **薪酬调整**：年度调薪包=部门包×绩效系数→个人调薪=基准×职级系数×绩效系数
4. **股权激励**：年度配股额度=职级饱和度-当前持股→TUP5年分期兑现

## 核心原则

1. **[[以奋斗者为本_Ch1_全力创造价值|以奋斗者为本]]**：不让奋斗者吃亏，不让雷锋穿破衣服。激励向奋斗者、突出贡献者倾斜。

2. **获取分享而非提成**：团队从创造的价值中获取，团队内分享。导向协作和增量，而非个人私分和存量。

3. **增量分配导向冲锋**：激励方案围绕"挣钱"设计，做大增量蛋糕，三个人干五人活拿四人工资。

4. **物质激励+精神激励双轮驱动**：钱给多了不是人才也变成人才，但光靠钱不够，还要使命认同和成长空间。

5. **饱和配股防止食利阶层**：持股按职级设上限，TUP5年分期兑现不永久持有，避免老员工躺在新员工劳动成果上食利。

## 典型案例与实践要点

**案例1：从提成制到获取分享制**。提成制导致销售个人英雄主义、客户私有化、不愿开拓新市场。华为改为获取分享制：团队创造价值→获取奖金包→团队内按贡献分享。效果：团队协作增强、新市场开拓意愿提升、客户资源公司化。

**案例2：饱和配股**。2008年经济危机期推出饱和配股：13级上限2万股、14级上限5万股，达到上限不再配股。目的：避免早期员工大量持股形成食利阶层（每年分红躺着赚），让新增激励额度投向新贡献者。后来又推出TUP（5年分期兑现），进一步防止股权沉淀。

**案例3：熵减与活力**。任正非引入"熵"的概念：企业天然熵增（混乱+僵化+惰怠），必须持续做熵减（激励+流动+淘汰）。"深掏滩低作堰"——内部持续挖出淤泥（淘汰惰怠），对外低作堰（舍得分钱给合作伙伴和员工）。

**实践要点**：
- 奖金分配"拧麻花"：销售奖金与交付质量挂钩，交付奖金与销售满意度挂钩，防止各自为政
- 专项奖：重大突破/关键里程碑/创新贡献等即时激励
- 荣誉体系：金牌奖/明日之星/总裁奖等精神激励
- 激励可见性：让员工清晰看到贡献与回报的关系
- 薪酬包弹性控制：收入降则薪酬降，倒逼提效减人

## 与华为其他体系的关联

- 激励与绩效管理联动：PBC考核结果决定奖金系数和调薪
- 激励与干4部管理联动：干部的激励与组织绩效挂钩，非个人提成
- 股权激励与任职资格联动：职级决定配股上限
- 激励与战略解码联动：战略解码输出的组织KPI决定奖金包
- 激励与组织建设联动："拧麻花"机制打破部门墙，跨部门奖金互挂
