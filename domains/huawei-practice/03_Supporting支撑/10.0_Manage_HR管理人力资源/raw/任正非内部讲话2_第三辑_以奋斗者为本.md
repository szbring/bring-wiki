# 以奋斗者为本：让雷锋先富起来

## 3.1 以奋斗者为本的核心内涵

- 奋斗者是华为价值的创造者
- 让奋斗者得到合理回报
- 不让雷锋吃亏，不让焦裕禄穿破鞋

## 3.2 价值创造→价值评价→价值分配

```
价值创造（全力创造价值）
    ↓
价值评价（正确评价价值）
    ↓
价值分配（合理分配价值）
    ↓
激发更大奋斗 → 回到价值创造
```

## 3.3 让雷锋先富起来的机制

- 股权激励让奋斗者成为股东
- 绩效考核向奋斗者倾斜
- 破格提拔让有贡献的人快速晋升

## 3.4 利益分享与捆绑

- 员工持股形成利益共同体
- 获取分享制：先创造再分享
- 利益捆绑形成命运共同体

## 核心要点

1. **奋斗者=价值创造者**：不是加班者，是创造价值的人
2. **不让雷锋吃亏**：精神奖励+物质奖励双保障
3. **价值闭环**：创造→评价→分配→更大创造
4. **利益共同体**：员工持股让个人与企业命运绑定
