# 绩效考核体系：成果比过程更重要

## 一、中高层述职 + KPI考核

华为对中高层绩效管理通过**述职 + KPI考核**完成：年终述职全面检查上一年工作，制定下一年KPI的措施与策略。

### 1. 述职方式

逐级向上，多为中期述职：
- 总裁→董事会
- 委员会负责人/部门正职→总裁
- 部门副职→各委员会
- 二级部门负责人→部门正职

每季度第一个月中旬开始述职，有利于定期审视和评估。

### 2. 述职模型——综合平衡计分卡

将公司战略分解为四个方面：

| 维度 | 内容 |
|------|------|
| 客户 | 客户满意度、内部客户满意度 |
| 财务 | KPI完成情况、与竞争对手比较、成绩和不足 |
| 内部流程 | 部门业务策略、核心竞争力提升措施、部门重点工作、项目实施 |
| 学习与成长 | 职业化及技能提升、组织氛围营造 |

### 3. 述职内容（八个方面）

1. **目标完成情况**：按最主要、主要、次要顺序列出不足和成绩，分析原因
2. **外部环境分析**：比较客户、竞争对手和自己的地位差异，评估风险与机会
3. **KPI实现程度**：报告近几期KPI完成情况并对比，分析差距和原因
4. **提升核心竞争力的措施**：完成KPI和增强管理潜力的措施计划
5. **客户满意度分析**：需有准确数据，分析满意与否的原因
6. **学习与成长**：重大管理项目推进情况、员工技能提升效果、组织氛围指数
7. **预算与KPI承诺**：提出挑战目标并做出承诺
8. **反馈意见**：列出所需人、财、物及技术资源

## 二、中基层员工KPI考核

### 考核原则

1. **责任结果导向原则**：以最终结果为导向，采取高效措施
2. **目标承诺原则**：充分沟通确立绩效目标，员工做出承诺
3. **考核与评价相结合原则**：直接管理者考核+功能部门评价
4. **客观性原则**：以日常观察记录为基础，用数据和事实说话

### 部门负责人考核内容

- 部门量化指标（可量化的关键业绩指标）
- 部门非量化指标（不可量化但重要的指标）
- 追加目标和任务考核
- 工作行为与态度考核
- 管理行为考核
- 不良事故考核

### 考核注意事项

- 注意过程记录、绩效辅导和及时反馈
- 考核者如实记录在"行为指导记录"中
- 考核结束后与员工面谈，分析绩效优劣，总结改进措施
- 被考核员工对结果不满意有权投诉（一周内向上级或HR投诉）

## 三、平衡计分卡（BSC）

BSC包含财务、客户、内部流程、学习与成长四个方面。华为实行BSC考核中高层员工，改变了只重视财务指标的做法。

### BSC打造的五种平衡

1. **财务指标和非财务指标的平衡**
2. **长期目标和短期目标的平衡**
3. **结果性指标与动因性指标之间的平衡**
4. **企业内部群体与外部群体的平衡**
5. **领先指标与滞后指标之间的平衡**

### 构建步骤

1. **设定财务目标**：以战略计划相关的财务措施为基础
2. **面向客户**："如果我们打算完成财务目标，客户必须怎样看待我们"
3. **明确内部过程**：确认向客户转移价值所必需的内部过程

### 三个原则

- 因果关系原则
- 成果量度与绩效驱动因素原则
- 与财务联结原则

## 四、EVA考评

**经济增加值（EVA）**= 税后净营运利润 - 投入资本

### EVA的特点

- 注重资本费用，体现企业真正盈利能力
- 一个指标联系所有过程：如何增加EVA
- 考虑所有资本的机会成本，避免操纵绩效
- 以价值创造为核心，反映资产创造价值的能力

### 华为EVA实践：虚拟利润分享制

2000年，华为选择全资子公司华为电气做试点：
- **虚拟利润**= 剩余收益 - 员工实发工资 - 投资性费用
- 虚拟利润 × 分配系数 = 华为电气全体员工下一年度薪酬总额
- 改变了追求规模扩张的片面性，调动全体员工积极性
- 被称为"中国企业成功资本运作第一例"

### EVA实施要点

1. **合理划分EVA中心和非EVA中心**：只有能确认收入成本费用、区分占用资本责任的部门才可用EVA衡量
2. **以EVA为核心构建全面考核指标体系**：EVA作为核心考核，保留KPI作为参考
3. **制定侧重EVA兼具全面性的经理人薪酬结构**：部分薪酬与EVA直接联系，同时结合其他指标

## 五、360度考评

360度考评从员工本人、上司、下属、同事各角度了解个人绩效。

### 四个评价维度

1. **自我评价**：让员工清楚认识自己，设定下一个绩效目标
2. **同事互评**：同事之间了解更深，互评更客观
3. **下属评价**：上级可对照自我评价了解管理能力差距
4. **主管评价**：最了解员工工作表现和效果

### 华为的360度考评特点

- **矩阵式绩效评估**：一个员工同时有几个部门的主管，综合考量各主管评价
- 避免个人主观性、趋同性心理
- 增加员工自主性及对组织认同性

### 360度考评的注意事项

- **能力开发才是价值所在**：不应简单用于人才评估，应与职业发展规划结合
- **需要高层领导支持**：涉及组织各层面人员
- **适用于企业稳定发展阶段**：重组、合并、大裁员时期不宜采用
- **匿名考核**可规避个人情感因素
- **利用统计技术和软件**节约成本

## 六、绩效评价ABC

华为员工绩效评价分为A、B、C三个档次：

| 档次 | 比例 | 说明 |
|------|------|------|
| A | 约5% | 优秀 |
| B | 约45% | 良好 |
| C | 约45% | 合格 |
| 待查 | 约5% | 末位 |

- 实行每月考评一次
- 每个档次之间年总绩效奖金差别在5000元以上
- 客户满意度是总裁到各级干部的重要考核指标
- 绩效目标以客户需求为导向，贯穿绩效管理全过程
