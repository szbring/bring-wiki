# 绩效目标解码：目标是一切管理的开始

## 核心命题

专注的目标管理系统和严谨的项目管理模式，让华为面对繁杂的项目开发和系统建设时做到高质、高效，迅速抢占市场。

## 一、工作目标必须明确

大多数工作目标无法实现的原因是目标模棱两可，不能给人具体的行动指引。目标可以是"跳起来摘桃"，但不能是"跳起来摘星星"。

### 三大量化指标

华为在目标管理中注重**时量、数量、质量**三个关键量化指标，贯穿工作完成全过程：

- **时量**：完成工作的时间要求
- **数量**：完成工作的数量指标
- **质量**：完成工作的程度和标准

**示例**："4小时内完成20000人的考勤统计，形成考勤表，上报行政主管"——完整包含三个量化指标。

### 量化与质化原则

华为遵循**"能量化的量化，不能量化的质化"**原则：

- 可直接量化的目标从数量角度衡量（产品生产数量、检查次数等）
- 无法直接量化的目标从质量、时量角度考虑（如人员投诉率、服务及时性、文件通过率等）
- 杜绝使用"基本了解"、"控制"、"良好"等模糊描述

## 二、PBC绩效管理（个人绩效承诺）

Personal Business Commitment（PBC）是华为绩效管理的重要工具，包括四个循环步骤：

### 1. 绩效目标

PBC模板包含三大承诺：
- **结果目标承诺（Win）**
- **执行措施承诺（Execute）**
- **团队合作承诺（Team）**

绩效内容四个来源：
1. 公司长期目标和部门整体目标
2. 该职位的职责
3. 业务流程的目标
4. 相关部门之间协作的要求

绩效目标自上而下层层分解，将企业目标分解到部门，团队目标分解到个人。

**绩效等级划分**：
| 等级 | 定义 |
|------|------|
| 杰出 | 经常显著超出预期，各方面都取得特别出色成绩 |
| 良好 | 达到或部分超过预期，主要方面取得出色成绩 |
| 正常 | 基本达到预期，无明显失误 |
| 需改进 | 未达到预期，多方面存在明显不足或失误 |

### 2. 绩效辅导

伴随绩效管理始终，关键在于**有效沟通**：
- 讨论工作进展、潜在障碍和问题
- 帮助员工明确职位、职责和目标
- 前瞻性地发现并解决问题
- 适时反馈，让员工明白自己所处位置并及时调整

### 3. 绩效评价

以事实和数据说话，定量与定性相结合：
- 项目组是绩效评价者，提供事实依据
- 直接主管是考核责任者，综合评价意见得出结果
- 上级主管是考核复核者，监督指导和沟通协调
- 所有考核结果告知员工，确认无异议后才最终确定

### 4. 绩效反馈

根据考核结果差异化处理：
- 奖励高绩效员工、鼓励良好和正常绩效员工、辅导督促低绩效员工

**华为荣誉体系**：个人荣誉奖、集体荣誉奖、金牌奖。个人金牌奖一年一次，为公司对个人奖励最高级别。

**纪律处分八种**：通报批评、警告、严重警告、罚款、降薪、降级、下岗、辞退。

## 三、SMART标准

绩效目标设定必须符合SMART标准：

| 原则 | 要求 | 华为实践 |
|------|------|---------|
| **Specific（明确性）** | 目标必须明确 | 采用5W1H法 |
| **Measurable（可度量）** | 目标可以衡量 | 细化或流程化 |
| **Actionable（可实现）** | 目标可以达成 | 充分调研基础上设定 |
| **Realistic（结果导向）** | 局部目标符合整体目标 | 个人目标符合团队目标 |
| **Time-based（时间限定）** | 有明确时间要求 | 节约时间=提高效率 |

### 5W1H法详解

- **What**：目标是什么——明确制定目的、前提条件、操作方法、执行重点
- **When**：什么时候执行——确定具体完成时间，留出意外状况时间，做好应急预案
- **Where**：选择什么地点——发挥"地利"优势，节约成本
- **Who**：选择最适宜人选——不同工作选最适合的员工
- **Why**：这样操作的原因——先进行可行性研究
- **How**：怎样做最有效率——重视有效的和省时的工作方法

## 四、目标管控与量化到位

时间量、数量、质量是目标可量化的三个关键指标。量化指标可进一步细化为分值、百分比（如客户满意度、产品达标率、客户投诉率）。

华为考核数据标准：**简单明了、精确、易操作**，保证数据客观性，公正表达目标执行预期，为执行提供衡量标准。

## 五、目标任务计划可视化

高效能的人比普通人具有更强的计划、统筹和执行能力。

### 华为工作清单方法

1. 按地点（电脑旁、办公室、电话旁、仓库）分别记录任务
2. 尽可能详细记录，避免遗漏
3. 在手写物品上注明日期

### 任务整理三原则

1. 从最上面一项开始处理，尽可能迅速完成，不逃避
2. 一次只处理一件事情，迫使集中注意力
3. 决定开始处理时立刻判定实质及处理方法，杜绝二次处理的时间浪费

## 六、华为目标体系六大板块

目标与实现期限挂钩，分成日、周、月、季、年、n年六个时间段：

1. **发展规模目标**：盈利规模、员工规模等宏观目标
2. **行业发展目标**：市场增长率、经营收入额、产品线深度/宽度、专利个数、投资回报率等
3. **产品市场发展目标**：市场区域增长率、销售收入、市场占有率、销售利润率
4. **资金战略目标**：总资产/固定资产/流动资产/货币资金增长率、负债比例等
5. **人才战略目标**：人才的引入、规划、培训、指导
6. **创新战略目标**：产品创新、工艺创新、市场创新、管理创新；方式包括自主创新和引进创新
