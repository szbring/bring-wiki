# 利益分享：华为员工持股的哲学

## 核心命题

与奋斗者分享利益是华为[[以奋斗者为本]]企业文化的体现，也是华为价值管理的重要环节。与奋斗者分享利益，既是对奋斗者绩效的回报，也是对华为未来绩效的投资。

## 一、员工持股制度——利益共同体

华为几乎从一开始就实行全员持股，1997年深圳市政府出台相关暂行规定后对外公开并改制。

### 股权结构

- 30%优秀员工集体控股
- 40%骨干员工有比例地持股
- 10%-20%低级员工和新员工适当参股
- 员工持有股份根据"才能、责任、贡献、工作态度、风险承诺"动态调整
- 员工离开公司时可随时套现

### 全员持股的优势

1. 员工利益与公司利益捆绑
2. 获得税收优惠
3. 避免恶意收购，增加现金流
4. 吸引人才的办法

任正非在华为股份不到5%，愿意将财富分配给员工，体现责任观。

### 华为不上市的五个原因

1. 股权太分散（股东超过7万人，法规限制200人）
2. 任正非占股比例太小，上市会失去控制权
3. 上市会产生成千上万个千万或亿万富翁，与"长期坚持艰苦奋斗"核心价值观背道而驰
4. 上市公司季报年报的短期财务指标要求，与华为以10年为单位的规划模式相悖
5. 华为最不缺钱

"小富快跑，暴富跌倒"——上市暴富与艰苦奋斗核心价值观完全背道而驰。

## 二、分配体系向奋斗者倾斜

任正非："效益优先、兼顾公平是市场经济的特点。社会要富裕，必须有火车头拉着跑。火车头拉的时候要有动力，这个动力就是差异。"

### 差异化策略四个方面

1. **工资分配实行基于员工能力的职能工资制**：根据[[华为任职资格体系]]体系区分能力，任职资格不同，工资不同
2. **奖金分配与员工个人和部门绩效挂钩**：绩效评价分为不同等级，相应获得不同奖金（股权）分配
3. **福利分配以工作态度等考评结果为依据**：围绕企业文化（奉献精神、服从调动、到艰苦一线、自我批评、团队协作）
4. **医疗保险待遇按贡献确定**：高级管理者、普通管理者、一般员工实行差别待遇

### 三位一体的动态结构

1999年起华为实行**绩效管理体系+薪酬分配体系+任职资格评价体系**三位一体：
- 按照员工责任和贡献确定任职资格
- 根据任职资格确定员工的职能工资
- 根据个人和部门绩效完成水平确定奖金（股权）分配

### 2006年薪酬制度改革

"以岗定级、以级定薪、人岗匹配、易岗易薪"：
- 根据岗位责任和贡献产出确定工资级别
- 员工匹配上岗获得相应工资待遇
- 岗位调整则工资待遇随之调整

### 打破平衡

2009年EMT纪要：率先在奖金[[华为的激励机制]]上打破平衡——打破跨区域平衡、区域内部平衡、人与人之间平衡。明确5%的人员奖金为零。

## 三、三次股权激励变革

### 1. 创业期股票激励（1990年）

- 每股10元，以利润15%作为股权分红（税后）
- 员工进入公司一年后依据职级、季度绩效分配
- 用年度奖金购买，不够则公司帮助银行贷款
- 增强员工归属感和主人翁意识

### 2. "华为冬天"时期虚拟受限股（2000年）

IT泡沫破灭后实行"虚拟受限股"期权改革：
- 新员工不再分配一元一股的股票
- 老员工股票逐渐向期股转化
- 取消固定分红，从期股对应的公司净资产增值部分获取
- 期权期限4年，每年兑现1/4
- 四种兑现方式：兑现差价、以1元/股购买、以后兑现、放弃
- 持股较多的核心员工每年兑现不超过1/10
- 配股锁定期限内离开则所配股票无效

### 3. 经济危机时期饱和配股（2008年）

- 依据职位级别分配不同期股额度
- 不同职位级别匹配不同期股量（如13级上限2万股，14级5万股）
- 持股已达级别上限不参与配股
- 缓解资金压力+提高员工收益+绩效与职位等级挂钩

## 四、虚拟饱和配股制度

### 核心原则

华为配股从虚拟受限配股改革为虚拟饱和配股，目的是给予优秀员工更多配股机会，贯彻"始终按贡献大小定待遇"方针。

### 三项执行要点

1. **不按条文简单区分**：实事求是评价每个员工贡献，让真正出色员工的利益最大化
2. **识别奋斗者与非奋斗者**：区分优秀奋斗者和普通奋斗者，配股不是单纯奖励
3. **立足于劳动绩效评估**：根据工作表现和贡献给予合理比例配股额度

### 确保期股落实到奋斗者

1. **资源共享，确保评价准确**：行政部门提供奖惩数据、人力资源部提供考勤状况、主管提供工作态度状况
2. **完善日常管理机制**：工作报表制度、出勤制度、汇报制度
3. **公示、培训掌握工作方法**：绩效管理部门与各级主管充分沟通

### 执行原则

- 优秀奋斗者：给予饱和配股
- 一般奋斗者：给予限额配股
- 不愿奋斗者：不给予配股

## 五、华为价值分配体系

### 价值分配五要素

| 要素 | 内容 |
|------|------|
| **原则** | 按劳分配与按资分配相结合，比例可调节 |
| **对象** | 组织权力和经济利益 |
| **依据** | 才能、责任、贡献、风险承诺、工作态度 |
| **实现形式** | 职权、工资、奖金、红利、股权、机会、福利及其他人事待遇 |
| **确定依据** | 职能工资制、与利润挂钩的奖金、与贡献责任挂钩的分配、以态度分配退休待遇、以贡献确定医疗保险、以贡献责任和工时确定股权 |

### 评价标准

- 企业整体业绩是否提高及提高快慢
- 是否高于行业平均速度和竞争对手速度
- 是否调动员工积极性
- 员工是否认同公司价值理念和文化，形成"利益、命运的共同体"
