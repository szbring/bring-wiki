<!-- chunk: 00-概述.四-华为变革史全景 -->
## 四、华为变革史全景

华为发展史本质上是一部变革史，四个阶段的管理变革逐步构建了世界级的组织能力：

| 阶段 | 时间 | 标志性变革 |
|------|------|---------|
| 创业生长 | 1987–1997 | 《华为基本法》、市场部大辞职 |
| 规模扩张 | 1998–2005 | IPD变革、ISC变革、FE项目、财务四统一 |
| 迈向世界级 | 2006–2013 | LTC变革、IFS变革（8年）、HR三支柱、BLM |
| 做产业领先者 | 2014–今 | IPD+/CRM+集成变革、ISC+数字化、HIS平台 |

> 详细内容见：`01_知识框架/华为变革史总览.md`

## 七、与其他模块的关系

本模块（BT+IT）是华为管理体系的"元能力"——变革能力本身，是其他所有模块得以成功推行的前提：

```
本模块：12.0_Manage_BTIT
（变革与IT治理能力）
         ↓ 支撑
┌────────────────────────────────────────┐
│  其他华为管理模块的成功推行              │
│                                        │
│  IPD集成产品开发  →  需要BT推行变革     │
│  LTC线索到回款   →  需要BT推行变革     │
│  IFS集成财务服务 →  需要BT推行变革（8年）│
│  HR三支柱体系    →  需要BT推行变革     │
│  ISC+数字化      →  需要IT架构+BT协同  │
└────────────────────────────────────────┘
```

**与IPD模块的关系**：IPD是最重要的关联模块。FE项目（1999年）正是为了支撑IPD落地而建立的，两个模块在历史上是协同推进的。

**与HR模块的关系**：变革管理的"人的差距"与HR模块高度交叉，人员能力建设、绩效管理设计需要HR模块的专业支撑。

**与财经模块的关系**：IFS（集成财务服务）是BT体系下的具体变革项目，也是财经模块的核心内容，两模块共享部分知识资产。
