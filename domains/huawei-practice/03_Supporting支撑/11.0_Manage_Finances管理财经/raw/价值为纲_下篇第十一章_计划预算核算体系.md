# 加强计划预算核算体系建设

## 计划-预算-核算的关系

```
战略规划(SP) → 年度业务计划(BP) → 全面预算 → 核算分析 → 绩效评价
     ↑                                                    ↓
     └────────────── 滚动调整 ←────────────────────────────┘
```

## 全面预算体系

- **自上而下**：战略目标分解
- **自下而上**：业务单元提报
- **上下结合**：多轮沟通达成共识

## 滚动预测

- 不做一年一度的静态预算
- 按月/季度滚动预测，动态调整
- 预测是管理工具，不是考核工具

## 核算分析

- 核算是预算执行的结果反映
- 预算与核算的差异分析驱动管理改进
- 核算数据要及时、准确、完整

## 核心要点

1. **计划是龙头**：没有好的计划就没有好的预算
2. **预算是工具**：预算服务于业务，不是限制业务
3. **核算是基础**：准确核算是分析和改进的前提
4. **滚动调整**：环境变化需要动态调整预算
