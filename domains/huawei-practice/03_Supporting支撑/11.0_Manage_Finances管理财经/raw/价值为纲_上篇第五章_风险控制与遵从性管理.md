# 加强风险控制与遵从性管理

## 5.1 扩张的同时必须能够控制得住

> 控制是为了更有效地扩张，不是为控制而控制。

- 不能只看到机会而忽视风险
- 追求有效增长，抑制盲目扩张
- **保持扩张与控制的平衡**

## 5.2 华为财务风险控制的多维运作机制

### 5.2.1 "4个3"运作机制

华为风险控制的核心框架"4个3"：

| 维度 | 内容 |
|------|------|
| 3道防线 | 业务一线/流程owner → 合规/风控部门 → 内部审计 |
| 3个层级 | 公司级 → 区域级 → 国家级 |
| 3个环节 | 事前预防 → 事中控制 → 事后监督 |
| 3类风险 | 战略风险/运营风险/财务风险 |

### 5.2.2 财务风险控制的两个视角

- **经营视角**：保障业务健康发展
- **合规视角**：确保合法合规运营

### 5.2.3 三角联动机制

财务、业务、风控三方联动，形成风险控制的闭环。

## 5.3 金融风险防控

- 要有应对金融危机的预案
- **提高合同质量是最根本的措施**
- 多元化融资和多元化结算
- 不因风险就不前进，也不因前进而不顾风险

## 5.4 业务连续性管理

- 维护网络稳定是职业操守
- 建立战略储备，防范供应风险
- 避免战乱、动乱、疫区国家落入应急触发式管理循环

## 5.5 以法律遵从的确定性应对国际政治的不确定性

- 法律遵从是在全世界生存、服务、贡献最重要的基础
- 像重视网络安全一样关注用户隐私保护

## 5.6 合规管理与子公司董事会的监督职责

- 加强合规管理，多打粮食
- 子公司董事会代表公司一线实地综合监督，但不干预一线业务运作

## 核心框架

```
风险控制体系
├── 4个3运作机制（三道防线/三个层级/三个环节/三类风险）
├── 两个视角（经营+合规）
├── 三角联动（财务-业务-风控）
├── 金融风险防控（合同质量+多元融资）
├── 业务连续性（网络稳定+战略储备）
├── 法律遵从（确定性应对不确定性）
└── 合规管理（子公司董事会监督）
```
