# 面向端到端业务流程的财经管理

## 核心概念：端到端

财经管理不是孤立的，而是嵌入端到端业务流程中：

| 流程 | 财经切入点 |
|------|-----------|
| **OTC（订单到回款）** | 合同财经评审→发货控制→开票→回款 |
| **[[IPD]]（产品开发）** | 产品成本核算→投资回报分析→定价 |
| **PTP（采购到付款）** | 采购预算→应付管理→付款控制 |
| **[[ITR]]（问题到解决）** | 保修成本→服务成本核算 |

## 8.1 OTC流程的财经管理

- 合同签订前的财经评审
- 发货和收入确认控制
- 应收账款管理和回款

## 8.2 IPD流程的财经管理

- 产品成本核算和定价
- 研发投入和投资回报分析
- 产品盈利性管理

## 8.3 PTP流程的财经管理

- 采购预算和供应商付款管理
- 应付账款和现金流管理

## 核心要点

- **业财融合**：财经管理嵌入业务流程，而非事后核算
- **端到端**：从业务起点到终点的全流程财经管控
- **过程控制**：在过程中控制，不是事后审计
