# 财经管理诊断工具

<!-- chunk: 财经管理诊断.成熟度评估 -->
## 成熟度评估

| 级别 | 特征 |
|---|---|
| L1 | 财务=记账，月底出报表，无法支撑实时决策 |
| L2 | 有财务预算，但执行追踪滞后，预测不准 |
| L3 | 月度管理报告，财务BP开始支持业务决策 |
| L4 | 实时经营看板，财务数据驱动资源配置 |
| L5 | 财务深度嵌入业务，超前预警，AI辅助预测 |

---

<!-- chunk: 财经管理诊断.快速诊断问题 -->
## 快速诊断问题

1. **「管理层拍决策时，能看到实时的经营数据吗？」**
   - 靠感觉和月报 → L1-L2
   - 有月报但滞后 → L2
   - 有近实时看板 → L3+

2. **「财务团队有没有参与业务决策（如新项目立项、定价、资源投入）？」**
   - 基本不参与 → L1-L2，财务=行政
   - 偶尔参与 → L2-L3
   - 财务BP嵌入各业务 → L3+

3. **「预算执行情况，偏差超过10%的比例是多少？」**
   - 不知道/超过30% → L1-L2
   - 10-30% → L2-L3
   - 10%以内且有分析 → L3+

4. **「上市前，财务信息披露体系准备好了吗？」**
   - 还没启动 → 紧迫
   - 有初步准备 → 需要专项辅导

## 服务包

**[[华为财务管理]]体系建立（2-3个月，15-25人天）**：
- 管理报告体系设计（月度/季度）
- 财务BP岗位设计与培训
- [[价值为纲]]流程优化
- 经营分析看板设计

**上市财务规范化（3-4个月，20-30人天）**：
- 会计政策梳理与规范
- 内控体系建设
- 信息披露制度
- 关联交易规范

**报价**：25-60万（视范围）
