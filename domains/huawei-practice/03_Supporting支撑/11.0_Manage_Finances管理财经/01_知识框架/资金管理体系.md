# 资金管理体系

<!-- chunk: 资金管理体系.一句话定义 -->
## 一句话定义
集中管理全球资金，匹配公司战略，通过合理规划资产结构和安全高效运作，保障资金供给，管控流动性、汇率、利率、信用四大风险。

## 一、华为资金管理总体架构

华为资金管理部的主要职责：
- 管理集团资产负债表及现金流量表
- 匹配公司战略，保障资金供给
- 支撑业务发展，提升资产运营效率
- 管理财务风险（流动性、汇率、利率、信用）

## 二、现金管理

### 2.1 资金集中管理（Cash Pooling）

华为通过资金池实现全球资金集中管理：

| 机制 | 说明 |
|---|---|
| 资金池（Cash Pooling） | 全球子公司账户归集到总部主账户 |
| 零余额管理 | 子公司日终余额自动归集 |
| 集中收付 | 收支两条线，统一收付平台 |
| 内部结算 | 集团内部往来通过内部账户轧差 |

### 2.2 现金流规划与预测

| 维度 | 频率 | 内容 |
|---|---|---|
| 日度现金流 | 每日 | 银行账户余额、收支明细 |
| 周度滚动预测 | 每周 | 未来4周收支预测 |
| 月度资金计划 | 每月 | 资金配置、大额收付安排 |
| 年度资金规划 | 每年 | 与战略规划联动，长期资金需求 |

### 2.3 现金监控与风险管理

- 严格执行现金管控制度，及时掌握现金流数据
- 设置最低现金余额警戒线
- 备用信贷额度管理
- 在利润保证基础上进行规模扩张

### 2.4 运营资产管理

华为高度重视运营资产管理效率：
- **应收账款**：全球统一信用政策，端到端销售流程关键节点设管控点
- **存货**：目标牵引各业务单元提升存货管理效率
- **预付/其他**：定期清理，减少资金占用

## 三、融资管理

### 3.1 融资策略

华为融资遵循"多元、安全、低成本"原则：

| 融资渠道 | 适用场景 | 特点 |
|---|---|---|
| 银行授信 | 日常经营周转 | 灵活、可随时提款 |
| 债券发行 | 中长期资金需求 | 利率低、期限长 |
| 供应链融资 | 与供应商协同 | 降低供应链整体成本 |
| 客户融资 | 支撑销售 | 帮助客户获得融资购买华为产品 |
| 内部融资 | 集团内部资金调配 | 成本最低，受监管约束 |

### 3.2 资本结构优化

- 保持合理的资产负债率（华为常年维持在60-68%区间）
- 长短期债务搭配，保障资金稳定
- 拓宽融资渠道，增加资金渠道多样性
- 利用全球布局，在低成本市场融资

### 3.3 客户融资管理

华为深度理解客户需求，为客户量身打造多样化融资解决方案：
- 建立完善的风险管理体系
- 融资项目评审制度
- 客户信用等级评定
- 风险处理机制（对已出险客户启动风险处理）

## 四、外汇管理

华为作为全球化企业，外汇管理至关重要。

### 4.1 外汇风险管理策略

| 策略 | 方法 | 优先级 |
|---|---|---|
| 自然对冲 | 匹配销售与采购的币种，平衡外币流入流出 | 首选 |
| 财务对冲 | 远期合约、期权等金融工具对冲 | 辅助 |
| 币种选择 | 融资、采购中优先选择收入币种 | 日常 |
| 现汇买卖 | 即期外汇交易平衡敞口 | 操作层面 |

### 4.2 外汇敞口管理

```
外汇风险敞口 = 外币资产 - 外币负债
                    │
                    ▼
            敞口超限 → 触发对冲
            敞口可控 → 自然对冲
```

华为建立了外汇风险敞口的管理机制，定期审视各币种敞口，匹配风险策略进行对冲。

### 4.3 全球主要风险币种

对于全球化企业，需重点关注：美元、欧元、日元、英镑、新兴市场货币（如巴西雷亚尔、印度卢比、俄罗斯卢布）等波动性较大的币种。

## 五、财务风险管理

华为资金管理部门经董事会财经委员会批准，制定了一系列财务风险管理政策：

### 5.1 流动性风险

- 建立成熟的现金流规划和预测体系
- 评估整体长短期现金盈缺
- 保持适当的现金余额
- 获取充分的备用信贷额度
- 在全球建立广泛的融资资源

### 5.2 利率风险

- 组合运用浮动利率与固定利率的银行借款
- 浮动利率借款带来利息现金流风险
- 固定利率借款带来借款公允价值变动的风险
- 通过利率互换等工具管理

### 5.3 信用风险

- 制定和实施全球统一的信用管理政策、流程、IT系统和风险量化评估工具
- 在各区域和业务单元建立专门的信用管理组织
- 利用风险量化模型评定客户信用等级，确定授信额度
- 在端到端销售流程关键环节设置风险管控点
- 定期审视全球信用风险敞口，预测可能损失，计提坏账准备
