# 华为财务管理

<!-- chunk: 华为财务管理.一-华为财务管理的哲学基础 -->
## 一、华为财务管理的哲学基础

任正非对华为会计核算、财务管理与审计监控三者关系的核心论断：**"全球统一的会计核算和审计监控是长江的两道堤坝，只有这两道堤坝足够坚固，财经管理职能才能从容有效地开展。"**

华为财务管理的六条哲学：
1. **现金为王**：现金是企业的血液
2. **收入是利润的发动机**：可持续的收入增长是关键
3. **称职的CFO随时可接替CEO**：CFO站在业务最后端，俯瞰公司管理全貌
4. **不深入业务，财务只能提供浅层次服务**
5. **预测是管理之魂**：预测准确性是检验财务工作的标尺
6. **企业管理以财务管理为核心，财务管理以资金管理为核心**

<!-- chunk: 华为财务管理.二-ifs集成财经服务变革 -->
## 二、IFS集成财经服务变革

### 2.1 变革背景

2007年华为营业利润率从2003年的19%下降到7%，净利润率从14%下降到5%。随着业务突飞猛进，利润率逐年下滑，任正非意识到没有全球化财务管理，财务风险难以控制。

### 2.2 变革历程

- **2007年**：任正非亲自给IBM CEO彭明盛写信，启动[[IFS]]（集成财务转型）项目
- **2005年起**：账务陆续规划新四统一项目（科目统一、编码统一、制度统一、流程统一）、海外ERP实施、七大共享中心建设
- **2007年**：正式启动[[IFS]]变革，IBM派各CFO级别人员参与

### 2.3 变革目标与成效

变革蓝图：**"加速现金流入，准确确认收入，项目损益可见，经营风险可控"**

成效：
- 实现收入与利润的平衡发展
- 为华为培养数千名合格CFO，深入各业务部门
- 实现了"计划-预算-核算"的全流程管理
- 从没有财务报告，到5天发布报告

### 2.4 变革的关键认知

任正非强调："财经变革是公司层级的变革，不是财经体系的变革，华为要有削足适履的决心接受变革带来的不适。"

<!-- chunk: 华为财务管理.三-财务三支柱架构 -->
## 三、财务三支柱架构

华为财经组织发展形成三支柱架构：

| 支柱 | 定位 | 功能 |
|------|------|------|
| **BP（业务伙伴）** | 离客户最近，和业务一起服务客户 | 在一起、懂业务、提建议 |
| **COE（能力中心）** | 支撑能力专业化 | 提供税务、资金等专业解决方案 |
| **SSC（共享中心）** | 标准服务共享化 | 统一处理标准化财务作业 |

华为的说法："脑袋对着客户、屁股对着领导"——BP承接外部客户需求，通过COE和SSC支撑，实现经营目标。

<!-- chunk: 华为财务管理.四-财务bp的核心要求 -->
## 四、财务BP的核心要求

### 4.1 九字真经

**"在一起、懂业务、提建议"**

- **在一起**：把财务BP配置到相应组织；和业务人员坐在一起；心要和业务在一起
- **懂业务**：学习业务知识，成为"业务里最懂财务的，财务里最懂业务的"
- **提建议**：控风险 + 促增长

### 4.2 项目财务管理"5懂"

1. **懂项目管理**：了解项目不同阶段的人力需求变化
2. **懂合同**：理解付款条款等约定
3. **懂产品**：了解产品特性、施工顺序
4. **懂会计**：收入确认时点、业财结合
5. **懂组织绩效设计**：临时项目团队的绩效设计

### 4.3 项目"概预核决"四算管理

| 环节 | 功能 | 核心目的 |
|------|------|----------|
| **概算** | 估算合同利润，充分评估风险 | 决策是否签约 |
| **预算** | 承接概算利润目标 | 落实经营责任人并充分授权 |
| **核算** | 对照预算目标 | 过程监控 |
| **决算** | 项目执行完成后 | 经验总结，提升组织能力 |

项目"概预核决"支撑[[LTC]]主流程落地。项目财务是预算、预测、核算准确性的第一责任人。

<!-- chunk: 华为财务管理.五-责任中心体系 -->
## 五、责任中心体系

### 5.1 五大责任中心

| 责任中心 | 核心责任 | 考核重点 |
|----------|----------|----------|
| **利润中心** | 对利润负责 | 利润、人均获利能力 |
| **收入中心** | 对收入负责 | 销售收入、回款、坏账率 |
| **费用中心** | 对期间费用负责 | 管理费用、财务费用 |
| **成本中心** | 对可控成本负责 | 成本变动额/率 |
| **投资中心** | 对投资效果负责 | 投资利润率、剩余收益 |

### 5.2 华为利润中心运作机制

核心特点：
- **区域为利润中心**，划小利润核算单元、考核单元
- **BG为收入中心**（以销售额为中心），区域为利润中心（以利润为中心）
- 战争决策权交给区域，BG关注资源建设、战略建设
- 实行"要货成本"定价方案：统一海关到岸价 + 不确定性成本授权代表处决策
- 把指挥所建到"听得见炮声的地方"

### 5.3 为什么华为用责任中心而非阿米巴

利润中心管理的核心目的：**明确责任、简化管理、激活组织**。华为通过分权制衡达到明确责任——将审核责任从四层变为两层，第一层为最终审核责任，第二层为整改责任。

<!-- chunk: 华为财务管理.六-计划-预算-核算体系 -->
## 六、计划-预算-核算体系

### 6.1 五大循环

| 循环 | 内容 |
|------|------|
| 第一循环：规划预算环 | SP → BP → 预算签署PBC → 季度/月度管理 |
| 第二循环：预测运营环 | 综合运营会议、销售运营会议(OTO)、交付运营会议(OTC) |
| 第三循环：计划集成环 | 订货→要货→生产采购→运输发货→到货→工程安装 |
| 第四循环：项目管理环 | 项目机会→投标→交付→回款→决算 |
| 第五循环：绩效基线环 | PBC指标分解落地 + 基线建设与维护 |

### 6.2 预算编制十步法

1. 审视机会，确定产出目标（主要是订货）
2. 细分目标
3. 分析全流程成本问题/建立基线编制原型预算
4. 分清成本费用控制目标和责任
5. 按人均效益提升水平确定人员规划
6. 确定回款及运营资产管理目标
7. 综合平衡（如现金流目标平衡）
8. 汇报评审确定规划和预算
9. 确定预算
10. 制定行动计划

### 6.3 全面预算管理的"8个大坑"

（核心教训总结）
- 预算与战略脱节
- 预算编制缺乏业务假设支撑
- 费用与收入不挂钩
- 缺乏弹性预算机制
- 预算执行无监控
- 考核与预算脱节
- 管理核算方案不合理
- 战略投入与当期经营混为一谈

<!-- chunk: 华为财务管理.七-拧毛巾-三招四式 -->
## 七、"拧毛巾"三招四式

2012年华为连续5个月亏损，集团财经费用管理部通过"拧毛巾"组合拳拧出十亿美元费用：

### 三招

1. **制定平台组织费用预算"高压线"**：参照业界制定费用预算基线，费用额零增长
2. **弹性预算扭转亏损**：上半年费用只授予全年40%，根据收入/销毛完成率弹性授予
3. **管住权签人手中"一支笔"**：刷新费用制度，虚假报销纳入BCG管控

### 四式

4. **战略投入预算单列**："再穷不能穷战略"，专款专用，独立核算
5. **客户界面与内部分开**："再省不能省客户"，客户界面"节约不归己"
6. **人员费用与业务费用分类管理**：雇员性费用匹配中长期规划，业务性费用与产出挂钩
7. **责任中心与资源部门互相PK**：签订内部结算协议，建立"价格听证制度"

<!-- chunk: 华为财务管理.八-七大共享中心 -->
## 八、七大共享中心

华为在全球建立七大账务共享中心：
- 利用时差优势，7x24小时协同作业
- "日不落"循环结账，以最快速度支撑一线
- 单张发票处理成本下降75%
- 审计调整率0.01%
- 5天发布财务报告

共享中心的价值：**"像一道坚固的大坝，把相似的业务放到一个地方高效处理，既起监督控制作用，又节约成本、提供更好服务。"**

<!-- chunk: 华为财务管理.九-内部审计体系 -->
## 九、内部审计体系

### 9.1 三级审计监控体系

华为建立了分层级的三级审计监控体系，注重向增值方向转变。

### 9.2 宠物审计的关键原则

- **超然的独立性**：审计组织直接受机关管理
- **核威慑而非事无巨细**：建立恐吓和威慑系统，帮助建立责任体系
- **审计人员轮换**：30%-40%来自业务部门专家
- **IT审计**：大量招收IT审计经理，评估ERP系统、网络安全等

### 9.3 计划-统计-审计三角形管理

任正非比喻：**"碳元素平行排列，形成石墨；正三角形排列，形成金刚石。三角形循环管理的组织、流程体系是大发展的基础。"**

<!-- chunk: 华为财务管理.十-任正非的财务管理思想 -->
## 十、任正非的财务管理思想

| 思想 | 核心观点 |
|------|----------|
| 堤坝论 | 会计核算和审计监控是两道堤坝 |
| 财务金三角 | 核算、监控、服务三位一体 |
| 计划预算核算 | 计划是龙头，预算是量化，核算是校验 |
| 预测 | 预测是管理之魂 |
| 成本控制 | 不提倡所有成本降低，提倡可有可无成本的降低 |
| 利出一孔 | 资源聚焦，避免分散 |
| 深淘滩低作堰 | 不追求利润最大化，提升行业门槛 |

<!-- chunk: 华为财务管理.十一-财务管理的四个辩证统一 -->
## 十一、财务管理的四个辩证统一

1. **利润最大化与可持续增长**：不单纯追求利润最大化
2. **人工成本与企业利润**：分好蛋糕才能做大蛋糕
3. **会计核算与审计监控**：审计监控是为提高核算质量
4. **财务与业务**：财务必须融入业务——"业务为主导、财务为监督"

<!-- chunk: 华为财务管理.十二-风险管理的核心机制 -->
## 十二、风险管理的核心机制

### 12.1 预算白皮书

启动预算时输出白皮书，从**机会、增长、投入、回报、效率、风险**6个方面提供方向性指导。

### 12.2 BCM业务连续性管理

华为长期建立BCM流程，涵盖销售、研发、采购、财经等各环节，确保恶劣条件下项目交付和业务稳定。

### 12.3 动态预算执行

- **目标是刚性的**（不可以PK），预算是弹性的（基于现实主义）
- 滚动预测：业务计划→预算→预测，计划在前，预算在中间，预测在最后
- 资源投入基于滚动预测的季度业务计划
- **"节约不归己"**原则：研发费用等战略投入，节约部分仍按预算计入考核

### 12.4 利润中心考核三类指标

- **多产粮食**：当期经营成果
- **增加土壤肥力**：长期竞争力建设
- **内外合规**：风险管理

<!-- chunk: 华为财务管理.十三-资本运营 -->
## 十三、资本运营

华为资本运营的三大路径：
1. **政府资源**：通过合资公司吸纳政府背景资本，获得融资便利
2. **员工持股**：1990年起实行，从内部集资演变为奖励分配制度，2001年转为虚拟受限股
3. **迂回上市**：保持非上市，内部股成为"金手铐"

员工持股的核心价值：无息贷款效应、资金稳定长期、降低运营成本。
