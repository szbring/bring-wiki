# 成本管理体系

<!-- chunk: 成本管理体系.一句话定义 -->
## 一句话定义
从源头（设计端）构建成本优势基因，端到端管理产品全生命周期成本，让成本优势看得见、管得住。

---

<!-- chunk: 成本管理体系.一-华为成本管理演进 -->
## 一、华为成本管理演进

华为成本管理经历了三个阶段演进：

| 阶段 | 特征 | 典型做法 |
|---|---|---|
| 1.0 财务核算阶段 | 成本=事后核算，"算清楚" | 标准成本法，月底出成本报表 |
| 2.0 过程管控阶段 | 成本=过程控制，"管得住" | 目标成本法，预算+基线管控 |
| 3.0 竞争优势阶段 | 成本=核心竞争力，"赢在前端" | 全流程端到端，设计端构建成本基因 |

任正非指出：**"成本不是省出来的，是设计出来的。"** 华为成本管理的核心是从产品开发源头开始，端到端管理全生命周期成本。

## 三、标准成本法

标准成本法是华为成本核算的基础方法：

| 要素 | 说明 |
|---|---|
| 标准成本定义 | 在正常经营条件下，生产单位产品应有的成本 |
| 差异分析 | 实际成本 vs 标准成本 = 成本差异 |
| 差异分类 | 材料价格差异、材料用量差异、人工效率差异、制造费用差异 |
| 管理用途 | 差异分析驱动持续改进 |

### 标准成本的设定

```
标准成本 = 标准用量 x 标准价格
              │           │
              ▼           ▼
         工艺定额      采购基线
         (BOM)        (历史/标杆)
```

<!-- chunk: 成本管理体系.四-作业成本法-abc -->
## 四、作业成本法（ABC）

华为在间接费用分摊、产品盈利分析中应用作业成本法：

### ABC核心理念

**传统成本法**：间接费用按产量或工时分摊 → 扭曲产品成本

**作业成本法**：产品消耗作业，作业消耗资源 → 准确反映成本

```
资源 ──→ 作业（Activity）──→ 成本标的（产品/项目/客户）
          ↑
    成本动因（Cost Driver）
```

### ABC实施步骤

| 步骤 | 内容 | 关键产出 |
|---|---|---|
| 1. 识别作业 | 梳理业务流程，分解为具体作业 | 作业清单 |
| 2. 归集资源 | 将各项费用归集到作业成本库 | 作业成本库 |
| 3. 确定成本动因 | 识别驱动作业消耗的因素 | 成本动因表 |
| 4. 计算分配率 | 作业成本/成本动因总量 | 分配率 |
| 5. 分摊至成本标的 | 动因消耗量 x 分配率 | 产品/客户真实成本 |

### ABC适用场景

- 产品线多、间接费用占比高的制造企业
- 需要精准分析产品/客户盈利能力
- 共享服务费用分摊（如IT、行政）
- 项目制企业的间接成本核算

## 五、目标成本法

目标成本法是华为在设计端构建成本优势的核心方法：

```
目标成本 = 目标售价 - 目标利润
                │           │
                ▼           ▼
           市场竞争定价    公司战略要求
```

### 目标成本管理流程

| 阶段 | 活动 | 产出 |
|---|---|---|
| 目标设定 | 基于市场和竞争分析设定目标成本 | 目标成本清单 |
| 成本分解 | 将目标分解到BOM各层级、各功能模块 | 成本分解表 |
| 差距分析 | 当前成本 vs 目标成本，识别差距 | 降本机会清单 |
| 措施落地 | 价值工程、四减（降规格/减配置/减材料/减工艺）、五化（标准化/模块化/兼容化/归一化/自动化） | 降本方案 |
| 持续改进 | 迭代优化，庖丁解牛式深入分析 | 成本基线更新 |

## 六、端到端全流程成本管理

华为成本管理强调跨部门协同，打破"部门墙"：

| 环节 | 成本管理要点 | 核心方法 |
|---|---|---|
| 需求/规划 | SKU管理，减少产品型号冗余 | 需求管理，归一化 |
| 研发设计 | 70%成本在设计阶段决定 | 价值工程，DFX，目标成本法 |
| 采购 | 供应商协同降本，战略采购 | 集中采购，供应商早期介入 |
| 制造 | 精益生产，自动化 | DFM，精益改进 |
| 质量 | 质量成本管理 | 质量检验标准优化，减少质量损失 |

### 研发设计端14种降本方法

**两大分析方法**：
1. 价值工程 — 功能分析 + 性价比分析，找到成本薄弱点
2. 庖丁解牛（长城图）— 产品解剖与竞品对标，找到细分降本机会

**TCO理念**：总拥有成本（Total Cost of Ownership），解决产品成本+使用成本问题

**两个管理机制**：
- 目标成本管理机制 — 解决成本措施落地问题
- DFX管理机制 — 解决研发与采购制造协同问题

**四减**：降规格、减配置、减材料、减工艺

**五化**：标准化、模块化/平台化、兼容化、归一化、自动化

## 七、成本考核与激励

华为让成本工作"充满活力"的核心机制：

| 机制 | 说明 |
|---|---|
| 成本节约分成 | 降本成果按比例奖励团队，"多降多奖" |
| 成本红线 | 关键成本指标纳入组织PBC，不达标问责 |
| 战略投入豁免 | 战略性研发投入不受短期降本考核影响 |
| 标杆对标 | 持续与业界最优对比，设定成本基线 |
