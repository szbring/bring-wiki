# SP / BP 业务计划书模板

> 适用场景：年度/三年战略规划输出，MM Step 5 的核心交付物
> 使用角色：战略部、业务负责人、产品总监
> 标签：#模板 #SP #BP #DSTE #BLM #战略规划

---

<!-- chunk: sp-bp业务计划书模板.一-sp-与-bp-的区别 -->
## 一、SP 与 BP 的区别

| 对比项 | SP（战略规划）| BP（年度经营计划）|
|--------|-------------|----------------|
| 时间跨度 | 3-5年（滚动更新）| 1年（固定）|
| 输出方法论 | **BLM**（业务领先模型）| **BEM**（业务执行力模型）|
| 核心问题 | Where to Play / How to Win | 今年做什么 / 怎么做 |
| 颗粒度 | 方向性、战略性 | 可执行、可考核 |
| 财务内容 | 3年收入/利润大方向预测 | 年度详细收入/成本/利润预算 |
| 输入来源 | BLM分析 + MM市场洞察 | SP + 上年度BP执行结果 |

## 二、SP 战略规划（BLM框架）

### 2.1 BLM 模型结构

```
         ┌─────────────────────────────────┐
         │         战略意图                 │
         │  愿景 · 使命 · 战略目标          │
         └──────────┬──────────────────────┘
                    │
    ┌───────────────┴───────────────┐
    │   市场洞察                     │   战略制定
    │   （外部环境分析）              │   ↓
    │   · 市场趋势                   │   业务设计
    │   · 竞争格局                   │   · 客户选择
    │   · 客户需求变化               │   · 价值主张
    └───────────────────────────────┘   · 盈利模式
                                        · 战略控制点
    ┌───────────────────────────────┐
    │   差距分析                     │   组织能力
    │   现状 vs 目标的差距           │   · 文化与价值观
    │                               │   · 人才与技能
    └───────────────────────────────┘   · 流程与系统
```

### 2.2 SP 文档标准章节

```markdown
# [业务单元名称] 战略规划（20XX-20XX）

<!-- chunk: sp-bp业务计划书模板.执行摘要-1页 -->
## 执行摘要（1页）
- 战略方向（一句话）
- 3年目标（收入/利润/市占率）
- TOP3 战略举措

<!-- chunk: sp-bp业务计划书模板.第一章-战略意图 -->
## 第一章：战略意图
- 1.1 使命与愿景
- 1.2 3-5年战略目标（SMART）
- 1.3 核心价值主张

<!-- chunk: sp-bp业务计划书模板.第二章-市场洞察-mm-step-1-2输出 -->
## 第二章：市场洞察（MM Step 1-2输出）
- 2.1 行业趋势（3-5年预测）
- 2.2 目标细分市场分析（TAM/SAM/SOM）
- 2.3 客户需求演变
- 2.4 竞争格局分析
- 2.5 技术趋势影响
- 2.6 宏观环境（PESTEL）

<!-- chunk: sp-bp业务计划书模板.第三章-业务设计 -->
## 第三章：业务设计
- 3.1 客户选择（目标细分市场 + ICP）
- 3.2 价值主张（我们为谁创造什么价值）
- 3.3 盈利模式（收入来源 + 成本结构）
- 3.4 战略控制点（护城河在哪里）

<!-- chunk: sp-bp业务计划书模板.第四章-差距分析 -->
## 第四章：差距分析
- 4.1 当前能力 vs 战略目标的差距
- 4.2 关键能力短板（TOP3）
- 4.3 竞争对手能力对比

<!-- chunk: sp-bp业务计划书模板.第五章-战略举措-3-5年路标 -->
## 第五章：战略举措（3-5年路标）
- 5.1 举措一：[名称 + 目标 + 里程碑]
- 5.2 举措二：
- 5.3 举措三：

<!-- chunk: sp-bp业务计划书模板.第六章-财务展望 -->
## 第六章：财务展望
- 6.1 收入预测（3年）
- 6.2 成本结构演进
- 6.3 利润目标
- 6.4 主要投资项目

<!-- chunk: sp-bp业务计划书模板.第七章-组织能力建设 -->
## 第七章：组织能力建设
- 7.1 人才策略
- 7.2 关键流程优化
- 7.3 系统/工具支撑

<!-- chunk: sp-bp业务计划书模板.第八章-风险识别 -->
## 第八章：风险识别
- 8.1 主要风险（TOP5）
- 8.2 应对预案
```

## 三、BP 年度经营计划（BEM框架）

### 3.1 BEM 解码逻辑

```
SP 战略举措
    ↓ 解码
年度经营主题（CEO级优先级）
    ↓ 解码（BSC平衡计分卡）
四维目标：财务 / 客户 / 流程 / 学习成长
    ↓ 分解
部门/团队年度KPI
    ↓ 分解
个人PBC（绩效承诺书）
```

### 3.2 BP 文档标准章节

```markdown
# [业务单元名称] 年度经营计划（20XX年）

<!-- chunk: sp-bp业务计划书模板.执行摘要 -->
## 执行摘要
- 年度经营主题（一句话）
- 年度核心目标（收入/利润/市占率/客户满意度）
- 重点工作 TOP5

<!-- chunk: sp-bp业务计划书模板.第一章-上年度回顾 -->
## 第一章：上年度回顾
- 1.1 目标达成情况（红绿灯评分）
- 1.2 主要成果
- 1.3 主要差距与根因分析
- 1.4 经验教训

<!-- chunk: sp-bp业务计划书模板.第二章-市场环境-年度视角 -->
## 第二章：市场环境（年度视角）
- 2.1 市场机会点
- 2.2 主要威胁/风险
- 2.3 竞争动态更新

<!-- chunk: sp-bp业务计划书模板.第三章-年度经营目标-bsc四维 -->
## 第三章：年度经营目标（BSC四维）
- 3.1 财务目标：收入 / 利润率 / 回款率
- 3.2 客户目标：NPS / 市场份额 / 新客户数
- 3.3 内部流程目标：交付周期 / 质量指标 / 研发里程碑
- 3.4 学习成长目标：能力建设 / 人才梯队

<!-- chunk: sp-bp业务计划书模板.第四章-年度重点工作-详细行动计划 -->
## 第四章：年度重点工作（详细行动计划）
| 重点工作 | 责任人 | Q1 | Q2 | Q3 | Q4 | 成功标准 |
|---------|--------|----|----|----|----|---------|
| ...     |        |    |    |    |    |         |

<!-- chunk: sp-bp业务计划书模板.第五章-资源需求与预算 -->
## 第五章：资源需求与预算
- 5.1 人力需求（招聘计划）
- 5.2 年度预算（收入/成本/费用）
- 5.3 关键投资项目

<!-- chunk: sp-bp业务计划书模板.第六章-风险管控 -->
## 第六章：风险管控
- 主要风险清单及应对措施

<!-- chunk: sp-bp业务计划书模板.第七章-考核机制 -->
## 第七章：考核机制
- KPI 分解到部门/个人
- 月度/季度 Review 机制
```

## 四、SP/BP 制定时间节点

| 时间 | 活动 |
|------|------|
| **8-9月** | 市场洞察更新，BLM分析启动 |
| **10月** | SP草稿完成，内部讨论 |
| **11月** | SP定稿，BP制定启动 |
| **12月** | BP草稿完成，预算审批 |
| **1月** | BP正式发布，KPI分解 |
| **每季度** | BP执行回顾（Strategy Review）|
