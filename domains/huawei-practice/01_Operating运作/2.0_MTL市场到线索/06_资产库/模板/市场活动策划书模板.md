## 一、活动目标

### 1.1 业务目标（量化）

| 目标类型 | 目标值 | 衡量方式 |
|---------|--------|---------|
| 线索捕获（MQL）| ≥ ___ 条 | 签到表 + 表单提交 |
| 品牌曝光 | ≥ ___ 人触达 | 到场人数 + 线上传播 |
| 合作商机 | ≥ ___ 个深度沟通 | 销售跟进记录 |
| 内容传播 | 活动总结文章阅读 ≥ ___ | 公众号数据 |

### 1.2 目标受众（对应 ICP）

- **主要受众：** [职位/行业/规模]
- **次要受众：** [合作伙伴/媒体/KOL]
- **预期到场比例：** 目标客户 ___% / 现有客户 ___% / 合作伙伴 ___%

<!-- chunk: 市场活动策划书模板.二-活动内容设计 -->
## 二、活动内容设计

### 2.1 核心议题

| 议题 | 演讲嘉宾 | 时长 | 目的 |
|------|---------|------|------|
| 主题演讲：[题目] | [姓名/职位] | 30min | 建立思想领导力 |
| 案例分享：[客户案例] | 客户 + 薄云 | 20min | 建立信任/口碑 |
| 实战工作坊：[主题] | 薄云顾问 | 60min | 深度互动/线索筛选 |
| 圆桌讨论：[话题] | 多方嘉宾 | 30min | 碰撞洞见 |

### 2.2 演讲内容检查清单

- [ ] 演讲逻辑：问题 → 框架 → 案例 → 结论（不是产品推销）
- [ ] 客户语言：用听众能理解的词汇，不用内部术语
- [ ] 数据支撑：每个观点有数据或案例支撑
- [ ] 行动号召（CTA）：每场结束后有明确引导（下载/咨询/扫码）

## 三、线索捕获计划

| 环节 | 捕获方式 | 责任人 |
|------|---------|--------|
| 活动前报名 | 官网表单（含公司/职位/联系方式）| 市场运营 |
| 现场签到 | 二维码 + 名片 | 前台 |
| 工作坊报名 | 独立报名表（更深信息）| 现场销售 |
| 演讲后互动 | 扫码下载配套资料（需填信息）| 讲师 |
| 会后跟进 | 48小时内邮件 + 销售电话 | 销售团队 |
