# 市场洞察五看模型

> 华为[[BLM]]/DSTE战略洞察的核心方法论——通过"看行业、看市场、看竞争、看自己、看机会"系统识别战略机会点，最终以SPAN矩阵输出市场选择决策。

---

<!-- chunk: 市场洞察五看模型.一-五看模型的定位 -->
## 一、五看模型的定位

五看模型是华为战略洞察的标准方法论，嵌入DSTE（开发战略到执行）和BLM（业务领先模型）的战略分析环节，也是MTL流程中MI（市场洞察）模块的核心工具。

```
DSTE战略管理循环
┌──────────────────────────────────────────────────┐
│  战略洞察（五看）→ 战略制定（三定）→ 战略解码 → 执行评估  │
│  ← 本文重点 →                                    │
└──────────────────────────────────────────────────┘
```

**核心逻辑**：先有"五看"识别机会，再有"三定"（定战略控制点、定目标、定策略）制定战略。五看是输入，三定是输出。

<!-- chunk: 市场洞察五看模型.二-看行业-趋势 -->
## 二、看行业/趋势

### 2.1 核心目标

分析行业变化和趋势，回答：**行业价值向哪里转移？哪些领域"有钱味"？**

### 2.2 四个分析维度

#### 维度一：宏观环境（PEST分析）

| 因素 | 分析要点 | 对业务的影响 |
|------|---------|------------|
| 政治（P） | 政策方向、监管变化、贸易壁垒 | 市场准入、合规成本 |
| 经济（E） | 增长率、利率、汇率、投资周期 | 客户预算、采购意愿 |
| 社会（S） | 人口结构、消费观念、人才供给 | 需求变化、人力成本 |
| 技术（T） | 技术成熟度、颠覆性技术出现 | 产品迭代、竞争格局 |

#### 维度二：价值转移趋势

> "战略就是要闻到有钱味的地方。"——任正非

价值转移的驱动因素：
- **客户需求变化**：新人群、新场景、新痛点出现
- **技术变革**：新技术使原有价值链重构
- **商业模式创新**：新的利润模型改变行业格局
- **法律法规变化**：政策催生新市场或淘汰旧模式

**案例**：珀莱雅洞察到18-25岁女性对精华的需求，但买不起兰蔻级别的价格，通过"成分党"KOC在小红书等平台传播，实现了平价精华的爆发增长。

#### 维度三：技术趋势（Hype Cycle）

使用Gartner炒作曲线判断技术成熟度：
- 技术萌芽期 → 期望膨胀期 → 泡沫破裂期 → 稳步爬升期 → 生产成熟期
- 关键是判断当前处于哪个阶段，避免过早或过晚投入

#### 维度四：行业竞争态势（五力模型）

分析行业五种竞争力量：
- 供应商议价能力
- 买方议价能力
- 新进入者威胁
- 替代品威胁
- 现有竞争者竞争程度

<!-- chunk: 市场洞察五看模型.三-看市场-客户 -->
## 三、看市场/客户

### 3.1 核心目标

深入理解客户需求、行为和痛点，回答：**客户是谁？买什么？为什么买？**

### 3.2 市场地图绘制

基于行业定位，绘制公司市场地图：
- 客户是谁？（行业、规模、区域）
- 客户买什么？（产品、服务、解决方案）
- 谁买？谁决定买？（决策链分析）
- 什么渠道买？（直销、渠道、平台）

### 3.3 客户需求分析框架

**对标客户的SP（战略规划）**：
- 客户未来5年的战略方向是什么？
- 在客户发展战略中存在哪些痛点？
- 客户的用户（客户的客户）有什么未被满足的需求？

**客户系统经济学**：
- 需求背后的诉求是什么？
- 谁是需求的Owner？谁为需求买单？
- 采购决策的影响因素排序（性能/价格/服务/品牌/关系）

**客户购买行为分解**：
- 完整的购买旅程：感知需求 → 信息搜索 → 方案评估 → 采购决策 → 交付使用
- 在每个环节识别影响决策的关键因素和可能的营销介入点

### 3.4 市场细分

将市场按照有意义的维度进行切分：

| 细分维度 | 说明 | 示例 |
|---------|------|------|
| 行业 | 不同行业的差异化需求 | 金融、制造、教育 |
| 规模 | 大/中/小企业的采购模式差异 | 央企、独角兽、中小企业 |
| 区域 | 不同地区的市场特征 | 一线城市、下沉市场、海外 |
| 技术成熟度 | 客户技术接受能力 | 早期采用者、主流市场、落后者 |

<!-- chunk: 市场洞察五看模型.四-看竞争 -->
## 四、看竞争

### 4.1 核心目标

全面了解竞争对手的战略、优势和弱点，回答：**对手在做什么？我们与对手的差距在哪里？差异化竞争点是什么？**

### 4.2 竞争对手18要素分析

华为梳理了18个分析竞争对手的要素，从中选择5-6个重点进行深入分析：

| 类别 | 要素 |
|------|------|
| 财务表现 | 利润率、营收增长、研发投入占比 |
| 市场地位 | 市场份额、客户覆盖率、品牌认知度 |
| 产品能力 | 产品线完整性、技术领先度、质量水平 |
| 客户关系 | 关键客户、客户满意度、客户粘性 |
| 战略方向 | 未来3年产品规划、市场拓展方向、并购策略 |
| 组织能力 | 组织结构、人才密度、激励机制 |
| 供应链 | 供应商关系、交付能力、成本优势 |

### 4.3 竞争分析工具

**雷达图**：选择5-6个关键竞争因素，直观呈现我方与竞争对手的优劣势对比。

```
        技术
       /    \
  品牌        产品
   |    我方   |
  服务        价格
       \    /
        渠道
```

**竞争定位图**：按关键维度（如价格vs技术先进性）进行二维定位。

### 4.4 竞争情报体系

| 情报来源 | 获取方式 | 更新频率 |
|---------|---------|---------|
| 公开资料 | 年报、官网、发布会 | 季度 |
| 行业展会 | 现场观察、交流 | 按展会排期 |
| 客户反馈 | 客户访谈、招标信息 | 月度 |
| 合作伙伴 | 渠道伙伴反馈 | 月度 |
| 专业机构 | 行业研究报告 | 年度 |

## 五、看自己

### 5.1 核心目标

客观评估自身优势和劣势，回答：**我们有什么核心能力？哪些业务不是我们的菜？**

### 5.2 商业模型画布

通过商业模型画布的10个模块，系统性分析自身：

| 模块 | 审视要点 |
|------|---------|
| 客户细分 | 我们服务哪些客户？ |
| 价值主张 | 我们为客户解决什么问题？ |
| 渠道通路 | 如何触达客户？ |
| 客户关系 | 与客户建立什么类型的关系？ |
| 收入来源 | 客户为什么付费？ |
| 核心资源 | 我们拥有什么关键资源？ |
| 关键活动 | 我们必须做好哪些事？ |
| 关键伙伴 | 谁是我们的重要伙伴？ |
| 成本结构 | 主要成本来自哪里？ |
| 利润模型 | 如何赚钱？利润率如何？ |

> 如果10个框都非常好，业绩自然会好；如果大部分都比较差，需要掂量自己有没有能力做。

### 5.3 关键自省问题

1. 我们的核心竞争力是什么？能否持续？
2. 哪些机会"看起来很美"但不是我们的菜？
3. 我们的短板是否构成"致命伤"？
4. 如果明天竞争对手降价30%，我们能否扛住？

## 六、看机会 — SPAN矩阵

### 6.1 核心目标

在前四看的基础上，识别和评估战略机会点，回答：**我们应该在哪里投入资源？**

### 6.2 SPAN矩阵

SPAN（Strategy Positioning Analysis，战略定位分析）是五看的核心输出工具，从两个维度评估每个细分市场：

```
                 公司竞争地位
                  强        中        弱
         ┌──────────┬──────────┬──────────┐
    高   │  ★ 明星   │  ? 问题   │  ? 问题   │
市场     ├──────────┼──────────┼──────────┤
吸引力   中   │  ■ 选择投入│  □ 维持   │  □ 考虑  │
         ├──────────┼──────────┼──────────┤
    低   │  $ 金牛   │  $ 收割   │  ✕ 退出   │
         └──────────┴──────────┴──────────┘
```

**市场吸引力**评估因素：
- 市场空间大小
- 增长速度
- 利润率/毛利率
- 竞争激烈程度

**公司竞争地位**评估因素：
- 市场份额
- 控制点（技术、品牌、渠道等）
- 竞争优势的可持续性

### 6.3 四个象限的策略

| 象限 | 特征 | 策略 | 考核重点 |
|------|------|------|---------|
| **明星** | 高吸引力+强竞争地位 | 优先投入，扩大优势 | 市场份额增长 |
| **问题** | 高吸引力+弱竞争地位 | 选择性投入，构建竞争要素 | 关键能力建设 |
| **金牛** | 低吸引力+强竞争地位 | 收割利润，控制投入 | 净利润率 |
| **退出** | 低吸引力+弱竞争地位 | 逐步退出，释放资源 | 资源回收率 |

### 6.4 华为的实践要求

- **70%的业务收入应来自明星区域**——确保公司有未来
- 如果所有业务都在金牛区域，虽然公司很大但不强，缺乏未来增长点
- 市场空间大但无增长速度，不是最好的选择
- 利润率低的业务（如互联网低毛利模式）不符合华为的选择标准
- 增长速度越快的市场越有可能有变化，华为偏好这种市场

### 6.5 三层面业务组合

结合SPAN矩阵，华为采用三层面业务组合策略：

| 层面 | 对应SPAN位置 | 角色 | 资源配置 |
|------|------------|------|---------|
| 第一层面：核心业务 | 金牛/明星 | 提供现金流和利润 | 70%资源 |
| 第二层面：成长业务 | 问题→明星 | 未来的增长引擎 | 20%资源 |
| 第三层面：新兴机会 | 问题（早期） | 种子业务，探索未来 | 10%资源 |

## 七、五看的输出与衔接

### 7.1 五看→三定的衔接

```
五看（战略洞察）                    三定（战略制定）
┌─────────────┐                 ┌─────────────────┐
│ 看行业 → 行业趋势和机会  │                 │                  │
│ 看市场 → 客户需求和痛点  │  ────────→  │ 定战略控制点     │
│ 看竞争 → 竞争差距和优势  │  输入          │ 定目标           │
│ 看自己 → 核心能力和短板  │  机会点        │ 定策略           │
│ 看机会 → SPAN矩阵决策   │                 │                  │
└─────────────┘                 └─────────────────┘
```

### 7.2 机会点的描述规范

一个好的机会点描述应该是**一个目标细分市场**，且需要量化和具体化：

| 维度 | 要求 |
|------|------|
| 市场空间 | 具体数字（如：200亿TAM） |
| 增长率 | CAGR（如：30% CAGR） |
| 竞争格局 | 前3名市场份额 |
| 我方现状 | 当前份额和能力 |
| 投入需求 | 所需资源和时间 |

## 八、薄云咨询的应用建议

### 诊断应用
- 评估客户是否有系统性的市场洞察机制（而非靠老板"嗅觉"或销售个人经验）
- 检查五看的覆盖面——多数企业只"看竞争"，忽视了"看行业"和"看自己"
- 诊断SPAN矩阵的产出——是否有明确的H1/H2/H3业务组合

### 辅导应用
- 带领客户团队完成一次完整的五看研讨（通常2-3天）
- 培训使用PEST、五力模型、商业画布、SPAN矩阵等工具
- 帮助客户建立季度洞察更新的节奏机制

### 常见误区
- **五看变成"一看了之"**：只做了分析报告，没有转化为决策和行动
- **看竞争只看产品**：忽视对手的组织能力、战略方向和客户关系
- **SPAN矩阵画完就束之高阁**：没有将矩阵结论转化为资源配置决策
- **只看外部不看自己**：缺乏对自身核心能力的客观评估
