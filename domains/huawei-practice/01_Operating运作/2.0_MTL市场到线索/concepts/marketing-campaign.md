# 营销活动管理

## 定义
为达成特定营销目标（产品上市 / 客户拓新 / 心智占位）而设计的有组织、有预算、有评估闭环的集成传播战役，通常以"主题 → 内容 → 渠道 → 触达 → 转化 → 复盘"全链路运作。

## 输入 / 输出
- **输入**：营销目标（KPI：曝光 / 线索 / 商机）+ 预算 + 目标客户画像 + [[id:huawei.mtl.concept.brand-management]] 品牌定位
- **输出**：合格销售线索（数 + 质量）+ 品牌曝光 + 客户教育 + 媒体资产
- **关键阶段**：策划 → 内容生产 → 渠道分发 → 现场执行 → 线索追踪 → ROI 复盘

## 角色
- **指挥方**：营销活动经理（campaign manager）
- **执行方**：内容、设计、媒介、活动运营、数字营销
- **协同方**：销售（识别目标客户、活动后跟进）、产品（提供方案展示）、PR
- **客户方**：被定向触达的目标决策链角色

## 上下游
- **上游**：营销年度计划（来自 [[id:huawei.mtl.concept.mtl-framework]] BP 营销目标）
- **下游**：合格线索 → [[id:huawei.mtl.concept.lead-management]] 培育 → LTC 销售
- **配套**：CRM 系统（线索归属与追踪）+ 数字营销平台

## 典型误区
- **以活动数衡量营销**：办了多少场——但场次跟商业结果脱钩
- **目标模糊**：一场活动既要曝光又要线索又要交易——目标越多越难做透
- **不做客户画像就广撒网**：泛泛邀请，到场率高但目标客户少
- **不复盘 ROI**：活动结束就结束——下次同样的钱怎么花没数据支撑
