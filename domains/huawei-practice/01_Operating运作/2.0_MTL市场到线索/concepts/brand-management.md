# 品牌管理体系

## 定义
通过定位、表达、触达、维护四个环节系统化管理品牌资产的体系，确保客户对企业及其产品形成稳定、积极、差异化的认知，最终降低销售难度与提升溢价能力。

## 输入 / 输出
- **输入**：[[id:huawei.mtl.concept.market-insight]] 市场洞察输出 + 客户认知调研 + 竞争对手品牌定位 + 公司战略
- **输出**：品牌定位（差异化 + 价值主张）+ 品牌识别系统（视觉 + 话术）+ 品牌资产追踪指标
- **关键资产**：品牌知名度 / 美誉度 / 偏好度 / 忠诚度

## 角色
- **决策方**：CMO + CEO（重大品牌决策）
- **执行方**：品牌部门 + 创意 / 设计 / 公关
- **协同方**：产品部门（产品品牌）、销售（一线品牌触点）、HR（雇主品牌）

## 上下游
- **上游**：[[id:huawei.mtl.concept.market-insight]] 市场洞察 → 品牌定位
- **下游**：品牌定位 → [[id:huawei.mtl.concept.marketing-campaign]] 营销活动 → 客户触达
- **平行**：与 LTC 销售协作（销售触点 = 品牌触点）

## 典型误区
- **品牌 = logo + 口号**：把品牌窄化为视觉资产——品牌是客户心智占位的总和
- **品牌投资短期化**：CFO 看品牌支出 ROI 当季——品牌是 3-5 年才显效的累积资产
- **品牌与产品脱节**：品牌部门吹得很高大上，产品体验不达标——客户失望，品牌反伤
- **照抄 toC 品牌玩法**：B2B 品牌强调专业 + 信任 + 长期关系，与 toC 短情感冲击不同
