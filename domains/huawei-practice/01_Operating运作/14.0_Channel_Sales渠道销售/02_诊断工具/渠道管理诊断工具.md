# 渠道管理诊断工具

<!-- chunk: 渠道管理诊断工具.成熟度评估 -->
## 成熟度评估

| 级别 | 特征 |
|---|---|
| L1 | 无正式渠道，全靠直销，渠道偶发且无管理 |
| L2 | 有代理商，但管理粗放，价格混乱，冲突频繁 |
| L3 | 渠道分级，有政策体系，冲突有解决机制 |
| L4 | 渠道数字化，伙伴赋能体系完善，协同高效 |
| L5 | 生态化运营，战略伙伴共同研发/营销 |

---

<!-- chunk: 渠道管理诊断工具.快速诊断问题 -->
## 快速诊断问题

1. **「渠道和直销的冲突，有没有明确的规则来解决？」**
   - 靠协调，没有规则 → L1-L2，价格/客户冲突频繁
   - 有规则但执行不一 → L2-L3
   - 有清晰规则且执行严格 → L3+

2. **「渠道伙伴有分级吗？不同级别的待遇有差异化吗？」**
   - 没有分级 → L1-L2
   - 有分级但差异化不明显 → L2
   - 分级清晰，资源差异化投入 → L3+

3. **「你们如何帮助渠道提升销售能力？」**
   - 基本不管 → L1，伙伴靠自己
   - 有培训但不系统 → L2
   - 有完整赋能体系（培训/工具/支持）→ L3+

4. **「渠道伙伴的销售数据，你们能实时掌握吗？」**
   - 靠伙伴月报，滞后且不准 → L1-L2
   - 有系统但录入质量差 → L2
   - 有伙伴门户，数据实时 → L3+

## 访谈问题库

### 销售/渠道负责人
- 渠道伙伴总数多少？分布在哪些区域？
- 渠道和直销团队的客户边界是怎么划定的？
- 渠道伙伴最常见的投诉/不满是什么？
- 伙伴返点政策是怎样的？市场支持有哪些？
- 渠道招募、培训、认证的流程是什么？

### CEO
- [[华为的市场营销体系]]占总营收的比例是多少？目标是多少？
- 渠道管理目前最大的挑战是什么？
- 有没有因为渠道管理不善损失过重要市场？
