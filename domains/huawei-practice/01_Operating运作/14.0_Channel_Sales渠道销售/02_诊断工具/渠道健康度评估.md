# 渠道健康度评估

> 渠道体系健康度诊断工具——从覆盖率、活跃度、贡献度、满意度、利润度五个维度评估渠道体系健康状态。

---

<!-- chunk: 渠道健康度评估.一-五维健康度模型 -->
## 一、五维健康度模型

| 维度 | 核心问题 | 健康标准 |
|------|---------|---------|
| **覆盖率** | 渠道能否覆盖目标市场？ | 目标市场覆盖率≥80% |
| **活跃度** | 伙伴是否活跃在卖？ | 活跃伙伴占比≥60% |
| **贡献度** | 渠道对营收贡献多大？ | 渠道营收占比≥30% |
| **满意度** | 伙伴是否满意？ | 伙伴NPS≥40 |
| **利润度** | 伙伴能否赚钱？ | 伙伴平均毛利率≥20% |

---

<!-- chunk: 渠道健康度评估.二-覆盖率评估 -->
## 二、覆盖率评估

### 2.1 评估指标

| 指标 | 计算方式 | 目标 |
|------|---------|------|
| 区域覆盖率 | 有伙伴覆盖的区域/目标区域 | ≥80% |
| 行业覆盖率 | 有伙伴覆盖的行业/目标行业 | ≥70% |
| 产品线覆盖率 | 伙伴销售的产品线/总产品线 | ≥60% |
| 客户层级覆盖率 | 伙伴可触达的客户层级/目标层级 | ≥80% |

### 2.2 覆盖缺口分析

| 缺口类型 | 诊断 | 处方 |
|---------|------|------|
| 区域缺口 | 某区域无伙伴 | 定向招募+激励 |
| 行业缺口 | 某行业无伙伴 | 行业伙伴招募 |
| 产品缺口 | 伙伴只卖老产品 | 新产品培训+专项激励 |
| 层级缺口 | 伙伴只能触达中层 | 提升伙伴能力或招募高层伙伴 |

---

<!-- chunk: 渠道健康度评估.三-活跃度评估 -->
## 三、活跃度评估

### 3.1 伙伴活跃度分级

| 状态 | 定义 | 占比参考 |
|------|------|---------|
| **高活跃** | 季度内有采购+有项目+有认证 | ≥30% |
| **中活跃** | 季度内有采购但项目不多 | ≥30% |
| **低活跃** | 半年内有采购但频率低 | ≤30% |
| **休眠** | 半年以上无采购 | ≤10% |

### 3.2 活跃度提升策略

| 状态 | 原因分析 | 提升策略 |
|------|---------|---------|
| 低活跃 | 伙伴不够专注 | 专属跟进+增量激励 |
| 低活跃 | 伙伴能力不足 | 赋能培训+联合打单 |
| 低活跃 | 伙伴利润不够 | 优化返利和折扣 |
| 休眠 | 伙伴已转向其他厂商 | 评估是否值得重新激活 |

<!-- chunk: 渠道健康度评估.四-贡献度评估 -->
## 四、贡献度评估

| 指标 | 当前 | 目标 | 差距 |
|------|------|------|------|
| 渠道营收占总营收比 | | ≥30% | |
| Top10伙伴营收集中度 | | 40-60% | |
| 单伙伴平均营收 | | 持续增长 | |
| 新伙伴营收贡献 | | ≥10% | |

**健康判断**：
- 集中度>80%：过度依赖少数伙伴，风险高
- 集中度<30%：伙伴太分散，管理成本高

<!-- chunk: 渠道健康度评估.五-满意度评估-伙伴nps -->
## 五、满意度评估（伙伴NPS）

### 5.1 伙伴NPS问卷

| 问题 | 评分(0-10) |
|------|-----------|
| 您多大程度愿意向同行推荐与我们的合作？ | |
| 产品竞争力是否满意？ | |
| 利润空间是否满意？ | |
| 厂商支持是否到位？ | |
| 合作流程是否简单高效？ | |

### 5.2 NPS计算

- 推荐者（9-10分）占比 - 贬损者（0-6分）占比 = NPS
- NPS≥50：优秀
- NPS 30-50：良好
- NPS<30：需要改进

<!-- chunk: 渠道健康度评估.六-利润度评估 -->
## 六、利润度评估

| 评估项 | 标准 | 预警 |
|--------|------|------|
| 伙伴平均毛利率 | ≥20% | <15% |
| 伙伴平均净利润率 | ≥8% | <5% |
| 返利占利润比 | 20-40% | >60% |
| MDF使用率 | ≥70% | <30% |

**利润度低的后果**：伙伴不赚钱→不专注→业绩下降→退出
