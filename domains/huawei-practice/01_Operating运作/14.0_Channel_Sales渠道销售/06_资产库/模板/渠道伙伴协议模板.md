## 三、伙伴权利

### 3.1 商务权利

| 权利 | 内容 |
|------|------|
| 产品采购折扣 | 按伙伴级别享受对应折扣 |
| 返利 | 按季度考核结果享受返利 |
| MDF | 按采购额比例获得市场发展基金 |
| Deal Registration | 商机报备保护权 |
| 特价申请 | 战略/核心伙伴可申请特价 |

### 3.2 赋能权利

| 权利 | 内容 |
|------|------|
| 培训 | 按级别参加对应培训课程 |
| 认证补贴 | 按级别享受认证费用补贴 |
| 技术支持 | 按级别享受对应SLA技术支持 |
| 专属客户代表 | 战略伙伴享受专属代表 |

### 3.3 营销权利

| 权利 | 内容 |
|------|------|
| 品牌使用 | 在授权范围内使用甲方品牌标识 |
| 联合营销 | 按级别参与联合营销活动 |
| 市场物料 | 获取甲方提供的市场推广物料 |
| 标杆案例 | 可参与标杆案例共建 |

<!-- chunk: 渠道伙伴协议模板.五-考核与升降级 -->
## 五、考核与升降级

### 5.1 季度考核

| 考核维度 | 权重 | 达标标准 |
|---------|------|---------|
| 业绩 | 50% | 采购额≥季度目标80% |
| 能力 | 25% | 认证人数达标 |
| 满意度 | 25% | 客户NPS≥30 |

### 5.2 升降级规则

| 条件 | 处理 |
|------|------|
| 连续2季度达标且超过120% | 可申请升级 |
| 连续2季度不达标 | 降级处理 |
| 连续4季度零业绩 | 协议终止 |

<!-- chunk: 渠道伙伴协议模板.六-返利条款 -->
## 六、返利条款

| 返利类型 | 条件 | 比例 | 发放方式 |
|---------|------|------|---------|
| 基础返利 | 达到季度目标 | __% | 季度预发，年度清算 |
| 增量返利 | 超过去年同期 | __% | 年度清算 |
| 战略产品返利 | 销售指定产品 | __% | 季度发放 |

**返利扣减**：
- 客户投诉经确认：扣减当季返利10-50%
- 价格违规：扣减当季返利50-100%
- 窜货：扣减当季返利100%

## 七、保密与知识产权

| 条款 | 内容 |
|------|------|
| 保密范围 | 甲方的产品信息、价格体系、客户信息、技术资料 |
| 保密期限 | 协议期内及终止后2年 |
| 知识产权 | 甲方品牌、商标、技术归甲方所有 |
| 限制 | 乙方不得将甲方资料泄露给第三方 |

## 八、协议终止

| 终止条件 | 通知期 |
|---------|--------|
| 协议到期 | 自动终止，提前30天通知是否续签 |
| 任何一方违约 | 30天书面通知 |
| 严重违规（窜货/低价/腐败） | 立即终止 |
| 双方协商一致 | 协商确定 |
