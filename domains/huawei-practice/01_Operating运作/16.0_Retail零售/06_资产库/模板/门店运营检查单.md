# 门店运营检查单

> 零售门店日常运营的标准检查清单，确保门店运营规范统一、客户体验一致。

---

<!-- chunk: 门店运营检查单.一-开店前检查-每日 -->
## 一、开店前检查（每日）

| 序号 | 检查项 | 标准 | OK/NG |
|------|--------|------|-------|
| 1 | 门店外观整洁 | 招牌灯亮、玻璃干净、门口无杂物 | |
| 2 | 营业时间标识 | 营业时间牌正确、清晰 | |
| 3 | 设备开启 | 收银机、电子价签、音响、空调正常 | |
| 4 | 商品陈列 | 按陈列标准摆放、价格标签正确 | |
| 5 | 促销物料 | 促销海报/POP正确摆放 | |
| 6 | 卫生间检查 | 干净、用品充足 | |
| 7 | 员工仪容 | 着装规范、工牌佩戴 | |

---

<!-- chunk: 门店运营检查单.二-营业中检查-每日 -->
## 二、营业中检查（每日）

### 2.1 商品管理

| 序号 | 检查项 | 标准 | OK/NG |
|------|--------|------|-------|
| 1 | 缺货检查 | 促销品/畅销品无缺货 | |
| 2 | 商品陈列维护 | 客户翻动后及时整理 | |
| 3 | 价格标签 | 电子价签/纸质标签正确 | |
| 4 | 商品保质期 | 临期品已标记和处理 | |
| 5 | 试用品/样品 | 状态良好、充足 | |

### 2.2 客户服务

| 序号 | 检查项 | 标准 | OK/NG |
|------|--------|------|-------|
| 1 | 进店问候 | 客户进店3秒内问候 | |
| 2 | 咨询响应 | 客户咨询30秒内响应 | |
| 3 | 收银效率 | 排队等待≤3分钟 | |
| 4 | 会员引导 | 引导非会员注册会员 | |
| 5 | 投诉处理 | 客诉5分钟内响应 | |

### 2.3 环境维护

| 序号 | 检查项 | 标准 | OK/NG |
|------|--------|------|-------|
| 1 | 地面清洁 | 无污渍、无杂物 | |
| 2 | 灯光照明 | 无损坏、亮度适当 | |
| 3 | 温度/湿度 | 温度22-26°C | |
| 4 | 音乐/音量 | 背景音乐适当 | |
| 5 | 气味 | 无异味 | |

---

<!-- chunk: 门店运营检查单.三-闭店后检查-每日 -->
## 三、闭店后检查（每日）

| 序号 | 检查项 | 标准 | OK/NG |
|------|--------|------|-------|
| 1 | 日结对账 | 销售额与系统一致 | |
| 2 | 库存盘点 | 高单价商品日盘 | |
| 3 | 现金管理 | 现金入柜/入保险箱 | |
| 4 | 设备关闭 | 非必要设备断电 | |
| 5 | 安全检查 | 门窗锁好、监控正常 | |
| 6 | 交接记录 | 交接事项已记录 | |

---

<!-- chunk: 门店运营检查单.四-周度检查 -->
## 四、周度检查

| 序号 | 检查项 | 标准 | OK/NG |
|------|--------|------|-------|
| 1 | 全品类陈列检查 | 按陈列图检查100% | |
| 2 | 竞品动态 | 周边竞品活动记录 | |
| 3 | 员工培训 | 本周培训计划完成 | |
| 4 | 客户反馈汇总 | 本周客诉汇总分析 | |
| 5 | 促销活动复盘 | 本周活动效果分析 | |

---

<!-- chunk: 门店运营检查单.五-月度检查 -->
## 五、月度检查

| 序号 | 检查项 | 标准 | OK/NG |
|------|--------|------|-------|
| 1 | 门店KPI复盘 | 坪效/人效/转化率 | |
| 2 | 全量库存盘点 | 系统库存与实际一致 | |
| 3 | 设备维护 | 所有设备运行正常 | |
| 4 | 员工考核 | 本月考核完成 | |
| 5 | 会员数据分析 | 新增/活跃/复购率 | |

---

<!-- chunk: 门店运营检查单.六-检查结果处理 -->
## 六、检查结果处理

| 发现等级 | 处理方式 | 时限 |
|---------|---------|------|
| 立即整改 | 影响客户体验或安全的问题 | 当日 |
| 短期改进 | 运营规范不达标 | 3日内 |
| 持续优化 | 可提升的改善项 | 本月内 |
