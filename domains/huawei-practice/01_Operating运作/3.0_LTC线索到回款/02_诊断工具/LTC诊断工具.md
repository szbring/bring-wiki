# LTC诊断工具

<!-- chunk: ltc诊断工具.成熟度评估-五级 -->
## 成熟度评估（五级）

| 级别 | 特征 |
|---|---|
| L1 | 销售全靠个人关系，无流程，结果不可预测 |
| L2 | 有CRM但数据不全，销售漏斗不透明 |
| L3 | 销售流程规范，漏斗可见，有预测能力 |
| L4 | 数据驱动销售决策，赢率分析指导资源投入 |
| L5 | 智能化销售，AI辅助商机判断和客户洞察 |

## 快速诊断问题

1. **「销售团队现在手里有多少商机？下季度大概能完成多少目标？」**
   - 说不清楚 → L1-L2，销售漏斗不透明
   - 有大概数字但不准 → L2-L3
   - 有数据支撑的预测 → L3+

2. **「一个商机从出现到成交，平均要多久？成单率大概多少？」**
   - 不知道 → L1-L2
   - 有感觉但无数据 → L2
   - 有准确数据 → L3+

3. **「销售和技术/交付的协作是怎样的？有没有因为销售承诺过多导致交付出问题？」**
   - 经常出问题 → 无铁三角机制
   - 偶尔出问题 → 机制不完善
   - 基本对齐 → 铁三角运作良好

4. **「大客户是靠哪几个销售维系的？如果他们离职，客户关系怎么办？」**
   - 严重依赖个人 → 客户关系管理缺失
   - 有一定机构化关系 → L2-L3

## 访谈问题库

### 销售负责人
- [[LTC]]从线索到成单分几个阶段？每个阶段的退出标准是什么？
- CRM系统用得怎样？销售愿意录入数据吗？
- 销售预测准确率大概是多少？
- 大客户管理机制是什么？有没有客户分级？

### CEO
- 销售团队最大的问题是什么？
- 有没有因为销售承诺和交付能力不匹配而丢单或损失客户的情况？
- 对销售团队目前最不满意的是什么？
