# LTC — 从线索到回款（Lead to Cash）

<!-- chunk: ltc核心理念.一句话定义 -->
## 一句话定义
从销售线索到合同签署再到现金回款的端到端销售管理流程。

<!-- chunk: ltc核心理念.ltc的五大阶段 -->
## LTC的五大阶段

```
线索 → 机会点 → 方案与报价 → 谈判与合同 → 回款
```

| 阶段 | 核心工作 | 关键工具 |
|---|---|---|
| 线索管理 | 线索评分/筛选/分配 | 线索评分模型 |
| 机会点管理 | 客户需求挖掘/方案匹配 | 销售漏斗/MEDDIC |
| 方案与报价 | 定制方案/报价单 | 产品配置工具 |
| 合同管理 | 合同谈判/审批/签署 | 合同模板/审批流程 |
| 回款管理 | 发票/回款跟踪 | 财务系统 |

## 华为LTC的核心思想

### 铁三角（Iron Triangle）
华为LTC中最著名的组织创新：

```
       客户经理（AR）
       /          \
方案经理（SR）——交付经理（FR）
```

- **客户经理**：客户关系，负责关系建立和维护
- **方案经理**：技术方案，负责解决方案设计
- **交付经理**：项目交付，负责项目成功落地

三角形代表「以客户为中心，三个维度同时支撑」，消除了「销售拿单、技术做方案、交付背锅」的割裂。

### 关键指标
- 线索到商机转化率
- 商机到合同转化率（Win Rate）
- 销售周期（从线索到回款的时间）
- 合同质量（条款风险、回款条件）

## 薄云咨询在LTC的服务

**典型客户场景**：
- 销售线索管理混乱，靠销售个人维护
- 销售漏斗不可见，无法预测收入
- 合同条款不规范，回款风险高
- 销售与交付割裂，客户体验差

**核心服务**：
- LTC流程诊断与优化
- 销售漏斗管理体系建立
- CRM系统选型与导入
- 销售绩效体系设计
