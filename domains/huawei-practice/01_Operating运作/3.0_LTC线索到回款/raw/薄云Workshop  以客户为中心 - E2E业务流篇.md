# 薄云Workshop | 以客户为中心 - E2E业务流篇

腾讯文化3.0中，新愿景使命只八个字：“用户为本，科技向善”；

华为成立至今，始终坚持“以客户的价值观为导向，以客户满意度为评价标准”。

客户是把产品转化为商品的第一主体，以客户为中心在各大厂商中植根已久。

那么对于我们处于爬坡阶段的商业机构而言，该如何理解以客户为中心？

要素一：客户选择商业机构面临的首要问题是确定谁是我们的客户，做好客户选择是切实落实以客户为中心经营理念的基础。

01客户选择的核心是战略匹配——“门当户对”正确地选择客户，是在清晰企业战略的基础上，进行价值匹配，战略合作伙伴是命运共同体，客户选择要和我们今后的目标，和阶段性能力相匹配，要基于价值互相认可的等价交换原则来展开。

企业提供的价值犹如“高富帅”：产品好，服务好，价格优，帮助客户发展业务，共建生态，与客户一起面对未来有可能出现的问题。

企业追求的价值好比“白富美”：商业成功，有利润的收入，有现金流的利润，市场地位、规模等潜在获利机会。

建立客户战略伙伴关系是客户选择的终极目标，与客户构建良好的供应关系，是长期发展，持续经营的关键。
