# 供应商与客户同样重要

ISC 理念带给我们的启示
国内财务部/采购核算处
“客户就是上帝”，这是企业为赢得客户，争取更大市场份额所遵循的一句格言。但是，在当
今科技飞速发展和产品品种不断膨胀的过程中，在市场竞争日益激烈的压力下，在各种国际先进的
管理理念和经营手法的冲击下，要取得一定的市场份额，赢得更多的客户，仅仅停留在过去的观念
中往往会使企业走入经营的误区。客户对产品的交货期和服务要求的不断提高，使得企业必须提高
对市场的响应速度，而企业对其供货方的响应要求也因此提高。但是，是否所有人都能接受“供应
商与客户同样重要”这种观念呢？
国内财务部采购核算处因为工作性质决定了它与供应商的联系非常广泛，因此如何看待与供应
商的关系是摆在采购核算处每一位员工面前的考题。曾经，我们为每周能按期付款而暗暗自得，“瞧
我们的付款多么及时！”；我们为热情地回答供应商的查询电话而暗暗自夸，“我们的服务还不错
嘛！”，可是真的如我们所想的吗？供应商的感觉又如何呢？过去我们很少想过，我们甚至认为，
偶有拖欠，也是很正常的事情，为公司缓解资金压力嘛！
近两年来，在公司“集成供应链”项目及“财务四统一”项目的推进下，在财经管理部一次又
一次“增强责任心”、“提高服务意识”、“批评与自我批评”等活动的开展中，我们开始反思自
己。我们能为供应商做些什么？如何更好地提升我们的服务水平？过去我们常常挂在嘴边的一句话
“提高公司核心竞争力”，此刻我们似乎有了更深的理解。
通过近两年一系列的付款流程重整及持续优化，我们取消了重复的审核环节，使发票的入账时
间比以往缩短了6 倍之多，在采购全流程的效益提高的同时，对供应商的付款及时率也达到了95％
以上。也许有人会问，如此及时的付款，是否有考虑过市场回款的情况？的确这是一个非常尖锐的
矛盾，我们在市场发货以后，可能由于种种原因造成公司的收款情况不理想，而另一方面，我们又
要尽力维持对供应商的正常付款，如此下去，势必影响到我司的现金流量，造成资金压力。但是如

果我们因此而拖欠供应商的货款，就会降低公司信誉，影响与供应商的关系，最终影响对用户的及
时供货。实际上，从整个集成供应链的角度看，企业应该和他的供应商共同开发、互为补充、一荣
俱荣、一损俱损。如果企业目光短浅、盲目自大，并因此而失去供应商的信任，那么短期内我们失
去的可能是品质保证，长远丢掉的就将是市场份额和竞争能力，对一个企业，还有比这更可怕的吗？
有了这样的观念还不够，因为我们的目标是与国际化接轨、向业界最佳靠拢，为此，我们从小
处着手进行了一点一滴的改进：
1、建立与供应商的对账制度，使得双方的账务准确明晰，并能及时发现问题共同改进。
2、运用财务付款自动传真系统，将每周的付款情况通过传真的形式及时告诉给供应商，以便
于其对应收款的核销及时掌握回款情况。
3、主动走访供应商，了解问题并跟进解决。7 月份我们集中走访了部分业务量较大的供应商，
就我司的开票要求、付款政策等与供应商进行了充分的沟通，并现场解答了供应商的疑问，并将现
场不能解决的问题带回来，请相关部门配合解决跟进，将结果知会供应商，此举受到了供应商的好
评。
4、制定与采购部门的定期沟通制度，听取相关部门的意见
财务付款需要采购部门的大力配合，部门间有了这种定期沟通，我们可以将需要采购部联手解
决的问题及时提出来，同时听取采购部门对采购核算及付款的建议，不断改进我们的工作。
5、设立统一的对外接口平台，加强内部管理，提升服务水平
为方便与供应商的联络，避免岗位流动频繁给供应商带来麻烦，我们改变过去每位会计各管一
块的模式，设立统一的对外接口平台，由一个人专门负责接收供应商的发票及回答供应商的各种咨
询电话，同时制订了部门内部管理规定，对员工的服务质量制订了严格的考核标准，从制度上保证
了“提升服务水平”这句话落到实处。
仅仅做到以上这些，并不意味着我们对供应商的服务已经足够了，因为供应商最关心的莫过于

能否按时收回到期款项。因此，我们通过多方听取供应商、采购部门的意见，在国内财务部领导的
大力支持下，着手对提高付款的及时性方面做了较大的改进，并在试行期间得到了采购部门和供应
商的广泛认同，我们的采购核算水平又上了一个新的台阶。另外，对于倍受困扰的异地收货问题，
我们也提出了建议，推动相关部门着手拟制相关流程。
我们所做的一切都是为了供应商的满意，因为在我们的眼里，供应商与客户同样重要。但是我
们所做的与我们能做的以及我们应该做的还有不小的差距，目前我们与供应商的沟通中仍然存在一
些问题，我们将不断发掘工作中的不足，并加以改进，总结经验教训，继续深入贯彻这种全新的经
营理念，以业界最佳为榜样，全面提升对供应商的服务水平，使我司的采购核算水平再上新台阶，
为率先实现采购核算与国际接轨而追求不息。
（选自2001.1.18 第113 期《华为人》)
