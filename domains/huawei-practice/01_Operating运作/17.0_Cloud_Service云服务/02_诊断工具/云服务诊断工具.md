# 云服务/SaaS运营诊断工具

<!-- chunk: 云服务诊断工具.成熟度评估 -->
## 成熟度评估

| 级别 | 特征 |
|---|---|
| L1 | 卖完就走，无客户成功，续费靠关系，流失率高 |
| L2 | 有客服，但被动响应，不追踪客户健康度 |
| L3 | 有CSM，定期回访，NPS可追踪，续费率>80% |
| L4 | 客户健康度模型运转，主动预警，扩展收入增长 |
| L5 | AI驱动客户成功，个性化干预，NRR>120% |

---

<!-- chunk: 云服务诊断工具.核心指标-saas必看 -->
## 核心指标（SaaS必看）

| 指标 | 说明 | 健康值 |
|---|---|---|
| 客户留存率（Retention） | 续费客户比例 | >85% |
| 收入留存率（NRR） | 含扩展的收入留存 | >100% |
| 流失率（Churn Rate） | 流失客户/收入比例 | <10%/年 |
| LTV/CAC | 客户生命周期价值/获客成本 | >3 |
| 产品使用率（DAU/MAU） | 活跃用户比例 | 因产品而异 |

<!-- chunk: 云服务诊断工具.快速诊断问题 -->
## 快速诊断问题

1. **「你们的年度续费率是多少？客户流失最主要的原因是什么？」**
   - 不知道/低于80% → 紧急
   - 80-90% → 中等，有改善空间
   - 90%+ → 良好，关注扩展

2. **「有没有CSM（客户成功经理）角色？他们主要做什么？」**
   - 没有 → L1-L2
   - 有但主要做客服/续费催款 → L2
   - 主动帮客户实现业务价值 → L3+

3. **「如何判断一个客户是否有流失风险？」**
   - 客户主动说要不续费才知道 → L1-L2
   - 有健康度指标但不系统 → L2-L3
   - 有健康度模型，低分自动预警 → L3+

## 服务包

**SaaS客户成功体系建设（3-4个月，20-30人天）**：
- 客户健康度模型设计
- CSM角色定义与培训
- 全生命周期管理流程（入驻→激活→成熟→扩展→续费）
- 预警与干预机制
- 续费转化流程优化

**报价**：35-60万
