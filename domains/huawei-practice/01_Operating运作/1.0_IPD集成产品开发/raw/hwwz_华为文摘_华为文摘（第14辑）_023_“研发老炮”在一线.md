# “研发老炮”在一线

王来清
“我能胜任一线的工作吗？我该如何一步一步去实现自己的转身？”2017 年1 月，在飞往印尼
的飞机上，我一直在问自己，脑袋里还回响着自己在将士出征大会上的豪言壮语。直到飞机降落在
首都雅加达，我的心情才随之落地，不想那么多，来了就好好干！
意料之外的热情“款待”
事实上，来一线报到之前，我就收到了明确的训战项目和任务。有了工作的“纲”，心里多少
没那么慌张。后来我得知，预备队和代表处前期做了密集沟通，就是想着怎么能根据项目和我们每
个人的特点，尽可能地做好安排。
代表处对我们的重视也是我之前万万没想到的。我是春节前就出征的首批队员。除夕夜，代表
把我们聚在一起，吃烤全羊、喝酒、畅谈未来，就像一家人吃年夜饭一样，热闹程度完全不输于我
们以前“打沟通牌”。还有好几个同事热情邀请我们去家里吃年饭，戏说要照顾好我们这些“单身
狗”。
重大项目部和地区部的主管根据每个队员特点，提出针对性的建议，这对我们从研发转换跑道
直面市场帮助很大。收到这么多的关心，我内心充满感激，也暗暗告诉自己一定不能辜负他们的期
望。出征前一直担心的“职级倒挂、炮灰、到了一线可能被排挤，手有金刚钻但可能使不上劲”，
真是有点多虑了。
遇到的第一个挑战是语言关。第一次见客户之前，我特意自己“收拾”了一番，详细思考了言
行的注意点。到了现场，却发现最大的问题不是衣着言行，而是他们说的英语我压根听不懂，这超
出了我的预料。于是，接下来的日子，我把耳机天天挂在脖子上，有空了就听一会，每次和本地员

工沟通都竖起耳朵听，慢慢找到了感觉。其实，不同的研发将士转身到一线，可能有不同的短木板，
但只要能做到“勤”，自有天道来酬。
终于吃上自己的“狗粮”
为了让我们尽可能发挥所长，平稳融入项目，代表处给我的第一个任务是为客户演示新产品A
和B，这两个产品正是我在研发时负责开发的。在异国他乡看到它们，有种莫名的亲切感，也略带
着点小小的不安——自己做出的“狗粮”终于要轮到自己吃了。
为演示出产品A 的最佳性能，我对备选站点逐一勘察，奔波了一个多星期。最终，站址竟然被
客户选在了客户办公大楼里，而不是空阔的场景。对无线产品来说，这可不是什么好主意，性能演
示风险很大。但客户已明确没有其他的备选站址，项目组非常纠结，如果演示不好对我司的品牌将
可能是重大挫伤，但不演示的又可能失去展示我们的机会。
两难境地下，我只能耐下心来抽丝剥茧，和机关讨论，结合以往的经验，我整理出一个克服站
址问题的方案，并大胆说服项目组拍板就在这个站址上测。
牛皮吹出去了，一下所有的困难都集中在测试执行上了。测试中，需要把16 部终端分别安装在
客户办公大楼对面的新建筑上，之所以强调是“新建筑”，是因为它仅仅完成了混凝土部分，工程
电梯不能用，楼梯没护栏，处处是上下通透的深井、堆砌的杂物，拐角处更是蚊子直扑脸，我只好
把衣领竖起来，能挡一点是一点。16 部终端都分布在不同楼层，操作人员大部分是临时雇的本地人，
沟通和协调也成了执行的难点。
唯一的办法只能是靠自己的两条腿，楼上楼下来回跑，一是优化手机的部署，二是协同和监督
本地人。每次测试下来，满身尘土。事后我跟同事开玩笑“短短十天爬的楼比我一辈子爬的楼还多”。
期间，还出现了个小插曲：为了测出最佳性能，一边调试终端一边退着走，突然听到本地人一声大
叫，我一站住才发现就差一步就退到悬空的楼梯井里了，那可是7 层楼的高度，现在想着还很后怕。

在大家的共同努力下，2 天完成建站、1 天调通、1 天性能优化到最佳，成为这个产品在华为历
史上，建站最快、开通最快、冲峰值最快的一次，实测性能也是产品此类场景下的最佳性能。我们
用最佳的网络性能先后接待了3 批客户参观，向客户展示了此领域的业界领先竞争力，最终赢取了
客户的赞赏。
产品B 的演示进度压力也非常大，客户尤为重视。2 月20 日演示，2 月17 日还有一个物料在
海关那里要开箱检查。我怀着忐忑的心情，在群里求助“xx 物料阻塞了项目进展”，同事们的响应
速度超出我的意料，他们马上和海关沟通、迅速安排运输。晚上8 点，物料就齐套了，接下来的演
示也非常顺利。
我的真实感受是：与一线的融合，需要谦卑开放，但更关键是要把事情搞定。该反馈问题时一
定要提出来。代表处的一个兄弟说得特别好：“思想上不长草，什么困难都好解决。”
对研发干部来说，经验要延续，但内心状态一定要清零，工作上瞄准目标，只要是自己能干的
就尽量多干，比如勘站、关键部分走线及硬件审查、在工地指导测试、准备客户沟通材料……在一线，
不能奢望别人安排好你的一切，我们应该从能承担的工作先切入进去。活干起来了，其他的自然会
顺畅起来。
我经历的那些“第一次”
演示任务刚完成，客户的标书就来了。
我第一次参与应标，按捺不住地兴奋。要知道，这可是一个学习的大好机会，能借此进一步了
解客户的现网、以及客户所期望的目标网，还能接触端到端的答标流程和方法。我花了很多时间熟
悉标书，承担了一部分的答标工作，深刻体会到什么是“在战争中学习战争，在游泳中学习游泳”。
招标时间紧、任务重，大家连续奋战多天，交标当天凌晨4 点，完成所负责的部分后，导师叫
我先去办公室休息一下，一觉醒来已经是早上8 点。回到会议室，竟然发现很多人都在忙碌着，一

夜未眠，这才算切实感受到了市场一线冲锋的那股子狠劲。中午按时交标，不少兄弟直接在会议室
原地休息，有趴在桌子上的，有斜躺在椅子上的，真是八仙过海，各显睡功。
辛苦耕耘终有回报，我们一举中标。在庆祝宴会上大家都喝得很high，个个激动得热泪盈眶。
还有更多的第一次。巴展之后，客户高层提出在印尼搞个“小巴展”，通过workshop 讨论出
新技术落地印尼的运作机制，这个组织任务落到了我肩上。说实话，我从来没见过小巴展或峰会，
没见过猪跑的人要杀猪啦，如何下手？
思来想去，不懂就问，多跑腿，多动嘴皮子。我参考巴展信息和客户的战略诉求、组织同事讨
论，向领导求助等，多管齐下，好不容易为客户定制小巴展的议题，这才发现组织工作不简单，头
绪特别多，时间又紧急。想来想去，只有呼唤炮火。机关专家亲临一线支持，成立混编现场作战部。
这样一来，峰会材料的准备和评审、与客户对齐议题、客户邀请、会场的布置等，紧张有序地开展
起来了。
客户高层出席了小巴展并现场致开幕词，认真参观了每个展台，并展开了深入探讨。围绕新技
术和解决方案如何落地、以及新业务开展为主线，参会客户深入探讨未来两年的可执行规划。我的
第一次峰会组织工作画上了圆满的句号。
我想，接下来还会有越来越多的“第一次”等待我去开启，而这些经验都将沉淀为美好又珍贵
的记忆。
除了工作外，一线还有精彩的生活：有时，坐在开阔的大海边，慢慢地享用着鲜美的seafood，
喝点小酒，近期的劳累就不知不觉地散去。有时，和同事一起外出跑步，享受大汗淋漓的快感。有
时，和同事一起逛商场、一起做饭……
从刚来一线的慌乱，到现在慢慢地找到了感觉，真心感谢代表处兄弟的领路。还记得重大项目
部主管座谈时说的：过去的辉煌都已成为过去，未来就看你们怎么闯出又一片天了。我知道，过程
充满艰辛，但选了这条路就一定会努力走下去。
