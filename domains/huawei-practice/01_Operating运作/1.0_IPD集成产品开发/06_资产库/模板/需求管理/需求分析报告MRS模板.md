## 第五章：非功能需求（DFX需求）

> DFX = Design For X，包括可靠性/可维护性/安全性/可测试性等质量属性

| 类别 | 需求描述 | 量化指标 | 优先级 |
|------|---------|---------|--------|
| **可靠性** | 平均无故障时间（MTBF）| ≥ ___ 小时 | Must |
| **可用性** | 系统可用率 | ≥ ___% | Must |
| **性能** | 响应时间 | ≤ ___ ms | Must |
| **安全性** | 认证/加密标准 | | Must |
| **可维护性** | 平均修复时间（MTTR）| ≤ ___ 小时 | Should |
| **可测试性** | 自动化测试覆盖率 | ≥ ___% | Should |
