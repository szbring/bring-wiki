# 华为BPA流程架构总览

> 华为业务流程架构（BPA）共17个一级流程，分为运作、使能、支撑三大类。
> 每个一级流程设唯一责任人（GPO，Global Process Owner），包产到户，对流程规划、设计、运行和结果承担全责。

<!-- chunk: 华为bpa流程架构总览.三类流程的关系 -->
## 三类流程的关系

```
客户需求 ──→ IPD(1.0) ──→ 产品/解决方案
市场洞察 ──→ MTL(2.0) ──→ 销售线索
销售线索 ──→ LTC(3.0) ──→ 合同/回款
客户问题 ──→ ITR(4.0) ──→ 问题解决

使能流程（5-9、15）为运作流程提供专业支撑
支撑流程（10-13）为整体公司运营保驾护航
```
