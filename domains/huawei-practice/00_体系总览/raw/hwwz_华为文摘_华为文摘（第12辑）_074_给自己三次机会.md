# 给自己三次机会

宫玉庆
宫玉庆，2005 年加入华为接入网软件开发一部，2006 年从研发转行销来到阿尔及利亚。2011
年在北非地区部进行区域项目支持，曾参与埃塞电信项目。
在海外差不多7 年了,也是个人从23 岁到30 岁的重要成长阶段。印象里触动比较大的是2008
到2009 年。
2008 年阿电招标接入网，当时是自己第一次操作项目，没有经验，结果技术标下来以最后一
名进入测试与商务。领导很恼火，骂我把公司技术第一的成熟产品卖了个最后一名。跟领导沟通2008
年三季度的PBC 时，领导大笔一挥写了一行字“接入网测试第一”，接着说：“做不到就考虑换人。”
那时才意识到了事情的严重性，虽然在个人能力方面，包括外语、技术等都不是很好,领导也
一直不太满意，但总不能就这样回去。接下来的测试中，我憋着一口气把每项工作都做到自己感觉
再没空间为止。记得当时测试的每一根尾纤都是我一个人整理保管，跟研发测试的同事一起把每一
个特性、每一个需求、每个测试指标，都做了引导。有一天感觉压力大得透不过气，跑到隔壁的机
房,悄悄大哭了一场。没想到被一个客户发现了,他很感动，当场把测试结果拿给我看，说：“华为现
在是最好的。小伙子，你不要担心。”男人哭吧哭吧不是罪，最终我们以技术第一名进入商务标，
那年我拿了华为生涯的第一个A。
进入商务标，本以为很快可以看到好结果，却因为局方内部变动，整个标就废掉了，要来年重
新发标。听到这个消息,充足的劲一下子就泄了下来，再也没有那么大劲头了。但项目还是要继续，
来年的技术标、测试、商务标还是一如既往。每天都对自己说男人要坚持，跟客户经理一起，把所
有细节做到极致。功夫不负有心人，我们以3 个第一名拿到了项目最大份额，同时突破了NGN 与

IP 城域网。紧接着2010 年接入网招标，我们以高商务拿到高份额又取得胜利。2010 年，我获得公
司金牌个人奖。
成功很少是运气促成，需要个人不断努力，而失败后的成功，喜悦可以加倍。
2012 年参与的埃塞重大项目，是我在华为经历过规模最大、合作人数最多的一次，我很感激
有这个机会。为了这个项目，在答标的两个月里，整个团队几乎都是通宵，而且精神饱满。我看到
华为这些年轻人的无限潜力，每天一睁眼就是方案，一遍遍刷新、一遍遍评审。面对西方友商，我
们能够取胜，不是我们个人有多专业多高端，而是整个团队的团结一致。一帮不牛的人团结在一起，
在后方的支持下，却能做出很牛以至于让人敬佩的成绩，站在公司这个巨人的肩膀上，我直到现在
还为整个团队感动。
埃塞项目成功后，任总签发了总裁嘉奖令。对于总裁嘉奖，需要理性看待。代表处的兄弟们顶
着巨大压力连续奋战了两年，才有了项目整体的好结果。公司给予嘉奖，是一种肯定和鼓励，对于
每个参与项目支持的人，奖励可能不同。但即使没有这个奖励，能参与这个项目得到锻炼，我个人
也觉得知足。在个人职业生涯中，这种机会不多。
代表处的生活很有规律，除了工作，我每天坚持学习法语，周末打篮球。在海外除了工作，还
要让自己得到更多的锻炼。我对自己的外语、宣讲、技术能力各方面还是不满意，要在这个工作环
境中生存下去，必须不停地做加法提高自己。别人看一遍就能讲好的方案，我可能要练几十遍；对
解决方案的理解，我可能要花上别人几倍的时间，除了技术本身，还希望了解这个技术的背景与基
础，总是不希望做最后一名。
我的性格可能是有点自卑。自卑的好处是总觉得自己挺差的，所以要一直不停地学习、进步。
如果说技术和客户关系要靠天赋，那么抓住机会要靠勤快，我相信勤能补拙。
我也相信天道酬勤。可能你付出了一次，没有得到认可。请继续保持这份热情，给自己三次机
会，早晚会得到认可。

我想感谢我老婆，大家别嫌我不够大气，我应该感谢她。到海外快7 年了，孩子快3 岁了，
但我只见过孩子3 次。在外一直出差，也没法把家属带来，多亏了她的支持和付出，才让我能够全
身心投入工作。
（选自2013.05.03 第259 期《华为人》）
