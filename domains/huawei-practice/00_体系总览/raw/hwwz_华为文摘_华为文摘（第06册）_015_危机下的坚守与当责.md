# 危机下的坚守与当责

里氏9.0 级，3 月11 日，日本发生了千年一遇的大地震，由此引发了剧烈海啸和一时难以控
制的核泄漏。房屋倒塌、数万灾民无家可归、电力供应不足、通信中断……这样危难的时刻，这个民
族表现出的坚毅、献身、冷静、秩序、谦和让世人看到他们对于职业的精神、自身的气度和内心力
量。
在地震发生后，华为公司第一时间和客户取得联系，协助客户恢复通信；无论中方员工还是日
本员工，都坚守岗位，做好自己手中的事。
一、华为协助客户共渡难关
3 月11 日本仙台地区遭遇里氏9 级地震，人民生命财产遭受巨大损失。
华为日本代表处在第一时间便行动起来，协助各运营商抗击这场自然灾害。华为公司全力以赴，
全球调拨人力与资源与客户共渡难关。
在强震刚过去，余震不断的情况下，日本代表处员工便纷纷自动回到工作岗位，积极联系客户，
查看网络状况，积极恢复各类通信。
3 月15 日，应客户要求，华为员工分别采用在家办公，以及远程操作的方式协助客户工作。当
日，恢复因电力中断的E-Mobile 客户315 台WCDMA 基站业务。
3 月16 日，新发货的115 台WCDMA 基站达到东京；华为公司日本救灾工作组在公司网站上
进行了相关情况通报。
3 月17 日，E-Mobile 华为WCDMA 基站业务新恢复33 台，累计恢复381 台。本日，E-Mobile
在大坂NOC 给华为留了5 个位子集中办公，日本代表处安排相关人员前往大坂，到客户处报到并

开展工作。当地时间15:00 客户则在大手町办公室与华为开会讨论未来项目开展计划。
3 月17 日，公司机关所采购的口罩、防雨帽、药品、食品、卫星电话等物资由中国发往救灾
一线部门。
3 月18 日，由于华为的配合，软银WCP 项目进展没有任何延误，客户非常满意，发来感谢邮
件。在公司，相关抢通保障组、安全与健康指导组、行政物资及装备保障组、信息通报组等正式成
立并立即投入紧张的运作当中。
在机关，销售与服务、研发等各部门中方人员纷纷请战前往日本救灾一线，机关在员工自愿的
基础上，进行严格筛选，选派身体、心理素质过硬的数十名员工，启动相关签证申请。
3 月21 日，日本福岛核电站辐射泄露逐步得到缓解。客户开始启动相关测试项目，并开始前
往灾区现场勘察。华为协助客户的救灾工作即将进入新的阶段。
二、和客户一起面对
周四军王莹莹
3 月11 日日本发生9 级特大地震和海啸，3 月12 日后日本东京电力福岛核电站发生爆炸和核
辐射危机，这些给正在进行的各个项目带来了困难，但是各项目组的同事，依然用他们的方式与客
户一起面对困难。
维持原计划不变
地震发生后，日本软银WCP AXGP 项目组NTS 组吕岩、杨卉等带领成员4-5 名员工，在余震
不断、交通没有完全恢复的情况下仍然在离宿舍很远的仿真办公室工作到深夜，加快项目的规划仿
真，将报告和规划结果提前发给客户。
研发现场负责人李兴带领现场测试团队通宵加班，优先把现场的设备测试工作完成，以便发生

任何情况时，都能按时完成Level3 的测试报告。
3 月17 日抵达日本的MVI 货物，项目物流小组成员曾祥国3 月18 日就安排完成清关运输；3
月18 日货物运至大手町, 李湛、王毅等TSD 安装小组在余震经常不断的情况下完成现场安装。
紧接着，在项目经理周四军和产品经理哈勤和的组织下，项目组核心成员和W 客户负责人进
行了交流，大部分工作都是按进度计划在进行，并在会上接受客户提议的由华为帮助客户建设测试
环境，提供培训和支持的要求。项目组的坚守使客户深为感动。W 客户也发来了感谢信：“昨天余
震还在不断发生，而且停电造成交通不便，在这种情况下各位都参加会议，我们为此表示谢意。”
继续测试
3 月12 号，客户到丰州机房，余震不断，看到机房情况确实不安全，把服务器搬到新宿机房。
由于4 月1 号是KDDI 客户PTN 专线项目给我们的商用时间，也是KDDI 给客户承诺的商用时间。
项目组没有时间再进行休整，在客户计划的时间没有变更之前，必须和KDDI 同舟共济。
公共交通还是没有完全恢复，从宿舍到机房，大家转了几趟车，连续几天如此，项目组绝大多
数成员都是80 后，第一次遇到这种灾难，第一次遇到这种关键考验，压力越来越大，测试仍然在继
续，各种混乱的消息不断传来，都在考验大家的心理素质。
经过几天的讨论，项目组的目标和共识也越来越清晰：只要客户在，我们就一定在，只要客户
不说推迟，就一定不会说推迟，必须按时完成各项测试和升级工作。
把样机寄回宿舍
Blaze 是我们为软银定制的第一款智能手机，这个项目的成功与否影响软银对我们手机能力的
判断，非常关键。
产品在下个月就要上市，之前要做好最后一次样机确认，软银的客户要在下周二上班就能收到
测试样机。办公室不开放，几位留守东京的同事把一百多台终端邮寄到了宿舍，客厅仅有的十几平

米，摆了一地的样机。大家把这近百台样机分批一台台按要求做好软件升级，确保正常可用，同时
对对应版本完成现网测试。
一位日本客户说：“感谢你们即使在这样（困难）的情况下，仍然能够提供持续的服务支持。”
简单朴实的话告诉我们，越是在灾难面前，越需要我们坚守在这里，这才是对客户的真正支持。
三、另一种支持
胡日查
日本发生有观测史以来最大级别的9 级地震，随后6 米海啸，东北海岸线遭受双重灾难袭击。
东京虽然距离震中大约300 公里的样子，可坐在办公里面仍旧能感觉到6 级震颤。
震动过后，从大手町办公楼看对面的摩天大楼，相互摆角几乎有5－10 度的样子。平时感觉厚
重敦实的钢铁大楼，现在看起来象纸做的盒子，在风中摇曳，平生第一次听到大厦钢铁框架发出吱
吱咔咔的声音。
最初的震动过后，大家都从安全楼梯离开办公室，最后只剩下4－5 个日本人在议论，本地秘
书过来催促到前面的皇宫空地集合。先下去的同事打电话，让我给她们拿衣服，收集了两件大衣和
手袋下楼。后来听说，在走安全楼梯时，有两个日本女孩，不知道是不是大楼管理人员，她们一直
站在楼梯门口拉着门，让其他的人先行通过。等所有人走完，她们自己才走。
大震五分钟之后走在丸之内的大街上，看到日本人在等红绿灯，除了地面上多了很多的人之外，
所有事情都还在有条不紊地进行，没有任何喧哗，也没有警察维持秩序之类的。一起步行的本地员
工还给我讲了一个小故事，以前她小时候的地方发生地震火山灾难，大家都很害怕，一天震动上百
次，有个面包店的老板一周后，开发出来火山面包（里面是红色的果酱代表火山熔岩），生意很好。
天又阴了一点，公司同事大都聚集成2－3 拨，相互关怀。我动身离开广场，开始筹划回家。
东京JR 线全线停运，地铁大厅滞留了很多人。所有的公共电话，公共汽车和公共电视都排起了长队，

大家有序地等着打电话回家报平安，等着公车回家，等着电视看最新的媒体播报。电视上开始出现
实况直播，东京湾港口大火，海啸2 米警报更新成6 米。广播不断播放JR 的致歉。
干道都是汽车，一点一点往前蹭，没有堵死。路两边满是回家的人群，很安静，还是没有任何
喧哗的声音，看上去像晚餐后的消食散步。路上没有一块玻璃一个路灯故障，所有的商店灯火通明，
每个人都目标明确，显得那么坚定不移。
走在平和大桥上，凛冽的海风从乌黑的海面上刮来，我们裹夹在人群里面，一路余震一路向前。
大家想尽各种办法回家，当晚有同事步行了15 公里才走回去。
在第一波地震发生后，我拨通了公寓电话，告诉家人两件事：这在东京是常事；如果感觉比较
厉害，就拿上一瓶水到卫生间。
在东京，一切都还是有序进行，每个人好像都不愿意再多添麻烦，这也是一种安静有力的支持
吧。
四、供应链在行动
订单管理部
3 月11 日下午14:30，午休后的同事们听说了日本地震的消息，订单管理部凭经验意识到“救
灾就在当下”。紧急联系一线人员，但是电话那头每次都是无法连接的忙音。
下午15:20，电话终于接通了，确认人员都是平安的，首批要货需求还不清楚。
当天是周五，第二天是周末，而紧急需求随时都可能传回来，怎么办？
下午17:30，亚太合同履行部主管彭红平内部发布消息：取消周六的活动计划，所有统筹人员
务必保持24 小时通信畅通，时刻准备着。
凌晨1:50，彭红平被电话铃声叫醒，“日本地震，之前有一个客户合同正在履行，需要100
套无线DBS3900 设备，现在需要紧急发货用来救灾”，在了解详情之后彭红平便开始了紧急部署。

凌晨2:00，彭红平打电话给合同统筹黄梅和庞建平，“紧急前往公司，立即组建项目组，马上
召集电话会议”。
凌晨2:40，日本抗震救灾交付保障临时项目组成立，包括市场、区域供应链、统筹、成套、PMC、
物流各个环节
凌晨3:00，项目组第一个电话会议紧急召开，机关各环节在线上展开需求澄清、库存核查、流
程升级、运输资源确认；一线人员由于电话线路不好，只能Espace 参与讨论。
凌晨5:00，发货确认，第一期项目简报发布。
3 月13 日早上，设备顺利抵达日本成田机场！
自然灾害摧毁了家园，但只要每个人都齐心协力，都可以为抗震救灾奉献自己微薄却巨大的力
量。
（选自2011.4.3 第234 期《华为人》）
