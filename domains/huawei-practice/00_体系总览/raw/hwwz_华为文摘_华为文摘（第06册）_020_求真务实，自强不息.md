# 求真务实，自强不息

记FELLOW 吴东君
固定网络BU
华为首批FELLOW 中有这样一位引起了我们的注意：他戴着一副金边眼镜，偏黑的肤色，敦
厚的笑容，说起话来文质彬彬。1994 年大学毕业加入华为，从一位普通的工程师到公司首批
FELLOW，他用18 年的青春岁月走出了一条研发人员的成功之路。
他就是华为公司“土生土长”的FELLOW——吴东君。
拓荒的勇士
吴东君的微博中有这样一句话：“因为可能失败就放弃，也就失去了潘多拉魔盒里面唯一美好
的东西——希望。”
2001 年，华为公司在软交换产品遭遇挫折，之前开发的SoftX1000 只卖出了二十多套，
SoftX2000 一套都没有卖出去，华为被友商远远甩在了后面。为此，公司计划推出新的软交换产品
SoftX3000，以扭转当时的不利局面。如何让SoftX3000 具有强劲的竞争力？这对当时担任系统工
程师的吴东君来讲，是朝思暮想的事情。俗话讲“自古华山一条路”，有路就有希望，虽然艰险，
跨越过去，就是成功。吴东君一方面快速充电，一方面广泛征询建议，博采众长。我们一定要推出
“移动和固定两个领域融合的软交换产品”，在他的脑海中，SoftX3000 架构方案渐渐清晰。
2001 年4 月的一天，技术专家们齐聚南山脚下的一间茶馆，进行SoftX3000 架构方案终审。
面对众多专家抛出的疑问，吴东君逐一解答；面对专家们的意见，吴东君虚心汲取，现场记录修改
完善。他知道，只有充分吸取大家的精华，才有可能柳暗花明。最终，SoftX3000 架构方案规划获
得通过，这一方案不仅承载着“移动交换和固定交换两个领域的融合”，还承载着“2G 和3G 的融
合”。会议休息之余，吴东君环顾周围，生机勃勃，信心满满：新架构的移动固定软交换SoftX3000

一定能像南山上的绿林，枝繁叶茂，郁郁葱葱。之后不到半年，吴东君和他的团队顺利完成了整个
产品架构的搭建。2002 年，华为移动固定软交换SoftX3000 市场上一亮相，便震撼业界，此后数
年成为公司最赚钱的产品之一，引领核心网竞争力全球领先。
2004 年，吴东君转战IMS 项目，这是一个全新的产品，是ALL IP、固定移动融合演进的必经
之路。吴东君和他的团队，用不到半年的时间，一次性地完成了IMS 架构方案。今天，我们欣喜地
看到：IMS 全面领先，它不仅实现了单一产品的成功，还带动了一大批有线、无线网络产品的商用。
从移动固定软交换到IMS，再到VRP，吴东君不断进入新的领域。他是一个勇士，不惧困难，
迎难而上。有人问吴东君：“每一次投入新领域，感觉怎么样？”吴东君微笑着说：“每一次新领
域征战就像在‘拓荒’，当你充满好奇与渴望地投入其中，你会有新的发现。”
这就是FELLOW 带给我们的力量，在技术上追求卓越，敢为人先。
主航道的坚守者
吴东君说：“一个时期总会有一两个活跃的领域主导着市场，顶级科学家的核心价值就是找到
这些关键领域，像爱因斯坦提出相对论，玻尔提出原子结构、原子辐射理论一样。”在工作中，吴
东君就是这样要求自己的。他就像一座灯塔，不仅为主航道指出方向，同时也执着地坚守、捍卫着
主航道。
2008 年，公司启动ALL IP 战略，平台建设是重中之重。当时目标是“用3～5 年时间把VRP V8
打造成ALL IP 时代运营商网络设备的软件平台。”这一年，吴东君担任VRP V8 总设计师，这是吴
东君迄今为止接受的“最具挑战、最艰难的一项工作”。当时，对于如何做出业界竞争力领先的IP
软件平台，大家都感到迷茫，没有方向感。作为VRP V8 的总设计师，吴东君带领团队大胆地提出
“将V8 架构与V5 架构剥离，重构V8”的思路。这是一种全新的设计，与以前V5 的做法完全不
一样，在大家看来这存在着巨大的风险。有人问“V8 采用全新的架构是否太冒进了？”又有人说“放
弃已有的V5 架构是不是太可惜了？”还有人担心说“我们的队伍经验不足，能否完成这项创新？”……

面对大家的不解甚至质疑，吴东君组织召开了技术PK 会议，让所有与会者从不同角度去寻找问题、
风险。吴东君一个人站在讲台上，以他特有的慢语速，一遍遍不厌其烦地解答，每一次解答他都能
提供具体的指导和清晰的方向。
经过数次的PK 会议，无数次方案论证和修正，V8 最终采用了全新的革命性架构设计方案。架
构设计已经确定，然而设计、开发、实现过程更是困难重重，人员新，经验少，当时一位专家说：
“V8 开发，我们好比只有砖头却要盖摩天大厦。”吴东君没有过多的言语，一边自己上战场解决开
发中出现的问题，一边培养新人、建立架构维护等系统工程制度，一次次调测一遍遍对齐，经过两
年不懈努力，VRPV8R1 在阵痛中诞生，其性能全面领先，不负众望。
有人说：“吴东君身上最让人敬佩的是他的责任感和使命感。”2007 年，已离开核心网软件
设计岗位两年多的吴东君，仍然关注着产品线主力产品和平台的未来发展。这一年，对于核心网是
否要落地统一平台战略，One Core 朝哪个方向选择，管理团队一时未能达成共识。一天，核心网
管理团队收到吴东君的一封邮件：“……要在未来赢得竞争力，必须尽快建立核心网统一平台……”
在这封邮件中，吴东君系统地阐述了核心网未来平台发展的方向以及One Core 的战略选择。这封
邮件，一石激起千层浪，引起了管理层和专家的广泛研讨。最终产品线下定决心启动核心网统一平
台战略，为华为核心网产品竞争力的持续提升奠定了坚实基础。很多专家在回顾当时的情景时，不
由心生感慨道：“老吴在哪里，哪里就有定海神针。”
正是这般勇于创新的开拓精神和使命感，让吴东君始终走在技术潮流的前沿，捕捉先机，顺势
而上，不断地构想出面向未来领先的技术方案。
是啊，在探索未知领域的征途中，我们要敢于亮剑，也许正是这一个个idea，一封封邮件，一
次次交流，如涓涓细流般汇聚成河，在河流消逝的地方，海天一色，千帆竞发……
喜爱读书的“黑格尔”
吴东君喜欢思考，他选择乘班车上下班，说这样可以有更多时间来思考；他喜欢哲学，给自己

起了英文名字Hegel（黑格尔）；他喜欢读书，阅读范围非常广，涉及物理、化学、哲学、历史、
文学、社会学等等。在他的办公室中，总能看到一堆堆书籍。因为两地办公，有些书他随身携带，
从国内带到国外，又从国外带到国内。在一次新员工交流会上，有人问吴东君：“您坚持最久的爱
好是什么？它的吸引力在哪里？”吴东君微笑着说：“读书。读书可以让人摆脱时间、空间的限制，
可以与古时或不曾谋面的智者沟通思想。”他带着一丝神往补充道，“学生时代我的梦想是每天能
够躺在哈佛校园的草坪上静静地读书，现在没能如愿，不知道将来还有没有机会？不过读书可以现
在做起，我给自己的目标是争取每年读三十本书。”
广泛的阅读以及深邃的哲学思维，加深了吴东君对架构设计的理解。他常说：“架构师的工作
就好比作曲兼指挥。‘作曲’就是设计方案，先对客户需求认真分析，把已有的素材整合、组装，
再创造性地安排、使用，最终成为一套架构，也就是可以演奏的音乐；‘指挥’可以把系统工程师、
模块架构师、开发、测试等协调起来共同创造产品面向未来的统一的架构，就好比把各种不同的乐
器、不同水平和个性的演员统一成和谐的整体，进而创造出千姿百态的音乐来。”一个合格的架构
师要有这样的特质：对可用资源有着清晰的了解，对业界发展有着灵敏的嗅觉，对商业成功有着强
烈的渴望。
读书与思考，也许这就是吴东君最朴素的成功秘诀。
这就是吴东君：不断拓荒的勇士，主航道的坚守者，喜爱读书的思考者。他常对身边的人说：
“当你瞄准方向，把技术变成追求，那么无论是苦还是甜，都是‘有滋有味’的。”在他看来，“做
点实实在在对社会有意义的事，这样的人生才有价值。”
在这喧嚣的世界中，吴东君保持着内心的宁静，求真务实，自强不息。
（选自2012.8.8 第250 期《华为人》）
