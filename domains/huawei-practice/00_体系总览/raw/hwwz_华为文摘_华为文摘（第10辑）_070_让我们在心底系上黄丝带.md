# 让我们在心底系上黄丝带

若然
Tie A Yellow Ribbon Round The Old Oak Tree
I'm coming home;
I've done my time.
Now I've got to know
what is and isn't mine.
If you received my letter
telling you still want me,
if you still want me.
Tie a yellow ribbon
'round the old oak tree.
……
第一次听到《黄丝带》的旋律，还是在读大学的时候。有一档叫《偷闲黄昏》的音乐文字节目。
每晚黄昏的时候，大家就都守在收音机旁，开始一段美好的闲暇又纯净的音乐时光。记得有一天，
主持人播放了很多告别的歌，快到最后的时候，他说：“接下来我要播出的这首歌是在谈到分别的
时候我非常喜欢的一首，唱得非常伤感。”
后来，我还听到了中文版的优客李林的《黄丝带》：“现在你已将一切带走，我想这一次我也
失去所有；等待你的我的心还在守候……”
以后，毕业、工作、四海为家。
但无论我在旅途的寂寞中，在人生的彷徨路口，时刻会想起这首带着些淡淡的哀愁，又充满着

希望的旋律，常常伴随着脚步匆匆。
很久以后，突然好奇，为什么要用黄丝带来代表对于亲人的挂念？于是上网寻找，却发现后面
是这样一个美丽的故事：1971 年10 月14 日《纽约邮报》刊登了一篇小说：长途车上坐着一位沉
默不语的男子，在同车的年轻游客的盘问下终于开了口。原来他刚从监狱出来，释放前曾写信给妻
子：如果她已另有归宿，他也不责怪她；如果她还爱着他，愿意他回去，就在村口的老橡树上系一
根黄丝带；如果没有黄丝带，他就会随车而去，永远不会去打扰她。汽车快到目的地了，远远望去，
一条、两条、老橡树上挂了几十上百条黄丝带，车上的乘客都欢呼起来。
后来，这个动人的故事被写成了歌曲，传遍了全世界，黄丝带也因为代表着“平安归来”、“重
获新生”等美好的祝愿而广为传播。现在，世界各地的很多人们，为远方的亲人祝福时、为朋友祈
福时、盼望亲人平安归来时，都会扎起黄丝带，让美好的愿望随着风而飘扬。
当法航班机失事的消息传递到国内时，有员工在心声社区发起一个贴子《我们一起来为我们的
兄弟祈福吧——黄丝带》。数万人点击，数百人回复“祈祷奇迹发生”、“希望兄弟平安归来”、
“祝愿平安”……
我们为我们的同事祈祷，我们为乘坐同一班机的同胞们祈祷，我们更为所有乘坐该航班的乘客
祈祷。期望奇迹再现，甚至宁愿希望他们是被某种神奇力量带去另一个地方，肯定还能够回来。即
使这样的希望渺茫，但是我们不会放弃祝福，只要希望与爱在，世界就是美好的。
每天，在世界各地，为了自己的亲人、家庭幸福，华为的兄弟姐妹们像迁徙的鸟儿一样，不畏
风雨、勇往直前。当我们的亲人、朋友要远行的时候，我们能够做的，除了祝福之外，更请在你的
心中，为远行的他们系上黄丝带吧。那永远飘动的丝带，就像那些在我们人生中感动过我们的旋律，
会永远在我们内心深处，让亲人带着祝福和爱去走未来的路。
在大自然的灾难面前，生命也许显得异常脆弱和渺小，但也正是因为这种爱的祝福，在无形之
中能够给我们力量和勇气。

（选自2009.6.30 第213 期《华为人》）
