# 来自《华为本质》- 成长的逻辑

## 核心观点

华为的成长逻辑围绕三个关键词展开：**专注本业、为生存而战、与时俱进**。任正非将"活下去"作为华为最高战略，通过专注通信行业、坚持艰苦奋斗、主动拥抱变革、构建学习型组织，使华为从一家小企业成长为全球通信行业巨无霸。

## 一、生存为大

任正非一直强调华为的战略只有三个字——"活下去"。这一理念贯穿华为成长的全程，源于他幼年经历的战争与贫困，以及父亲在"文革"中的遭遇。"活下去"不是消极保守，而是在激烈市场竞争中先站稳脚跟，再谋求发展。

1991年华为研制程控交换机时，办公地既是车间又是卧室，首批3台BH-03交换机交付后，员工才得知公司已经濒临破产。正是这种"努力活下去"的成长战略支撑华为迅速发展。

即使成为行业巨无霸后，任正非仍说："对我个人而言，我并没有远大的理想，我思考的是这两三年要干什么，如何干才能活下去。活下去，永远是企业的硬道理。"

华为的艰苦奋斗精神体现在各种极端场景中：珠穆朗玛峰5200米和6500米处建设移动通信基站、阿尔及利亚6.8级大地震、智利8.8级大地震等，都有华为人坚守的身影。

## 二、不在非战略机会点上消耗战略竞争力量

### 专注自己的事业

华为二十多年只做了一件事——通信制造。面对房地产暴利、制造业利润丰厚、私募投资火爆等诱惑，任正非始终专注于电信行业，不搞多元化。他以山叶公司（雅马哈）的教训为例：山叶从乐器领域大举扩张到网球拍、摩托车、游艇等，最终因管理技术跟不上而陷入债务危机。

华为是典型的战略型成长企业：有清晰的战略意愿和方向，重视基础性长期性要素的投入，对成长路径有时间和空间上的谋划。

### 计划先行

任正非要求"先瞄准再开枪"，做事必须先做好策划再行动。华为销售人员不会盲目推销，而是先找准客户，进行周密的项目策划后再行动。这种"谋定而后动"的工作准则，避免了事倍功半、劳而无功的现象。

## 三、与时俱进，削足适履

### 变革战略力促华为快速成长

1997年任正非访问美国IBM等公司后，决定开启变革大门，向IBM学习。他强调完全照做，即使"削足也要适履"。"IT策略与规划"项目包括集成产品开发（[[IPD]]）、集成供应链、IT系统重整及财务四统一等8个项目。

面对内部阻力，任正非态度坚决："我最恨'聪明人'，认为自己多读了几本书就了不起。我们将通过培训、考试竞争上岗，即使有人认为自己比IBM还厉害，不能通过考试的也要下岗。"

引入IBM的[[IPD]]系统后，华为大大缩短了产品研发周期，降低了研发风险系数，在知识产权重视程度和市场竞争能力上都取得了显著提升。

### 灵活应变适应市场发展

华为不断检视自身运营状态，学习竞争对手的优点，提升管理、科研、营销等各方面应对市场的能力。任正非用"拧毛巾"比喻市场竞争：毛巾还能拧出水来，说明还有竞争空间；毛巾拧断了，企业必将宣告结束；只有将毛巾拧干而不断，才是最佳状态。

## 四、成长离不开学习

华为通过四种方式营造学习氛围：**终身学习、全员学习、全过程学习、团队学习**。

### 华为大学

2005年正式注册成立，以赋能为中心，围绕客户需求展开教学。讲师必须有实践经验，教学案例全部来自华为一线。任正非亲自担任指导委员会委员，三个轮值CEO做委员。华为大学不是培养理论家的地方，而是教授作战技巧的地方。

### 民主生活会

每月组织一次，每位参会员工进行自我剖析，将不足和错误讲清楚，挖揭根源，为他人竖起镜子。华为民主生活会在批判的同时严厉禁止歪曲事实和对他人展开人身攻击。通过民主生活会，华为塑造了员工的反思精神，推动了学习型组织的建设。

## 要点总结

| 成长要素 | 核心内容 |
|---------|---------|
| 生存为大 | "活下去"是第一战略，艰苦奋斗是成长基石 |
| 专注本业 | 不在非战略机会点上消耗力量，拒绝多元化诱惑 |
| 计划先行 | 谋定而后动，先瞄准再开枪 |
| 削足适履 | 引入先进管理要完全照做，先僵化后优化再固化 |
| 变革创新 | 根据市场变化灵活调整，主动拥抱变革 |
| 学习型组织 | 终身学习+全员学习+全过程学习+团队学习 |
